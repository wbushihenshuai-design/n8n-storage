# ​这3种父母，最容易把孩子养成“仇人”，尤其是第三种

我有个发小，前阵子在微信上跟我吐槽她妈。

起因是她妈给她寄了一箱苹果，箱子打开，一半都烂了。

她就拍了张照片，跟她妈说：

**“妈，以后别寄了，路上都坏了，我这边买也方便。”**

你们猜怎么着？

她妈直接在电话里就哭了，说她辛辛苦苦挑的苹果，女儿一点不知道感恩，嫌弃她了，白养了这么多年。

  


我发小当时就懵了，拿着电话跟我说：

**“我真的，就只是想告诉她苹果坏了啊！”**

你们有没有过这种时刻？

一种深深的无力感。

**你说的是A，你爸妈听到的是B，然后给你安上一个C的罪名。**

我见过很多家庭，聊过很多朋友。

我发现，亲子关系最后搞得鸡飞狗跳，甚至反目成仇，来来回回就那么几种模式。

**今天我就聊聊，我观察到的，最容易把孩子逼成“仇人”的三种父母。**

  


#  第一种：以“为你好”为名的，绝对控制者。

我大学有个室友，画画特别有灵气，一心想考美院。

她妈呢，是我们学校的会计。

一个非常体面，永远在微笑的阿姨。

**当时她妈就一句话：**

“画画能当饭吃吗？女孩子，安安稳稳最重要。我已经托人问好了，学会计，毕业了我安排你进银行。”

非常熟悉的话术，对不对？

室友抗争过，绝食过，把自己锁在房间里。

没用。

她妈就坐在房门口，一边流泪一边说：

**“妈妈都是为了你好啊，你现在不懂，以后会感谢我的。”**

  


**后来，室友还是去读了会计。**

毕业后，也真的进了银行。

工作稳定，收入不错，成了亲戚口中“别人家的孩子”。

**但是，她再也没碰过画笔。**

我们每次聚会，她从来不聊自己的工作，眼神里那种光，没了。

有一次喝多了，她跟我说：

**“我不是恨她。我只是觉得，她杀了我一次。”**

你看，这种父母，他们不打不骂，甚至为你铺平了所有道路。

他们的爱，是真的。

他们的“为你好”，也是真的。

但他们错在，把孩子当成了自己的附属品，一个需要被精心设计的“作品”。

**他们给的爱，太满了，满到没有留下一丝缝隙，让孩子自己去呼吸。**

这种爱，说白了，就是一把温柔的刀。

  


# 第二种：永远在牺牲，让你背负愧疚的“受难者”。

我有个表姐，从小到大，最怕听她妈说一句话：

**“要不是为了你，我早就跟你爸离婚了！”**

这句话是不是很耳熟？

**还有各种变体，比如：**

**“要不是为了你上学，我这辈子能这么累吗？”**

**“我这辈子没过过一天好日子，还不都是为了你？”**

我表姐说，她小时候，每次听到这些话，都觉得自己是个罪人。

是她，毁了妈妈的人生。

所以她拼命学习，拼命听话，想用自己的优秀，去“补偿”妈妈的牺牲。

**她活得不像自己，像一个赎罪的人。**

  


后来她嫁人了，想搬出去住。

她妈又开始那一套，说女儿翅膀硬了，不要她这个苦了一辈子的妈了。

**总之，一切事情，最后都能绕回到“我为你牺牲了一辈子，你现在必须听我的”这个逻辑上。**

这种父母，他们把自己活成了一个“家庭烈士”。

他们付出的，不是爱，而是一笔感情的高利贷。

**孩子一辈子都在偿还这笔永远还不清的债。**

这种“恩情”，太重了，重到最后孩子只想逃跑。

因为不跑，就会被压得窒息。

  


# 第三种：情绪极其不稳定，让你猜不透的“天气预报”。

这一种，是我觉得最可怕的。

**前两种，好歹他们的“逻辑”是稳定的。**

控制狂，永远想控制。

受难者，永远在牺牲。

你知道他们的套路。

**但第三种父母，你永远不知道他们的套路。**

他们今天对你温情脉脉，明天可能就因为你打破一个碗而歇斯底里。

你考了95分，他心情好，会抱着你亲，说你是他的骄傲。

下次你同样考了95分，他可能因为在公司受了气，回来就指着你鼻子骂：

**“你怎么这么笨！就不能考个100分吗！”**

  


跟这种父母生活在一起，你知道是什么感觉吗？

**就像在雷区里走路。**

你永远不知道哪一步会踩雷。

为了不踩雷，你只能小心翼翼，察言观色，把自己变成一个敏感又讨好的“小大人”。

你做任何事之前，想的不是“我想不想做”，而是“我爸我妈现在心情怎么样？我这么做会不会惹他们生气？”

  


我见过一个朋友，就是这样长大的。

**他现在是我们圈子里公认的“老好人”，对谁都客客气气，从来不敢发表反对意见。**

我们都觉得他活得太累了。

有一次我跟他深聊，他说：

**“我没办法。我只要一跟人意见不合，我心跳就加速，手心冒汗。我总觉得对方下一秒就要爆发了。”**

这种模式，对孩子的伤害是刻在骨子里的。

因为它摧毁了一个人最底层的东西：**安全感和自我认知** 。

你不知道自己到底好不好，因为评判标准一直在变。

你不知道什么是爱，因为给你爱的那个人，随时会变成伤害你的人。

孩子需要的不是一个完美的神，而是一个稳定的人。

  


**说到底，所有最终走向“仇人”的亲子关系，都源于一件事：**

**父母没有把孩子当成一个独立的，“人”。**

**而是一个“我的孩子”。**

你是我生命的延续，所以你要替我完成梦想。

你是我牺牲的价值，所以你要对我言听计从。

你是我的情绪垃圾桶，所以你要承受我所有的好与坏。

但我活了这么多年，见过那么多家庭的分分合合，我发现一个真理：

**好的关系，无论是夫妻，还是亲子，根基只有一个，那就是尊重。**

我尊重你是一个独立的个体。

你有你自己的想法，你的人生道路，你的喜怒哀乐。

我能做的，是陪你走一段路，给你一些建议，在你摔倒的时候扶你一把。

**而不是，抢过你的方向盘，告诉你该往哪儿走。**

  


**为人父母，可能最难的一课，就是学会得体地退出。**

从孩子的生命中，慢慢地，把自己从主角，变成配角，最后变成一个温暖的观众。

看着他，成为他自己。

这条路很难。

但只有难走的路，才通往更好的风景。

**也祝你，能和你爱的人，好好相处。**
