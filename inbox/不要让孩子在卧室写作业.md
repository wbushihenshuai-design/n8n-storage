# 不要让孩子在卧室写作业

我跟你说，当孩子背着书包一进门，多少爸妈的口头禅就是那句：

**“赶紧的，回你屋写作业去！”**

哎哟我的天，你以为这是爱，是给了娃一片清净的学习小天地？

清醒一点姐妹！

你这可能是在亲手给你家娃的未来挖坑，**一个关于专注力、睡眠和身体健康的巨大天坑！**

但凡家里还有点地方，求求了，赶紧把那张书桌从卧室给我搬出来！

为啥？

因为这背后的坏处，多到能让你惊掉下巴。

  


# 1、影响孩子睡觉，觉都睡不踏实

咱先掰扯一个最基本的问题：

卧室是干嘛的？

废话，当然是用来睡觉的啊！

我们必须在孩子的大脑里刻下一个公式：

**卧室=彻底放松+睡觉。**

可你一旦让写作业这件事入侵了卧室，这个神圣的公式就被彻底干废了。

**你想象一下那个场景：**

你家娃被一道破数学题虐得死去活来，抓耳挠腮了半小时，带着一脑子的挫败感和满格的烦躁，两腿一蹬爬上床。

你觉得，他能“嗖”地一下就睡着吗？

做梦吧！

就算身体睡着了，他的大脑还因为白天的紧张情绪，在开奥数研讨会呢，根本没法进入深度睡眠。

说白了，**把学习和睡觉的地方搞在一起，就跟在火锅店里练瑜伽一样，那味道，那感觉，能对吗？**

时间一长，孩子一走进卧室，感到的不是好舒服好放松，而是“啊我还有作业”的窒息感，连做梦都是函数和几何，你说这觉能睡得香吗？

  


# 2、卧室里的诱惑太多，根本静不下心

**别太迷信你家娃关上门后的自制力，OK？**

卧室，对孩子来说，那就是一个藏着无数秘密和快乐的潘多拉魔盒。

你以为把漫画书、玩具、游戏机都收起来就天下太平了？

姐妹，你还是太年轻了。

孩子的想象力能甩我们八条街。

 _**一张草稿纸，他能给你折出一整个航母编队；**_

 _**一根破笔，他能玩出花式转笔大赛冠军的风采；**_

 _**实在啥也没有了，他还能盯着墙上的一个斑点，思考半小时人生的终极意义。**_

**卧室那扇门，隔绝的是你的视线，但却给孩子的摸鱼、拖延、神游天外提供了最完美的天然屏障。**

你在门外听着一片寂静，以为他学到天昏地暗；

其实他在门里，正进行着一场一个人的狂欢。

我跟你说，这种看起来很努力的“假性努力”，才是最浪费生命、最让人火大的！

  


# 3、对身体不好，容易养成坏习惯

**卧室最大的“原罪”是什么？**

是它太舒服了！

而这种舒服，对学习时需要端正的身体姿态来说，简直就是“温柔的杀手”。

 _**没有大人在旁边叨叨，孩子写着写着，人就从椅子上“流”下来了，歪在床上写；**_

 _**或者干脆把书摊在地上，整个人趴着，练什么“蛤蟆功”。**_

这些千奇百怪的姿势，当下是舒服了，可背地里，正在一点点地毁掉孩子的脊椎和视力。

驼背、高低肩、近视度数蹭蹭往上涨……多少坏毛病，都是在卧室那扇紧闭的门后，在无人监管的舒适区里，悄悄养成的。

等你发现的时候，往往已经到了需要花大钱、花大力气去纠正的地步了，到时候哭都来不及！

  


# 4、那到底在哪儿学最好呢？

答案来了，记好了：

**客厅！（或者餐厅、开放式书房这种全家都能看见的地方）。**

先别急着反驳说客厅吵。

当孩子在写作业时，我们当家长的，能不能暂时放下手机里的搞笑视频和狗血电视剧？

咱也拿起一本书，或者打开电脑处理工作，安安静静地待在旁边。

这就营造出了一种特别高级的家庭氛围——**“我们各忙各的，但我们在一起”。**

这种氛围，本身就是一种最有效的**“隐性监督”** 。

孩子不需要你跟个监工一样坐在旁边，但他一抬头，看到那个生他养他的爹妈也在认真做事，这对他来说，就是最好的榜样，比你吼100句“快点写”都有用。

  


**他会从心里明白，现在是家里的学习时间，大家都在努力，我不能掉队。**

而且，你也根本不用再被那些几千块的“人体工学学习桌”或者死贵的“护眼灯”收割智商税了。

一张几十块的折叠桌就够了，关键是有人看着，坐得端正。

**真正护眼的，是让他放学后先在外面野一个小时再回家，是周末多带他去公园撒欢儿！**

等作业一写完，客厅瞬间切换回娱乐模式。

全家人瘫在沙发上，聊聊学校里的八卦，分享点水果，看个电影。

这种热气腾腾的亲子交流，不比各回各屋、对着屏幕傻笑要珍贵10000倍吗？

**当然，如果孩子已经上了初中高中，习惯养成了，咱也别硬掰，强扭的瓜不甜。**

但如果你的娃还在幼儿园或小学，恭喜你！

这正是帮他建立一个能让他受益终生的好习惯的黄金时期，抓紧行动起来吧！
