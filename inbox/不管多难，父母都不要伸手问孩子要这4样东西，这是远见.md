# 不管多难，父母都不要伸手问孩子要这4样东西，这是远见

都说望子成龙，这本没错，但“望子成龙”不等于“拔苗助长”。

父母的角色，是在孩子羽翼未丰时提供温暖的港湾；

而不是让他们刚踏入人生的学堂，就背负起超出年龄的重担。

孩子蹒跚学步，人生刚启程，

再爱再累，也请你别轻易向你那未成年的孩子要这4样东西。

  


# 01\. 别轻易问他要“高分和完美”

学习这回事，看起来是孩子的任务，却最能暴露父母的心态。

很多父母，在孩子刚进入小学一年级时，就渴望他们次次满分，名列前茅。

**他们会说：**

  * “看看隔壁莉莉又考了100分，你为什么不能？”
  * “你这道题又错了，是不是上课不认真？”



可是，孩子正处于学习基础和习惯的培养阶段，他们的世界不该只剩下分数和排名。

你让他把所有的精力都耗在追求那几分上，就像把一株刚破土的幼苗，强行拔高。

他嘴上不说，心里却早已断了那根**“学习乐趣的弦”** 。

有远见的父母，

看的从来不是孩子有没有考满分，

而是努力不给孩子**“唯分数论”** 的理由。

想想看，如果你能把孩子对学习的兴趣和探索欲守护好，

不给孩子添焦虑，本身就是一种的爱。

  


# 02\. 别把“过重家务和责任”压给他

有些人，觉得孩子长大了，就能做很多事了。

日常里控制不住的疲惫、期望、不满，全往孩子身上泼。

  * “你都1年级了，还不能把自己的碗洗了？”
  * “你赶紧把你弟弟哄睡了，我累死了！”
  * “你回家就知道玩，家里都是你妈在忙活！”



可是，孩子不是你人生的**“家务保姆”** 或**“育儿助理”** 。

你嘴上说的事，其实是把自身责任、过度期望、甚至无奈全搅在一起的**“负担”** 。

而这些你说出口的**“重担”** ，孩子不是做不到，而是不敢拒绝。

然后，他就会默默地收着、吞着、忍着，最后变成一个过早承担一切的“小大人”。

所以，别再对孩子输出你的疲惫和不满了，

父母应该成为他的**“童年守护者”** ，而不是**“负担倾泻口”** 。

  


# 03\. 别把“成人的情绪垃圾”倒给他

**弥补父母的烦恼，不是孩子的义务。**

你工作不顺心，就向他抱怨领导多么苛刻；

你和伴侣有摩擦，就拉着他评理，甚至让他站队；

你生活感到压力，就变着法让他成为你情绪的出口。

你以为是**“跟孩子分享”** 的信任，

在孩子看来却是你把他当做情绪的“消化器”。

可是，孩子是个独立的人，不是你成人世界的**“心理疏导师”** 。

他能走多远，是他的选择；

你帮他扫清前路，是你的本分；

但你不能替他承受，更不能让他成为你的**“情绪垃圾桶”** 。

你该做的，不是让他背着你的烦恼小心翼翼，

而是教他学会面对，给他敢于飞翔的底气。

  


# 04\. 别从他身上“寻求完全顺从”

**有些父母，孩子刚开始有自己的想法，就开始抱怨：**

  * “你为什么不听我的话？”
  * “你怎么总是想和那些调皮的孩子玩？”
  * “周末我有安排为什么不告诉我，你是不是不爱我了？”



你明明身体健康、生活自由，却开始把孩子的“独立”当成对你的“背叛”。

你总觉得孩子还没长大，离不开你，

可现实是，孩子终究是你人生的过客。

我们不该从孩子身上**“索取完全的顺从”** ，

而是要把自己活成一个能独立思考、不害怕孩子有主见的大人。

你要有自己的兴趣、有自己的生活，

这样哪怕孩子飞得很远，也知道——

**家，不是枷锁，是港湾。**

  


#  写在最后：

爱，远非一种简单的情感，

而是一门需要终身学习的艺术。

它不是被动地索取，

也不是偶然的得到，

而是主动地去给予。

真正的爱，如同成熟的果实，

是在守护孩子独立与完整的前提下，

实现彼此的连接。

这意味着我们不强求，

不压迫，不倾倒，不控制。

唯有如此，这份爱才能真正超越本能的羁绊，

成为家庭长久的底气。
