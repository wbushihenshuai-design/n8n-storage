# “野马效应”教孩子改掉爱发脾气的坏习惯

**明明是别人做错事，我们却气得吃不下饭、睡不好觉，甚至做出一些伤害自己的傻事儿。**

明明只是生活中的一点小不顺，我们却陷入负面情绪的泥潭，久久无法自拔。

这种**“替别人错误买单”** ，让自己情绪失控的现象，在心理学上被称为**“野马效应”** 。

其实，**“野马效应”** 本质上是指，**我们常常因为对外部刺激的过度反应，而让自己陷入更大的困境。**

你是不是也经常遇到这样的情况？

**孩子明明是因为一点小事，比如玩具被拿走、想要的零食没买到，就大发脾气，哭闹不止，甚至在地上打滚。**

作为家长，我们有时候也会感到束手无策，甚至跟着情绪失控。

结果却发现，这样不仅解决不了任何问题，反而让亲子关系变得更紧张，甚至还影响了孩子的健康成长。

这到底是咋回事呢？

这背后，其实藏着一个心理学上的小秘密，叫做**“野马效应”** 。

听起来是不是有点陌生？

别急，今天我就用最通俗易懂的方式，告诉你什么是**“野马效应”** ，

以及它为什么能成为你家**“熊孩子”脾气的“情绪灭火器”** ！

心理学家们研究发现，在非洲大草原上，**有一种吸血蝙蝠，它们专门靠吸食动物的血液生存。**

**野马是它们最常攻击的目标之一。**

**蝙蝠会叮咬野马，吸食它们的血液。**

一开始人们认为，有些野马之所以会丧命，是因为被蝙蝠吸血过多导致的。

但后来的动物学家研究发现，**蝙蝠吸食的血量其实非常少，根本不足以致命。**

真正导致野马死亡的原因，是它们自己！

原来，当野马被蝙蝠叮咬后，会变得异常暴躁和狂怒，为了摆脱蝙蝠的叮咬，它们会不停地狂奔、躲避，结果精疲力竭，活活累死。

**真正杀死野马的，不是蝙蝠吸的那点血，而是野马面对叮咬时，那失控的暴怒情绪和过度反应！**

**“野马效应”** 正是由此而来，它形象地说明了：

很多时候，就像野马会被自己的过度反应所伤害一样，孩子也常常会被自己的**“小脾气”** 所困扰。

想想看，当孩子因为一点小事而发脾气、哭闹、甚至打滚的时候，是不是很像那只被蝙蝠叮咬后狂奔的野马？

**也许“玩具被抢了”、“想吃的零食没买到” 这些事情本身，就像蝙蝠叮咬一样，只是一个小小的诱因，并不会对孩子造成实质性的伤害。**

然而，如果孩子像**“小野马”** 一样，任由自己的脾气**“狂奔”** ，用大哭大闹、 乱扔东西、甚至攻击他人等方式来发泄情绪，最终真正伤害到的，往往是孩子自己，以及他们与家人、朋友之间的关系。

就像狂奔的野马最终会力竭而死一样，孩子长期沉浸在负面情绪中，也会变得痛苦、不快乐，甚至影响他们的身心健康发展。

所以，**“野马效应”** 提醒我们，在面对孩子的坏脾气时，更重要的是帮助孩子学会管理自己的情绪，而不是一味地指责和压制。

**那么，如何运用“野马效应”，成为孩子脾气的“情绪灭火器”呢？其实很简单，只需记住这三招：**

  


>   
> 
> 
> #  第一招：“冷处理”孩子的坏脾气，降低负面关注。
> 
> **当孩子开始发脾气、哭闹、满地打滚的时候，爸妈们先要做的就是稳住！ 冷静！深呼吸！**
> 
> **在心里默念三遍：这是“野马效应”在作祟，我要冷静，冷静，再冷静！**
> 
> 然后，试试用**“眼神忽视”** 和**“语言肯定情绪”** 这招**“冷处理”** 大法。

  


>   
> 
> 
> **眼神忽视：**
> 
> **注意！这可不是真的不理孩子，** 而是**不要用那种焦虑、指责、愤怒的眼神死死盯着孩子，尽量避免眼神交流，** 减少孩子从你这里获得负面关注的机会。
> 
> 眼神越平静，孩子越觉得**“没意思”** ，闹起来也就没劲了。（相信很多家长看到这里，都会会心一笑，这确实是很多有经验的家长都在使用的方法！）

  


>   
> 
> 
> **语言肯定情绪：**
> 
> 用平静、温柔的语气，说出孩子当下的情绪，但不要评判对错，更别跟着孩子的情绪起舞。**比如，你可以这样说：**
> 
>   * 我知道你现在很生气，因为手机被拿走了，你很难过，是吗？
> 
>   * 你想要吃冰淇淋，妈妈没同意，你有点失望，是不是？
> 
> 

> 
> 这样做的目的，就像 悄悄撤走了“小野马”面前的障碍物，**让孩子意识到，再怎么挣扎哭闹，也得不到预期的“负面关注”，** 反而会慢慢平静下来， 就像泄了气的皮球一样。

  


>   
> 
> 
> # 第二招：“正向关注”孩子的好行为，强化积极行为。
> 
> **“野马效应”的反面，就是“正向关注”啦！**
> 
> 与其天天盯着孩子的坏脾气不放，不如把咱们的眼睛**聚焦在孩子的好行为上，多发现、多鼓励、多表扬！**
> 
> 当孩子表现出**耐心、分享、合作、情绪稳定等等积极行为** 的时候，爸妈们**一定要及时、具体、真诚地表扬和鼓励他们** ，让孩子知道，这样做才是棒棒的，是被爸爸妈妈认可和喜欢的！

  


>   
> 
> 
> **比如，你可以这样夸：**
> 
>   * 宝宝今天真棒！和小朋友一起玩玩具，没有抢也没有闹，真是一个懂事的好孩子！
> 
>   * 你刚才等妈妈忙完再说话，很有耐心，妈妈很感谢你！你真是个贴心的小宝贝！
> 
> 

> 
> 给予**“正向关注”** ，是启动孩子好行为正循环的关键，让他们知道，这样做是会被认可和赞赏的，从而更愿意重复这些好行为，变得越来越棒！

  


>   
> 
> 
> # 第三招：“提前预防”脾气爆发，建立良好沟通。
> 
> **“防患于未然”永远是最高明的育儿策略！**
> 
> 与其等到孩子脾气爆发了才手忙脚乱地去**“灭火”** ，不如提前做好预防工作，从源头上减少孩子发脾气的机会。
> 
> **了解孩子发脾气的“导火线”：**
> 
> 细心观察和记录一下，孩子通常在什么情况下容易发脾气？
> 
> **比如，是饿了、困了、累了？**
> 
> **还是玩具被抢了、要求没被满足？**
> 
> 找到**“导火线”** 后，尽量避免这些情境的发生。

  


>   
> 
> 
> **教孩子认识和表达情绪：**
> 
> 从小就教孩子认识各种情绪词汇（开心、生气、难过、害怕等等），鼓励他们用语言表达自己的感受，而不是只会用哭闹、发脾气来表达不满。
> 
> **“宝宝，你现在是不是有点生气呀？因为......吗？”**
> 
> **建立良好的亲子沟通：**
> 
> **每天抽出时间和孩子进行高质量的亲子互动，** 认真倾听他们的想法，耐心理解他们的需求，建立起信任和安全感的亲子关系，让孩子感受到爸爸妈妈满满的爱和支持。
> 
> **当孩子内心感到被理解、被尊重、被爱的时候，内心的那匹“小野马”自然也会变得温顺起来，脾气也会越来越好，越来越“乖”啦！**

  


>   
> 
> 
> **“野马效应”** 并不是要我们对孩子的坏脾气放任不管，而是要我们用更有效的方式去引导和教育孩子。
> 
> 放弃那些无效的吼叫和打骂吧！
> 
> **试试运用“野马效应”这个神奇的“情绪灭火器”，** 你会惊喜地发现，当你改变了对待孩子脾气的方式，孩子也会悄悄地发生改变，变得越来越可爱，越来越懂事！
