# 关系再亲近，也不要跟孩子分享这5件事，真的很傻

心理学上有个非常经典的防御机制，叫**「投射」** 。

**西格蒙德·弗洛伊德** 最早提出了这个概念，后来被他女儿**安娜·弗洛伊德** 进一步细化。

它指的是，**一个人潜意识里无法接纳自己糟糕的情绪、欲望或失败，于是下意识地把这些东西投射给其他人** 。

这就好比自己心里有鬼，看谁都像鬼。

有些父母，在职场受了气，在婚姻里受了委屈，自己消化不了。

于是，他们把这些无法处理的负面情绪，投射到了弱小的孩子身上。

别傻了。

这不叫亲密，这叫**情绪霸凌** 。

孩子的心智容量，根本承载不了成人世界的复杂和灰暗。

**关系再亲近，下面这5件事，请你藏在肚子里，不要跟孩子说。**

  


#  1.需要孩子被迫撒谎的事情

“这事儿别告诉你爸，这是咱们俩的小秘密。”

“这钱是你妈不知道的私房钱，你别说漏嘴了。”

看似跟孩子是队友，实则是把孩子拉进了你们夫妻博弈的斗争。

你让孩子在还没建立起是非观的时候，就先学会了欺骗。

更残忍的是，你让他陷入了**“忠诚度的撕裂”** 。

**为了对你好，他必须背叛另一个人。**

这种内心的煎熬，对成年人来说或许是心机，对孩子来说，是巨大的心理负担。

**如果你需要拉拢孩子来对抗伴侣，只能说明你的婚姻已经极其失败。**

  


#  2.对家庭成员的负面评价

“你妈太自私了，心里只有她自己。”

“你爸这个人，一辈子没出息，你要是像他就完了。”

你以为你在发泄不满，或者是想以此警醒孩子。

但在孩子听来，你其实是在否定他自己。

孩子一半来自父亲，一半来自母亲。

当你攻击他的父亲或母亲时，你其实是在攻击孩子的一半基因。

**心理咨询中经常发现，那些极度自卑的孩子，往往是因为父母一方长期在贬低另一方。**

孩子会觉得既然我爸妈那么糟糕，我也好不到哪去。

  


# 3.你婚姻冲突的细节

有些家长，吵完架喜欢拉着孩子评理。

“你说说，今天这事儿是不是你爸的不对？”

甚至把出轨、冷战的具体细节，一股脑倒给孩子。

**这是典型的没边界感。**

两个成年人处理不了彼此的问题，非要拉进第三个人来平衡。

可惜这第三个人，是完全依赖你们生活的孩子。

知道了父母婚姻不堪的细节，孩子不会觉得你可怜，他只会感到深深的恐惧**。

**因为在他们眼里，这个家快散了。**

  


#  4.工作的负面情绪

每天一回家，鞋子一踢，就开始抱怨：

“牛马的一天终于结束了。”

“这破班我是上一天都不想上了。”

你是痛快了，投射完成了。

孩子呢？

孩子的世界观还没建立起来，你就先给他涂上了一层底色：

**成人世界太苦了。**

真正成熟的父母，不是不喊累。

**而是会告诉孩子：**

今天工作确实遇到很难搞的挑战，我很累，但我搞定它了，或者是我正在想办法解决它。

你要给孩子展示的，不是一个完美的职场人，而是一个**虽然辛苦，但依然有掌控感的成年人** 。

**现在的孩子普遍缺乏内驱力，很大程度上是因为父母把未来的图景描绘得太灰暗了。**

  


#  5.你的个人遗憾

“妈妈当年就是看走了眼，才会嫁给你爸。”

“爸爸年轻时想当画家的，没当成，你一定要帮爸爸实现。”

这就是父母的**“代偿心理”** 。

你自己的人生剧本演砸了，不想着怎么修补，却想抢过孩子的笔，在他的人生白纸上乱涂乱画。

孩子是一个独立的个体，不是你人生的续集。

这种沉重的期待，往往是压垮孩子的最后一根稻草。

**你的遗憾是你自己的课题，请你自己去面对，别把它打包成“礼物”送给孩子。**

  


**最后总结一下。**

你说，养孩子要不要看书？

要不要学心理学？

当然要。

**注意，技巧是次要的，界限感才是核心。**

什么是成熟的父母？

成熟的父母，是**“自带容器”** 的人。

**你有能力容纳自己的焦虑、愤怒、失败和遗憾，并在自己的内部消化处理。**

而不是把气全撒在身边最弱小的孩子身上。

**好的亲子关系，确实需要分享。**

但请分享你的快乐，分享你的思考，分享你对世界的好奇。

**至于那些成人的泥沼、不堪和重负，请留给自己，或者留给你的心理咨询师。**
