# 不要太尊重你的孩子，哈佛大学研究表明：父母若将重大选择完全交给孩子，等同让他们在风暴中独自掌舵

**现在都强调“快乐教育”。**

很多父母小心翼翼，生怕扼杀了孩子的天性。

我们把**“尊重”** 挂在嘴边，甚至不敢替孩子做主，好像只要顺着他们就是最好的爱。

遗憾的是，**未成年人的天性，大多时候只是避难趋易的动物本能。**

你一旦把人生的方向盘，完全交到一个大脑尚未发育成熟、只看得见眼前快乐的孩子手中，这不叫尊重，而是放纵了。

真正的爱，不仅需要弯腰倾听的温柔，更需要站直了替他挡风遮雨的锋芒。

  


# 01

**顺应天性的尊重其实是在纵容动物本能**

##  什么是本能？

**人脑中有一个原始区域，负责本能欲望。**

对孩子来说，天性就是趋利避害的。

 _只有傻瓜才想做枯燥的数学题，天性都喜欢刷短视频；_

 _只有苦行僧才喜欢早起背单词，天性都喜欢睡懒觉吃薯片。_

如果你所谓的“尊重”，是孩子想不学就不学，想玩多久就玩多久。

**那你尊重的不是你的孩子，而是他体内那个贪图安逸、目光短浅的“动物性”。**

  


#  02

**把重大选择交给孩子是一场巨大的冒险**

哈佛大学儿童发展中心曾发布过一项关于大脑发育的研究：

人类负责理性决策、情绪控制的大脑区域的前额叶皮层，要到**25岁左右** 才完全发育成熟。

这意味着什么？

**意味着未成年的孩子，在生理上就是“短视”的。**

孩子的大脑构造决定了，他们很难理解当下的放弃是为了未来的收益。

**哈佛大学心理学家丹尼尔·西格尔指出：**

**孩子的决策能力在18岁前尚未成熟 ，父母若将重大选择完全交给孩子，等同于让他们在风暴中独自掌舵。**

  


#  03

**真正的尊重是温和而坚定的执行规则**

那么，还要不要尊重孩子？

当然要。

但我们要搞清楚，**尊重的是“感受”，而不是“行为”** 。

举个很具体的例子：

**孩子不想写作业，在那哭闹。**

❌ 错误的尊重（放纵）：“宝宝别哭了，不想写就不写了，妈妈尊重你的决定，开心最重要。”

➡️ **结果：孩子学会了用情绪勒索规则。**

❌ 错误的管教（独断）：“闭嘴！再哭我就不要你了，赶紧给我写！”

➡️ **结果：孩子产生了仇恨和恐惧。**

✅ 正确的尊重（规则型）：妈妈知道你很累，这道题很难，你现在感到很烦躁，我理解你的心情**（尊重感受）** 。

但是，今天的作业必须在8点前完成，这是规则**（坚定行为）** 。

你是想先做语文换换脑子，还是喝杯牛奶再继续？**（在界限内给予选择）**

这就叫温和而坚定。

**我对你的情绪表示接纳，我爱你；**

**但我对底线绝不退让，我有原则。**

  


#  04

**写在最后**

教育学家乌申斯基说过一句话，我一直记在备忘录里：

**“如果你养成好的习惯，你一辈子都享受不尽它的利息；”**

**“如果你养成了坏的习惯，你一辈子都在偿还无尽的债务。”**

孩子尚且年幼，他看不见那笔高额的“债务”。

甚至，他会因为你逼他“存钱”而恨你、冲你发脾气。

没关系。

做父母，本身就不是来讨好孩子的。我们是那个守夜人。

我们要做的，是在他羽翼未丰时，替他挡住诱惑的风暴；

在他还没能力掌舵时，把住大方向，别让他触礁。

**别让你泛滥的“尊重”，成为压垮孩子的稻草。**

**带点锋芒的爱，才能护他周全。**
