# 吼了孩子后，最有效的补救不是"道歉"，而是这3句话

> ❝
> 
> **被爱就是，这个人不审判你。**
> 
> **——卡尔·荣格**

这年头，当爹妈的，谁没因为失控吼过孩子？

但你想把这伤害降到最低，甚至把一次情绪失控变成一次高质量的情感链接，其实非常简单。

在你吼完之后，第一时间，死死管住你的嘴，不说那句最没用的“对不起”。

**你只要做到这一点，先用行动代替道歉，孩子的安全感，就会迅速被重建。**

而且会跟那些只会苍白道歉的父母，在孩子内心的信任账户上，瞬间拉开身位。

  


# 01 用平静的身体语言，轻轻抱着孩子说：“刚刚过去了，现在没事了，安全了。”

我很喜欢心理学里的一个词，叫**“共同调节”（Co-regulation）** 。

它听起来还有点学术，但你一定体验过的。

就像我们小时候害怕打雷，奶奶会把我们搂在怀里，她的体温、她平稳的呼吸，甚至她身上淡淡的肥皂味，都会让我们的心跳慢慢平复下来。

**我们的大脑和情绪，会不自觉地被身边那个更稳定的人“同步”。**

这就是用一个人的安稳，去熨平另一个人的慌乱。

所以，在我们对着孩子发脾气后，第一件要做的事，不是讲道理，也不是道歉，而是用我们的身体，去帮他“灭火”。

这时候的语言，要像消防毯一样，简单、直接、有包裹感。

  


## 试试这样做：

深呼吸一口气儿，把自己从刚才那个“失控的妈妈”身份里抽离出来。

再慢慢走到孩子身边，蹲下来，视线和他齐平。

不要急着说话，先张开双臂，轻轻地把他抱进怀里。

如果他抗拒，也没关系，你就在他身边坐下来，用你的存在告诉他：

**妈妈在这里。**

等孩子稍微放松一点，你可以把他抱在怀里，或者只是拍拍他的背，用你所能达到的最温柔、最平静的声音，在他耳边说：

**“刚刚过去了，现在没事了，安全了。”**

这句话里没有评判，没有指责，更没有要求他立刻原谅你。

它只是一个事实陈述，像一个安全开关，在告诉孩子那个高度紧张的警报可以解除了。

  


**孩子的世界很单纯，在他被吼的那一刻，身体会进入一种“战斗或逃跑”的应激状态。**

他感觉到的不是“妈妈在教育我”，而是“我遇到了危险”。

**所以，先用拥抱后再说出这句话，是把他的身体从“危险模式”切换回“安全模式”。**

这是所有后续沟通的唯一基础。

**先连接，再纠正。**

这顺序，永远不能错的。

  


# 02 看着孩子的眼睛，替他说出感受：“你刚才一定吓坏了，对不对？”

你知道吗，我们的大脑里有一个很有趣的机制，**这个发现来自加州大学洛杉矶分校的心理学教授马修·利伯曼（Matthew Lieberman）** 。

他通过脑部扫描发现，当我们能用语言准确地**“命名情绪”（Naming Emotions）** 时，大脑里负责恐惧和焦虑的杏仁核活动就会减弱，而负责理智思考的前额叶皮层会变得活跃。

简单来说，就是当我们能说出一种感受的名字时，这种感受对我们的控制力就会变小。

感觉就像一个藏在暗处的、面目模糊的怪物，你越是看不清它，就越是害怕。

可一旦你打开灯，看清了它的样子，并叫出它的名字——“哦，原来你叫‘害怕’啊”，它好像瞬间就没那么可怕了。

孩子被吼之后，内心就是一团乱麻。

他可能同时感受到了害怕、委屈、困惑、伤心……但他没有能力去分辨和表达，这些情绪混合在一起，就成了一个巨大的黑洞。

**这时，就需要我们来做那个帮他“开灯”和“命名”的人。**

  


当你完成了第一步，用身体安抚了他之后，可以稍微拉开一点距离，看着他的眼睛，用一种确认的、共情的语气说：

**“你刚才一定吓坏了，对不对？”**

或者，你也可以根据你的观察，说得更具体一些：

**“妈妈刚才那么大声，你的心是不是‘咚咚咚’跳得特别快？”**

**“我看到你眼泪在打转，你是不是觉得特别委屈？”**

当你说出这句话，就等于递给了孩子一把钥匙，帮他打开了情绪的出口。

他可能会“哇”地一声哭得更厉害，也可能会拼命点头。

别怕，这是好事。

说明你猜对了，他积压的情绪找到了释放的通道。

我们替他说出感受，其实是在传递一个无比重要的信息：

“我看见你了。我不仅看见了你的行为，更看见了你行为背后那个不知所措的内心。你的感受，是被允许的。”

**一个感受被看见、被接纳的孩子，才有能力去面对和处理问题。**

  


#  03 把问题变成一个共同的“敌人”：“刚刚那个对你大吼大叫的，不是真正的妈妈，是一个叫‘坏脾气’的小怪兽。”

叙事疗法（Narrative Therapy）的创始人迈克尔·怀特（Michael White）有一个颠覆性的观点，他说：

**“人不是问题，问题才是问题。”**

**这句话的力量在于，它能帮我们把人和行为分开。**

当我们吼了孩子，我们常常会给自己贴上标签：

**“我真是个坏妈妈。”**

而孩子也可能会认为：

**“妈妈不爱我了”，或者“我是个坏小孩，所以妈妈才吼我。”**

你看，**我们都习惯于把“问题”内化成自己的一部分** 。

而“外化”这个技巧，就是把这个问题从我们身上拎出来，放在我们和孩子之间，让我们俩站在同一边，去共同面对它。

所以，在安抚和共情之后，你可以尝试用一种讲故事的方式，来解释刚才发生的一切：

> ❝
> 
> 宝贝，你知道吗，刚刚那个对你大吼大叫的，不是真正的妈妈。
> 
> **妈妈身体里住着一个叫“坏脾气”的小怪兽。**
> 
> 有时候，它会趁妈妈很累或者很着急的时候，偷偷跑出来。
> 
> **刚刚，就是它占领了妈妈的身体，把那个爱你的温柔妈妈给吓跑了。**

然后你可以问他：

“我们下次一起想个办法，在这个小怪兽要出来的时候，用一个秘密暗号把它赶走，好不好？”

你看，这么一说，整个事情的性质就变了。

冲突的双方，不再是“我和你”，而是变成了“我们和那个叫坏脾气的小怪兽”。

这不仅保护了孩子的自尊，也给了我们自己一个台阶下。

它让孩子明白，妈妈的爱是恒定的，只是妈妈也会有被“小怪兽”打败的时候。

这比一句简单的“对不起”要深刻得多。

它是在教孩子如何用一种更智慧、更有弹性的方式去看待人际关系中的冲突和人的不完美。

  


# 04 写在最后

这些补救的方法，与其说是技巧，不如说是一种提醒。

**提醒我们，在亲子关系里，看见永远比说教重要，联结永远比道理优先。**

而我们每一次笨拙又真诚的修复，都在告诉孩子一件事：

**没关系，我们都会有搞砸的时候。**

**但你看，爱，总能把我们重新粘合在一起。**
