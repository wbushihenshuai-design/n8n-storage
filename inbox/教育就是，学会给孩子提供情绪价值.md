# 教育就是，学会给孩子提供情绪价值

很多家长，包括我自己，常常陷入一种教育焦虑：

**看到问题，就急着去修补，急着讲道理。**

但这就像孩子正在大哭咳嗽，你却非要往他嘴里塞一口饭，他是咽不下去的，只会呛着。

从心理学角度看，情绪价值并不是指你要一味地讨好孩子，而是在孩子情绪产生波动的时刻，你能提供一种**“接纳、共情、缓冲”** 的心理能量。

教育的本质，有时候不是灌输，而是滋养。

如何在琐碎的日常中给孩子提供高情绪价值？

其实只需要我们日常生活中稍微调整几个词，多做一点点停留。

  


# 1️⃣多用“哦”

当孩子气冲冲地回家，把书包往沙发上一摔，或者一脸委屈地抱怨老师偏心时，我们的第一反应通常是什么？

大多数家长——可能也包括之前的我，会下意识地开启纠错模式：

“你是不是又上课说话了？”

“你怎么能摔东西呢？”

这时候，请先深吸一口气，试着只说一个字：**“哦。”**

这听起来好像有点敷衍？

其实不是。

从脑科学的角度来看，当孩子处于负面情绪爆发时，他大脑中的“杏仁核”正处于高度激活状态。

**杏仁核是负责情绪报警的，它一旦拉响警报，负责理智思考的“前额叶”就会暂时下线。**

这时候，如果你立刻反驳、建议或讲道理，在孩子的潜意识里，这等同于“攻击”。

**他的杏仁核会受到进一步刺激，引发更强烈的对抗。**

这就是为什么孩子会捂住耳朵喊“我不听”。

**一个轻声的、语调下沉的“哦”，或者“哦，这样啊”，其实是一个情绪缓冲垫。**

它传递出的信号是：

“我听到了，我没有立刻想要攻击你，你是安全的。”

**只有当警报解除，理智才能回归。**

所以，别急着辩论，先用“哦”给情绪按个暂停键。

  


# 2️⃣多用“我看见”

很多时候，孩子闹腾、作妖，甚至故意犯错，潜台词其实只有一句：

 _“你看看我。”_

人类的大脑中有一种神奇的细胞，叫做**“镜像神经元”** 。

**它需要通过别人的反馈来确认自我的状态。**

对于孩子来说，父母就是那面最重要的镜子。

如果孩子很难过，而你视而不见，或者轻描淡写地说“这有什么大不了的”，那他在镜子里看到的是一片虚无，这会让他产生巨大的恐慌感，觉得自己的感受是被无视的。

**所以，我们要试着去描述我们看到的：**

 _我看见你把拳头捏得很紧，好像很生气”_

 _我看见你眼圈红了，是不是很难过？”_

 _我看见你坐在角落里半天没说话。_

**“被看见”是存在感的基础。**

当你准确地说出他的状态时，孩子会有一种心里的大石头落地的感觉。

因为他确认了自己的情绪被你“看见”了，没有被忽视，也没有被嫌弃。

**这种确认感，本身就是一种巨大的治愈。**

  


#  3️⃣多用“你感觉”

有些孩子，特别是青春期的孩子，往往也是**“情绪文盲”** 。

他们心里像一团乱麻，只知道难受，却不知道这叫什么，于是只能通过发火来宣泄。

这时候，作为父母，我们可以充当那个翻译官，**多用“你感觉”的句式，帮他把情绪贴上标签** 。

比如：

 _被同学误会了，你感觉很委屈，是吗？_

 _明天的考试让你压力很大，你感觉很焦虑，对不对？_

 _努力了却没有得到表扬，你感觉有些失落吧？_

心理学家丹·西格尔提出过一个概念，叫**“命名即驯服”** 。

**当我们将模糊、混乱的情绪命名出来，比如愤怒、嫉妒、挫败，右脑的情绪强度就会下降，左脑的逻辑功能开始介入。**

注意，这里要用一种猜测和探询的语气，而不是定论。

比如，允许孩子纠正你：

“不是委屈，我是愤怒！”

这也是一种很好的情绪梳理。

  


# 4️⃣多用“难怪”

**如果说前面几步是接纳情绪，那么“难怪”这个词，就是给情绪做“合理化认证”。**

很多孩子在产生负面情绪时，会有二次伤害，他们会自我攻击：

 _我怎么这么脆弱？_

 _我不应该哭的。_

这时候，一句“难怪”，简直就是救命稻草。

 _原来是你最好的朋友忘了你的生日，难怪你这么伤心，换做是我，我也很难过。_

 _这道题确实步骤很绕，难怪你会做错，很多大人都不一定能一下子做对。_

这两个字背后的逻辑是：

**你的情绪是有来由的，是符合逻辑的，你不是一个莫名其妙的人。**

当孩子觉得自己的反应被理解为“正常”时，他的防御机制就卸下来了。

他不需要再为了证明自己“有理由生气”而继续生气。

  


# 5️⃣多用“是因为”

**等到孩子的情绪平复了，愿意开口说话了，我们才适合进入“分析问题”的阶段。**

但即便到了这里，也要小心，不要把聊天变成审讯。

**把质问式的“为什么”，换成推测式的“是因为”。**

 _你不想去上补习班，是因为觉得老师讲得太快跟不上吗？_

 _你刚才对奶奶发脾气，是因为她打断了你的游戏，让你觉得不被尊重吗？_

“为什么”这个词，往往带着一种高高在上的指责意味，像是在逼问孩子。

而“是因为”，则充满了一种想要了解事实的好奇心和关切。

这种探究因果的过程，能帮助孩子从情绪的泥潭里拔出腿来，开始启动理性思考：

**原来我的情绪是由这个具体原因触发的，那我们解决这个原因就好了。**

  


#  6️⃣多用“我们一起”

这是提供情绪价值的最强“定海神针”。

在面对困难、错误或者是烂摊子的时候，孩子最恐惧的不是事情本身，而是被抛弃感：

 _爸爸妈妈会不会不管我了？_

 _爸爸妈妈会不会不爱我？_

所以，无论发生了什么，请多用**“我们一起”** 。

 _这次没考好没关系，我们一起来看看卷子，分析一下哪里出了问题。_

 _闯祸了不要怕，我们一起去道歉，我们一起承担后果。_

这句话传递的力量是：

**我不仅是你的裁判，更是你的盟友。**

当孩子知道身后有靠山，知道父母愿意和他站在一起面对风雨时，他内心会生长出一种叫做“安全感”的东西。

这种安全感，是他在未来人生中面对更大挑战时的底气。

  


# 7️⃣多用“试试”

**最后，当我们需要提出建议或者引导改变时，不要下达僵硬的指令，多用“试试”。**

 _你要不要试试每天早起10分钟？_

 _下次如果生气，我们要不要试试先深呼吸3下？_

“试试”这个词，自带一种轻松的实验感。

**它暗示孩子这不是一个必须完成的死命令，如果失败了也没关系，我们只是在做一次尝试。**

这能极大地降低孩子的心理负担，让他更愿意去迈出改变的第一步。

  


有时候我也会想，与其说是我们在教育孩子，不如说是**孩子在通过各种“不听话”，逼着我们去修缮那个急躁、匮乏的自己** 。

当我们忍住说教，试着对孩子说出“我看见你的委屈”时，其实也是在对自己内心那个曾经被忽视的小孩说话。

**给孩子提供情绪价值，是一场亲子关系的双向治愈。**

当我们学会了如何温柔地对待孩子，你会发现，那个总是紧绷着的自己，不知不觉间也变得松弛了。

最后，请各位妈妈请开始造句吧！

1️⃣哦，2️⃣我看见，3️⃣你感觉，4️⃣难怪，5️⃣是因为，6️⃣我们一起，7️⃣试试。
