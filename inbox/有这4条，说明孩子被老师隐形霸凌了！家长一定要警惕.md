# 有这4条，说明孩子被老师隐形霸凌了！家长一定要警惕

我们总以为，只有拳头落在身上才叫霸凌。

其实，并不是所有伤害，都带着伤口。

也并不是所有霸凌，都来自同学。

**我发现了一个挺可怕的盲区：**

如果孩子回家带着乌眼青，大家马上会炸毛，会冲到学校讨说法。

**但如果孩子是被老师“隐形霸凌”了，很多家长第一反应经常是：**

 _你是不是太敏感了？_

 _老师批评你也是为了你好，严师出高徒嘛。_

 _你怎么不反思反思，老师为什么不骂别人光骂你？_

说实话，这种来自成年人的精神打压，比挨两下打还要让人绝望。

**因为它大概率是“无痕”的，是披着“教育”外衣的。**

孩子在不知不觉中觉得自己一无是处，甚至觉得我就该被骂。

如果孩子回家有下面这4种情况，大家千万别大意，你的孩子被老师隐形霸凌了。

  


# 01.制造二选一困境，进行高级的道德绑架

大家可能不知道，我的读者里还有些是小学五六年级的孩子。

有个孩子跟我说，他拉肚子憋不住了，举手想上厕所。

老师没有直接拒绝，而是停下讲课，全班鸦雀无声，老师轻飘飘地来了一句：

**“大家都在专心听讲，你现在出去，打断了大家的思路，如果咱们班这次评比输了，算谁的？”**

这种话术，对于一个十几岁、三观还没完全成型的孩子来说，简直是降维打击。

好像只要你迈出教室一步，你就是不注重集体荣誉。

你没法反驳，没法辩解，只能憋着。

身体在受罪，心里很憋屈。

这种“二选一”，本质上就是一种权利的滥用。

**它在给孩子灌输一种毒逻辑：**

**照顾自己的感受是羞耻的，是自私的，没有集体意识。**

然后呢，孩子会变得极度讨好，哪怕受了天大的委屈也不敢吭声，因为他觉得自己不配。

  


# 02.区别对待极其明显，甚至煽动孤立

**如果说二选一是精神施压，那连带责任和双标就是公开侮辱。**

我看到一个孩子的留言，隔着屏幕都能感到那种窒息感。

他说：“我们老师就是那种，一个人的错全班受罚。”

比如一个人作业没交，全班都不许下课，所有人陪着罚站。

这还不算完，老师还要补上一句：

**“大家别怪我啊，要恨就恨某某某，是他害了大家。”**

**这就是故意在学生之间制造仇恨，引导孤立。**

那个犯错的孩子，瞬间就成了全班的公敌。

  


# 03.视而不见，把孩子当空气

**比起被骂，更让人难受的，大概是被无视。**

骂你，至少说明还看见了你；

无视你，是直接从精神上否定了你的存在。

这是一种极度冷暴力的手段。

**有个朋友跟我讲她正在读初二孩子的经历。**

孩子语文其实很好，也想努力表现，每次老师提问他都积极举手，手举得高高的。

但是老师就像看不见他一样，目光直接忽略他，永远只叫那些他喜欢的学生。

孩子不死心，想了很久，决定用“提问”来反抗。

孩子故意从不同角度去解读课文，提出一些深度的、甚至有点争议的问题，想引起老师注意。

结果呢？

老师只是觉得尴尬，敷衍地跳过话题，依然把他当空气......

当一个老师刻意忽略某个学生，不提问、不回应、不眼神接触，这其实是在释放一个信号：

你不重要，你在这个班级里是多余的。

**这对青春期孩子的自尊心打击，是毁灭性的。**

  


#  04.当众嘲笑，进行人格侮辱

这点最常见，也最直接，最伤人。

有些老师，特别喜欢拿“智商”说事儿，以此来彰显自己的优越感。

**“做不出来题的孩子，就是智商低。”**

“这种题我都讲了八百遍了，猫都学会了你还不会。”

还有那种极其不合理的惩罚。

规定一分钟内必须背下一个知识点，背不下来？

打骂、罚站、赶出教室。

“你别听了，反正听了也听不懂。”

人家想学都不让学，天天耀武扬威的。

这哪是教育啊？

说难听点，这就是拿学生当情绪的出气筒。

这种当众的羞辱，会把孩子的自尊心踩在脚底下反复碾压。

可现实中，他们只能默默忍受，甚至开始厌恶学习，厌恶这个老师。

  


**写这些，不是为了制造对立，也不是说所有老师都这样。**

负责任的好老师当然有很多，那是孩子的幸运。

但我也想给各位家长提个醒：

不要神话“老师”这个职业的光环。

**老师也是人，也有人性的弱点，也有情绪失控的时候，甚至也有品行不端的人混杂其中。**

我们能做的，是什么？

是看见。

是接住。

当孩子回家抱怨老师的时候，千万别急着打断，先停下来，听他说完。

**问问具体的细节，感受他的情绪。**

如果是孩子错了，我们引导他改正；

但如果是老师越界了，搞人身攻击、搞隐形霸凌了，

请一定要坚定地站在孩子这一边。

**你要做那个给孩子兜底的人。**
