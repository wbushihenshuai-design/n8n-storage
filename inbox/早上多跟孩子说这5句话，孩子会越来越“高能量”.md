# 早上多跟孩子说这5句话，孩子会越来越“高能量”

咱们先聊个故事。

8岁的爱迪生，被老师贴上“脑子不好使”的标签，直接给退学了。

换谁都觉得这孩子没戏了，街坊邻居更是说闲话。

可他妈妈南希，只是把儿子拉到身边，特别平静地跟他说：

**“宝贝，你比他们所有人都聪明，只是他们还不懂你。”**

就这么一句话，像一束光，一下子照亮了爱迪生的一生。

很多年后，这位拥有2000多项发明的伟大发明家，一提起妈妈就特别感激。

**他说，是妈妈成就了自己。**

妈妈那份坚定不移的信任，让他觉得，只要活着，就不能辜负这份爱。

说真的，**话语的力量，比我们想象中大得多。**

它能一句话把人的自信心打到谷底，也能一句话把人的灵魂重新点亮。

我们经常抱怨，一大早家里就鸡飞狗跳：

**催孩子的、埋怨的、吵架的声音混成一团。**

我们一边用焦虑和催促开启孩子的一天，一边又希望他能精神饱满、积极向上。

这不是很矛盾吗？

真正聪明的父母，都懂得用语言的魔法，给孩子的一天加满阳光。

他们知道，早上说对这5句话，就能叫醒孩子身体里那个“高能量”的小宇宙。

  


# 01.今天，你准备创造一个什么样的小奇迹？

想想我们平时早上是怎么喊孩子的？

  * **“快起床！”**
  * **“赶紧刷牙！”**
  * **“再不走就迟到了！”**



这些话听上去是不是特别催？

其实孩子接收到的信息是：

**“你太磨蹭了，得听我的。”**

但如果我们换个问法：

“今天，你准备创造一个什么样的小奇迹？”

这句话的妙处在于，它悄悄地告诉孩子：

**你很厉害，你是有能力让今天变得特别的创造者。**

我有个朋友就试过。

她儿子以前特不爱上学，每天早上都愁眉苦脸。

她开始这么问他，起初孩子还有点懵。

可没过几天，儿子出门前特兴奋地跟她说：

**“妈妈，我今天的小奇迹，是要教会同桌折一只纸青蛙！”**

你看，就这么一句话，就把“不得不去上学”的苦差事，变成了一场充满未知和期待的好玩探险。

**它把孩子的关注点，从“被逼着做事”拉到了“主动去创造”上。**

  


#  02.我感觉，你的大脑今天又变得更聪明了。

心理学上有个很有意思的“罗森塔尔效应”，说白了就是，**你真心相信一个人会变成什么样，他就真的会朝着那个方向发展。**

这就像一种积极的“预言”。

所以，当我们说“我感觉你的大脑更聪明了”，我们不是在说空话，而是在帮孩子建立一种**“我能行”的信念** ，也就是“成长型思维”——**相信自己的能力不是一成不变的，是可以通过努力变强的** 。

反过来想想，我们是不是总爱翻旧账：

  * **“昨天那道题怎么又忘了？”**
  * **“你记性怎么这么差！”**



这些话就像给孩子贴了“我不行”的标签，把他钉在那儿了。

斯坦福大学心理学教授卡罗尔·德韦克的研究都证明了，**相信自己能变聪明的孩子，遇到困难时更不容易放弃，也更容易成功。**

请一定记住，我们嘴上说出的期待，真的会慢慢变成孩子身上的样子。

  


# 03.告诉我一个让你感到兴奋的想法，无论多小都可以。

**兴奋感，绝对是最高效的能量来源。**

这句话的魔力在哪儿？

**它能让孩子的大脑切换一个频道。**

平时我们问“昨天作业写完了吗？”，大脑马上进入“检查任务”的紧张模式。

但问他有什么兴奋的想法，大脑就会切换到**“寻找快乐”的探索模式** ，去搜寻那些能让他眼睛发光的小事。

有位设计师妈妈，每天早上都和女儿聊这个。

女儿的兴奋点可能只是“楼下的小猫今天冲我叫了”，或者“我想用掉下来的叶子拼一幅画”。

就是这些不起眼的小兴奋，像保护罩一样，护住了孩子宝贵的好奇心和探索欲，让她对这个世界一直充满热情。

**想让孩子的内在小马达转起来，兴奋就是最好的燃料。**

  


#  04.我看见你眼睛里的光了。

这可能是最高级的一种夸奖，我管它叫**“看见”的力量** 。

我们平时夸孩子，总是有条件的：

  * “你考了100分，真棒！”
  * “你帮妈妈做家务了，真乖！”



我们夸的是他的行为和结果。

但“我看见你眼睛里的光了”，夸的是**孩子这个人本身** 。

是你专心拼图时的样子，是你聊起自己喜欢的东西时闪闪发光的眼神，是你做了件善良小事后那份纯粹的心。

**当你这么说时，孩子会感觉到自己被完完整整地、深深地看见了。**

这种感觉比任何奖励都珍贵，它传递的信息是：

**“原来我什么都不做，本身就值得被爱，我的内心世界原来这么棒。”**

这种看见，是一个人能给另一个人最温暖的肯定。

  


# 05\. 去吧，让今天因为你的存在而有所不同。

这句话，是帮孩子找到**“使命感”** 的钥匙。

它比“好好学习，听老师的话”这种要求，要高明太多了。

后者是让孩子当一个听话的“接收者”，而前者，是邀请他成为一个主动的**“贡献者”** 。

它给了孩子一种力量感和价值感：

**“我很重要，我的存在是有意义的。”**

他可以去帮帮同学，可以在课上提个好问题，甚至只是给别人一个微笑，这些都能让世界因为他，而变得更好一点点。

当一个孩子相信自己能给世界带来一点点好的改变时，他的内心就会长出责任感和主动性，这比你催他100遍都有用。

毕竟，教育的终极目的，不是培养一个听话的执行者，而是唤醒一个有担当的创造者。

  


# 写在最后

**爱尔兰诗人叶芝说过：**

**“教育不是注满一桶水，而是点燃一把火。”**

真正懂得能量秘密的父母，会在清晨用几句简单的话，为孩子点燃那一天的火焰：

  * 用**“创造奇迹”** 的期待，代替没完没了的催促。
  * 用**“你更聪明”** 的暗示，给孩子种下自信的种子。
  * 用**“寻找兴奋”** 的邀请，发动他自己的内在小马达。
  * 用**“看见光芒”** 的确认，让他知道你爱的是他本来的样子。
  * 用**“有所不同”** 的嘱托，赋予他改变世界的小小力量。



从明天早上开始，试试吧。

用你的话，去叫醒那个元气满满的孩子。
