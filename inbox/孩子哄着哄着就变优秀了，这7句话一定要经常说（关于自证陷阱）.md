# 孩子哄着哄着就变优秀了，这7句话一定要经常说（关于自证陷阱）

有没有过这种时候？

孩子哭着从学校跑回来，跟你说：“妈妈，同桌说我拿了他的橡皮，可我真的没拿！”

你心疼坏了，鼓励他去跟同学解释清楚。

结果呢，孩子越是急着说“不是我！我没有！”，大家看他的眼神就越奇怪，最后被孤立了，真是跳进黄河也洗不清。

这种情况，其实就是孩子社交里特别让人头疼的**“自证陷阱”** 。

说白了就是，一旦有人给你贴了个不好的标签，你越是拼命想撕掉它，反而越容易让别人觉得“你看，他急了，肯定是他干的”。

 _**因为有时候，那些怀疑你的人根本不在乎真相是什么，他们就是想找点蛛丝马迹来证明自己的想法是对的**_ 。

查理·芒格就说过一句特别有智慧的话：**放弃向他人证明自己** 。

有人问他，那要怎么做到呢？

他说： _**“学会专注忘我地去做你想做的事，心无旁骛地解决问题。”**_

下面这7句话，试着在平时聊天时教给孩子，帮他摆脱自证陷阱，以后再遇到误会，就能自己轻松搞定。

  


# 1、谁主张谁举证，证据在哪？

当孩子被人冤枉，急得脸都红了。

别再只教他翻来覆去地说“我没有”了，那太无力了。

**你可以这样跟他说：**

宝贝，我知道你很委屈，但咱们先不着急。

你心里要记住一个特别简单的道理：

 _**谁说你做了，就该由谁拿出证据来。**_

你不用去证明自己“没做过”什么，这太难了。

下次再有同学指着你说事，你就看着他，平心静气地问一句：

**“你这么说，有证据吗？”**

这句话是在教孩子最基本的**讲道理和守规矩的思维** 。

让他明白，说话要有根据，不是谁嗓门大谁就有理。

这会是他以后理性思考的开始。

  


# 2、别用“我”开头，多用“你”开头

当孩子被人说“你真自私”“你就是故意的”。

别让他掉进“我才不是……”这种吵不赢的坑里。

**试着这样教他：**

宝贝，咱们换个说法。

下次别急着反驳“我没有！”，咱们试试把话头抛给对方，就好像很好奇一样问他：

 _**“你为什么会这么想我呢？”**_

你看， _**把“我怎么怎么样”的解释，换成对“你怎么怎么样”的提问，主动权一下就回到你手里了**_ 。

这是一种很聪明的**说话小技巧** 。

它能让对方一下子 _**从发脾气模式，切换到讲道理模式**_ ，给了大家一个解决问题的机会，而不是让火药味越来越浓。

  


# 3、你这么清楚，是不是经常做？

当有人添油加醋地传孩子的谣言，说得跟真事儿一样。

有时候，一句巧妙的反问，比100句解释都管用。

**你可以这样给孩子支招：**

宝贝，如果有人把一件你根本没做过的事，说得有鼻子有眼的，好像他亲眼看见了一样。

你可以假装很惊讶地看着他，问一句：

 _**“哇，这事儿的细节你想得这么明白，你是不是自己就经常这么干呀？”**_

这是在教孩子一种**带点小智慧的“反击”** 。

它在暗示对方，别拿自己的小心思去揣测别人。

通常这句话一出，传闲话的人自己就先心虚了，旁边听着的人也能品出点味儿来。

  


# 4、你用这种方法冤枉过多少人？

如果孩子总是被同一个同学反复找茬、冤枉。

教他别老陷在这一件事里，要看看这个同学本身是不是就有问题。

**可以这样启发他：**

宝贝，如果老是那个人来烦你，下次你可以很认真地看着他的眼睛，问他：

“我发现你很喜欢在没搞清楚状况的时候就指责别人。”

 _**“我就想问问，你用这个方法冤枉过几个人了？”**_

这是在教孩子**把眼光放长远一点** 。

这句话直接点出了对方的行为模式有问题，而不是纠结于橡皮还是铅笔这点小事。

它等于在告诉对方：

你这样是不对的。

从根儿上削弱了他说话的分量。

  


# 5、你的认可对我有什么意义？

当孩子因为别人的一句坏话就难过半天，开始怀疑自己时。

这是帮他内心变强大的最好机会。

**你可以这样告诉他：**

宝贝，你要记住，你不可能让所有人都喜欢你。

如果有人总是故意说你不好，让你不开心，你就在心里悄悄问自己一个问题：

 _**“他喜不喜欢我，对我就那么重要吗？”**_

其实，一点也不重要。

我们自己是什么样的人，由我们自己说了算。

这句话是帮孩子**建立心理上的“防护罩”和稳稳的自信** 。

让他明白，不是谁的评价都要在意。

学会过滤掉那些不好的声音，是内心强大的必修课。

  


# 6、你急着要答案，是不是承受不了不确定？

当场面很乱，有人咄咄逼人地非要孩子立刻给个说法，让他压力山大时。

教他看穿对方着急背后的不安，一下就能镇定下来。

**告诉他可以这样做：**

宝贝，如果有人特别着急地逼你解释，你可以故意放慢速度，慢悠悠地问他：

 _**“你好像特别着急啊，是不是很害怕事情搞不清楚？”**_

这是在教孩子**看懂别人的情绪** 。

当你发现对方比你还慌的时候，你就不会被他的节奏带跑了。

你的冷静，就是你最好的武器。

  


# 7、自证的路没有尽头

当你和孩子都努力解释了，但对方就是不听不信，油盐不进时。

告诉孩子，有时候，聪明地“放弃”，才是一种真正的厉害。

**你可以这样对他说，也让他学会这样表达：**

 _**宝贝，一定记住，自证这条路是没有尽头的**_ 。

你永远叫不醒一个装睡的人。

这时候，你可以平静地告诉他：

 _**“我想说的都说完了。”**_

 _**“我不会再为这件事浪费我的时间了。”**_

然后，潇洒地走开。

这是在教孩子**“到此为止”的智慧** 。

让他明白，我们改变不了所有人的看法。

**当沟通没用时，保护好自己的心情和精力，选择转身离开，是更有力量的表现。**

  


#  8、几个关键的小提醒

  * **在家多演练** ：平时在家，可以跟孩子玩角色扮演的游戏，你当那个“坏同学”，让他来练习说这些话。在安全的环境里练熟了，真遇到事儿的时候才能脱口而出。

  * **爸妈永远是你最坚实的后盾** ：一定要告诉孩子：“不管发生什么，爸爸妈妈永远相信你，支持你。”这份底气，是他面对外面风雨的最大力量。

  * **先拥抱，再讲道理** ：孩子委屈地回来时，第一件事不是教他怎么做，而是先给他一个大大的拥抱，让他把心里的不开心都说出来。等情绪平复了，道理才能听得进去。




  


让孩子明白，真正的强大，不是费尽口舌去向世界证明“我很好”，而是当世界误解你时，你依然有底气说：“随便你怎么想，我不在乎”，然后继续走自己的路。
