# 陪孩子写作业4忌（附正确建议）

晚上辅导作业，像渡劫。

**我们嘴上说的是题，心里急的是他的未来。**

**其实，孩子的问题，常常是我们“陪”出来的。**

很多时候，我们只需要管住自己，问题就解决了一大半。

  


# 01 不要直接上手帮忙

孩子一卡壳，你就忍不住冲上去，讲思路、给答案。

你帮他一次，他下次还找你。

你成了他的“外挂”，他自己的脑子就懒得转了。

**试试这样做：**

**管住手，只动嘴。**

把笔还给他，然后问三个问题：

1.“你先跟我说说，这道题让你做什么？”——**确保他读懂了题** 。

2.“那你打算第一步做什么？”——**引导他启动思考** 。

3.“如果这个方法行不通，还有别的路可以走吗？”——**拓宽他的思路** 。

你的任务不是给答案，是帮他把混乱的思路理清楚。

  


# 02 不要一直在旁边指手画脚

你像个监工，他写一笔，你扫一眼。

字丑了、姿势不对了，立刻打断。

**他一半精力在做题，一半精力在防着你，学习效率能高吗？**

**试试这样做：**

物理隔离，保持一臂以上的距离。

你坐远点，看自己的书或处理工作。

**然后跟他约定一个规则：**

“你先自己写，写完喊我。遇到实在不会的题，先在旁边画个圈，跳过去，我们最后一起看。”

**这个小小的距离和约定，给的是信任，换来的是他的专注。**

  


#  03 不要一犯错就严厉指责

**看到同样的错误重复犯，你的火气比谁都大。**

但你一发火，他脑子就一片空白了，光害怕了，根本听不进你讲的道理。

你发泄了情绪，却错过了帮他进步的最好时机。

**试试这样做：**

**看到错题，先让自己冷静三秒。**

然后指着错题，心平气和地对他说：

“来，我们看看这个题错在哪了。把它弄明白，下次就不会再掉进同一个坑里了。”

**准备一个“错题本”，把典型的错题记下来。**

**你的目的不是批评，是和他结成“队友”，一起消灭问题。**

  


#  04 别在旁边不停地催

**“快点”这两个字，是效率最低的词。**

你越催，他越焦虑，越容易出错，最后还得花更多时间改，得不偿失。

**试试这样做：**

**用“番茄工作法”代替你的嘴。**

买一个番茄钟，或者用手机定时。

“我们定25分钟，这段时间你专心写，我也专心做我的事，谁也别打扰谁，怎么样？”

时间一到，让他休息5分钟，喝水、上厕所、看看窗外。

把时间管理的主动权还给他，他才能真正学会对自己负责。

  


# 05 写在最后

说了这么多“不要”，那到底要怎么做？

其实，最好的陪伴，是做一个**“陪跑员”** 。

运动员在场上，教练不能替他跑，但会在场边递水、喊加油、分析战术。

我们陪写作业，也是如此：

  * **给方法，不给答案。** （引导他思考）
  * **给空间，不给监视。** （让他能专注）
  * **给鼓励，不给指责。** （允许他犯错）
  * **给节奏，不给催促。** （培养他的自主性）



**教育家苏霍姆林斯基说过：“让学生体验到一种理智的、成功的快乐，乃是教育的奥秘。”**

陪写作业的目的，不是为了交出一份完美的作业，而是为了让孩子在这个过程中，学会独立思考，建立自信，并最终能享受学习的乐趣。

记住，总有一天，他需要独自面对那些难题。

**而你的耐心，会是他独立思考的底气。**
