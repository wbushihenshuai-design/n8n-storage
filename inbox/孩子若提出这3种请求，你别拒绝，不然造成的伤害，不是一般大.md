# 孩子若提出这3种请求，你别拒绝，不然造成的伤害，不是一般大

朋友聚会，一个妈妈满脸疲惫地抱怨：

**“我家孩子现在什么都跟我说，芝麻大的事都要汇报一遍，快把我听烦了。”**

我看着她，轻轻说了一句：

**“他不是在汇报。他是在一次次地，把他的整个世界，小心翼翼地捧给你看。”**

空气安静了。

我们总觉得孩子的请求是麻烦，是打扰，是他们不懂事。

我们很少蹲下来想一想，每一次开口，孩子可能都在用尽全力，向我们发出一个信号。

你听懂了，就接住了他。

你错过了，那扇通往他内心的小门，可能就永远关上了。

**有3种请求，你每一次的拒绝，对孩子来说都是一种无形的伤害。**

  


#  第一种：分享式请求“妈妈，你快来看！”

有一次我带孩子在邻居家玩，邻居家的男孩，兴冲冲地从房间跑出来，手里举着刚搭好的积木城堡。

**“妈妈，你快来看我的飞船基地！”**

**他妈妈正窝在沙发里看手机，头也没抬，敷衍了一句：**

“嗯嗯，看到了，真棒。”

男孩眼里的光，明显地暗了下去。

他站在原地，举着他的“飞船基地”，像一个不知所措的雕塑。

过了几秒，他默默地走回房间，我们听到了积木哗啦一声散架的声音。

他不是在求你夸奖那几块塑料。

他是在问你：

“妈妈，我创造出来的东西，在你眼里有价值吗？我脑子里的奇思妙想，你感兴趣吗？”

**你的漫不经心，翻译成孩子能懂的语言就是：“你的世界，我不关心。”**

当一个孩子停止向你分享，不是他长大了，懂事了。

是他彻底明白了，你对他的世界，没有兴趣。

这扇门一旦关上，再想打开，就难了。

  


# 第二种：确认式请求“我做得好不好？”

我见过一个女孩，小心翼翼地把自己的鞋子在鞋柜里摆得整整齐齐，然后跑到厨房门口，小声问：

**“妈妈，你看我把鞋放好了，可以吗？”**

她妈妈正忙着切菜，回头看了一眼，皱着眉说：

**“另一只歪了点，没看见吗？”**

女孩的脸瞬间就红了，赶紧跑回去，把那只鞋挪了又挪。

她真的在乎那只鞋歪了几毫米吗？

不是的。

**她是在用自己微小的努力，向你寻求一种确认。**

她想听到的是：

**“我看见了你的努力，你做得很好。”**

**这种确认，是在帮他建立内心的秩序感和价值感。**

  


你每一次不耐烦的挑剔，都在告诉他：

**“你做得不够好，你永远都达不到我的标准。”**

慢慢地，他就不再问你了。

他会开始怀疑自己做任何事的价值，变成一个做什么都畏首畏尾、极度需要看别人脸色的人。

**你今天吝啬的一句肯定，** 会变成他未来几十年里，内心深处那个永远在苛责自己的声音。

  


# 第三种：试错式请求“让我自己来，行不行？”

楼下的孩子，刚学会走路没多久，看到妈妈在扫地，也抢着要去拿扫帚。

妈妈一把夺过来：

**“你别动，净给我添乱，地越扫越脏。”**

孩子“哇”地一声就哭了。

他不是在无理取闹。

他是在用行动告诉你：

**“我想探索这个世界，我想知道我自己的能力边界在哪里。”**

我们害怕他把牛奶洒得到处都是，害怕他把衣服穿反，害怕他摔跤。

**我们用“保护”的名义，圈养了一个“无能”的孩子。**

你替他扫了地，他可能就失去了学习劳动的兴趣。

你替他穿了衣服，他可能就失去了发展自理能力的机会。

你怕他摔跤，把他抱在怀里，他可能就永远失去了奔向远方的勇气。

你现在省下的一分钟，未来可能要花十年，去弥补他那颗“我不行”的心。

  


# 你是孩子人生的第一面镜子

**社会心理学里有一个概念，叫“镜中自我理论”（Looking-glass self）。**

**意思是，我们每个人对自己的认知，很大程度上，是通过观察别人对我们的反应来形成的。**

**我们从别人的“镜子”里，看清自己是谁。**

对一个孩子来说，父母，就是他人生的第一面，也是最重要的一面镜子。

他举着积木向你跑来，是在镜子前问：**“我有趣吗？”**

他摆好鞋子来问你，是在镜子前问：**“我能干吗？”**

他想抢你的扫帚，是在镜子前问：**“我独立吗？”**

镜子是不会说话的。

**你的眼神、你的语气、你随口的一句话，就是镜子给他的回应。**

**你给他的回应，定义了他。**

  


很多年后，我们看着那个不爱表达、缺乏自信、不敢尝试的孩子，心里纳闷：**“他怎么会变成这样？”**

或许该回过头去想一想。

**在他童年那几年，在他一次次，满怀期待地跑到我们这面“镜子”前时，我们，究竟让他看到了什么？**
