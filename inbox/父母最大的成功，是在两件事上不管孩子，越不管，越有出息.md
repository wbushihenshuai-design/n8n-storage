# 父母最大的成功，是在两件事上不管孩子，越不管，越有出息

我以前做项目管理，最怕遇到一种合作方。

你跟他提需求，他说**“我懂”** 。

你给他看方案，他说**“我明白”** 。

然后，他就不停地给你发消息，问你进度，问你细节，甚至直接上手帮你改。

你写个文案，他觉得这个词不好；

你做个设计，他觉得那个颜色不对。

**美其名曰“帮忙”，说白了，就是不信任。**

他不是在帮你完成工作，他是在缓解自己的焦虑。

他忍受不了一丁点儿事情脱离自己的掌控。

这种人，你跟他合作会累死。

**因为他要的不是一个结果，他要的是一种“一切尽在掌握”的快感。**

  


**遗憾的是，这种人，真的太多太多了。**

**他会把自己的焦虑，变成对你的无尽索取。**

后来我发现，生活中，不是每个人都有机会去管一个项目，去控制一个团队的。

**但是，几乎人人都有一个机会，可以名正言顺地去“掌控”另一个人的人生。**

**那就是，当父母。**

当了父母，那种“一切要尽在掌握”的控制欲，就找到了最佳宣泄口。

它不需要你竞聘上岗，不需要你做出业绩，甚至不需要你讲道理。

**你生了他，这个权力就天然属于你。**

于是，那些用控制欲来缓解自身焦虑的人，当了父母，就成了一场灾难的开始。

他们会把手伸向孩子的两个领域，而这两个领域，一旦被管了，孩子这辈子基本就废了。

  


# 01、孩子那些“没用”的爱好。

**你会发现，很多家长允许孩子有爱好，但前提是，这个爱好得“有用”。**

弹钢琴可以，因为能考级，能锻炼气质。

学编程可以，因为那是未来趋势，是加分项。

但如果你的孩子，就喜欢蹲在地上看一整天蚂蚁搬家，喜欢在泥巴坑里玩一下午，喜欢拆家里所有能拆的东西......

家长的焦虑就上来了：

“这有什么用？”

**“有用”，成了悬在所有爱好头上的一把刀。**

  


**我告诉你有什么用。**

一个孩子，愿意投入时间，沉浸在一件纯粹因为喜欢、而不是因为“有用”的事情里，这东西叫**内驱力** 。

这是一个人精神世界的发动机，是他以后对抗所有人生无聊和虚无的燃料。

你现在用“有用”这把刀，砍掉了他所有的枝枝蔓芽，你觉得你帮他修剪出了一条通往成功的康庄大道。

其实你只是，亲手熄灭了他精神世界里所有的火花。

等他长大了，你又会反过来问他：

**“你怎么对什么都提不起兴趣？”**

废话，那个能燃起兴趣的发动机，不是早就被你拆掉了吗？

  


# 02、孩子“搞砸”一件事情的权利。

你有没有见过一种家长，孩子在学校跟同学闹了点小别扭，他第二天就冲到学校找老师、找对方家长。

孩子做个手工，做得歪歪扭扭，他一把抢过来说：**“我来帮你。”**

孩子考试考砸了，他比孩子还焦虑，到处找补习班，恨不得自己替他去学。

他们嘴上说着“为你好”，心里想的其实是：

“你不能失败，因为你的失败，会引发我的焦虑。”

**他们见不得孩子“搞砸”。**

这种“保护”，说白了就是**剥夺** 。

你剥夺了他体验真实挫败感的机会，就等于剥夺了他形成**“心理免疫系统”** 的机会。

一个人，从小没经历过自己摆平烂摊子，没体验过那种“虽然搞砸了，但天也没塌下来”的感觉，他心里那根弦，就一直是紧绷的。

**他会极度害怕犯错，极度害怕失败。**

  


等他真正进入社会，随便一个浪打过来，他就散架了。

因为他从小到大，都被罩在一个无菌的罩子里，他以为世界是无菌的。

你帮他搞定了一百次小麻烦，其实是为他未来第一百零一次的大崩溃，挖好了坑。

所以你看，很多育儿方法，教你怎么培养习惯，怎么高效学习，怎么规划人生。

这些东西，我现在觉得都是次要的。

**一个孩子的精神内核，才是根本。**

**一个有内驱力的孩子，和一个抗挫力强的孩子，他做什么事，都不会太差。**

而这两样东西，恰恰不是“管”出来的，是“不管”出来的。

你给他空间，让他去爱那些没用的事，他的火就不会灭。

你给他权利，让他去搞砸一些可控的事，他的心就不会脆。

说到底，很多父母焦虑的，压根不是孩子未来过得好不好。

**而是“孩子没有按照我设定的路线走”这件事本身。**

**你以为你在给他规划一条万无一失的路，其实你只是在满足自己对失控的恐惧。**
