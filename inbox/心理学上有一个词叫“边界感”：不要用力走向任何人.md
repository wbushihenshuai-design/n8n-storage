# 心理学上有一个词叫“边界感”：不要用力走向任何人

美国心理治疗师内德拉·格洛佛·塔瓦布发现一个普遍现象：

许多前来求助的人，无论是心力交瘁、自责不安，还是哀怨苦恼，其问题的根源往往都指向同一个地方。

**她会温和地告诉他们：“你的问题，出在界限上。”**

**界限的本质，是对人生所有权的认知。**

在所有关系中，亲子关系里的界限模糊，最容易酿成一场以爱为名的“灾难”。

很多时候，我们总会心里越位，毫无边界地，用力地想走向那个我们深爱的孩子。

然而，我写这篇文章想告诉你一个可能有点刺耳的真相。

  


# 1.你不是爱得太满，你是“自我”太稀薄

承认吧，你对孩子生活的过度参与，恰恰是你对自己人生的“缺席”。

当你的全部心神都用来规划他的路线，纠正他的姿势，预判他的风险时，你自己的那条人生路，早已荒草丛生。

你把孩子当成了你人生唯一的项目、唯一的作品、唯一的精神寄托。

你以为你在倾尽全力地爱，其实你只是在不计后果地填补自己的“空”。

这份“空”，源于你对自我价值的不确定，源于你未曾实现的梦想，源于你对失控的恐惧。

于是，孩子成了你人生的延续，一个修复遗憾的工具。

**你的孩子，不是你人生的续集，更不是你修复遗憾的工具人。**

**当一个家长开始24小时紧盯孩子时，他自己的人生，其实已经停摆了。**

  


#  2.你正在养出一个“出租型人格”的孩子

你毫无保留的“付出”，正在塑造一个怎样的孩子？

心理学上有一种描述叫**“出租型人格”** 。

就是说， _**这个孩子从小就把自己的“人生所有权”出租给了父母。**_

他的人生不是由“我想要”驱动，而是由“我爸妈想要”驱动；

他的喜怒哀乐，不是发自内心，而是为了迎合父母的期待而进行的表演。

这种“出租”的后果，是灾难性的。

**在校期间：**

他可能是个完美的“假人”。

成绩优异，听话懂事，但你若仔细看他的眼睛，会发现那里是空的。

他不知道为何而学，也不知道学的这一切最终要带他去向哪里。

  


**进入社会：**

真正的风暴开始。

他不知道自己想做什么工作，爱什么样的人，过怎样的生活，因为他从未为自己活过。

 _**他极易被外界操控，陷入一段又一段的讨好型关系；**_

 _**或者，在获得自由后彻底失控，用报复性的堕落来宣告一场迟到的独立。**_

你替他选了所有的路，他就失去了所有的方向。

很多时候，孩子的叛逆不是他变坏了。

**叛逆，是那个被你压抑已久的、真正的“我”，在发出最后的求救信号。**

  


#  3.练习“课题分离”，归还“人生主权”

**如何跳出这个“用力过猛”的怪圈？**

答案是两个字：**放手** 。

具体的方法，叫**“课题分离”** 。

这是阿德勒心理学的核心概念，意思是， _**要清晰地分辨什么是“你的课题”，什么是“别人的课题”**_ 。

学习、交友、选择未来，这是孩子的课题。

你只能提供支持和建议，但不能替他做决定，更不能替他承担后果。

**而你的课题是：**

 _**管理好自己的焦虑，经营好自己的生活，成为一个独立而完整的自己。**_

当你不再把孩子的成功等同于你人生的成功时，你才能真正地松开手。

**边界感的本质，就是归还每个人的人生主权，包括你自己。**

  


#  4.好的亲子关系，是“磁铁同极”：我很好，你随意

很多人认为，好的关系是“异极相吸”，彼此互补，紧密无间。

但在健康的亲子关系里，这恰恰是陷阱。

 _**一方拼命吸附，另一方无法逃离，最终紧紧捆绑，毫无生气。**_

真正好的亲子关系，更像是**“磁铁的同极”** 。

你们是两个各自独立、拥有强大内核和能量场的个体。

正因为你们都足够好、足够完整，你们才能在互不消耗的情况下，彼此吸引，彼此欣赏，彼此推动。

  


 _**想象一下那个画面：**_

你不再追着他问：“今天在学校怎么样？”而是专注于自己的工作或爱好。他反而会兴冲冲地跑来跟你分享，因为他把你当成一个可以平等交流的“朋友”，而不是一个时刻监视他的“审查官”。

他遇到困难，第一个想到你，不是因为你能替他解决一切，而是因为他知道，你会无条件地支持他自己去“战斗”，并成为他最坚实的后盾。

**这才是充满生命力的关系：各自闪耀，又能相互照亮。**

  


#  5.写在最后

“不要用力走向任何人”，这句话其实还有至关重要的后半句，那就是——**“先用力地走向你自己”** 。

与其耗费心力去雕琢一个孩子，不如倾尽全力去经营那个被你忽略了太久的自己。

 _**去学习，去成长，去交友，去爱你所爱。**_

当你成为一个有趣、充实、独立的人时，你会发现，那个你深爱的孩子，会带着好奇和欣赏，愿意与这样的你分享更多。

**最好的教育，可能就是让孩子看到：一个人可以如何精彩地活着。**
