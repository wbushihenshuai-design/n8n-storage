# 孩子哄着哄着就优秀了，这5句话一定要经常说（关于抗挫力）

当孩子学习遇到困难或考试成绩不满意时，作为父母，我们都想在这个时候教给孩子点什么，生怕他下次再犯。

然而事实上，当孩子遭遇挫折时，比考了多少分更重要的，是一种跌倒后如何重新振作的能力——**“抗挫力”（Resilience）。**

**美国心理学会（APA）** 将抗挫力定义为：

**“面对逆境、创伤、悲剧、威胁或重大压力源时，能够良好适应的过程。”**

在这个AI飞速发展、充满不确定性的时代，孩子未来面临的挑战远比我们想象的复杂。

成绩只是一时的，而这种**“摔倒了能自愈，失败了能复盘”** 的心理底色，才是孩子们在未来很长一段时间内需要不断学习的一项非常重要的技能。

  


你会发现，真正值得我们花更多心思去关注的，**只有两件事** ：

**1.情绪的韧性：**

**当孩子面对一道怎么都解不出来的难题、或者一张不如意的试卷时，他的第一反应是什么？**

他是会崩溃大哭、自我否定，还是能慢慢让自己平静下来、甚至安慰自己“这次没搞定没关系”？

**2.行动的逻辑：**

**犯错之后，他是怎么思考和行动的？**

是把试卷塞进抽屉再也不看，还是愿意坐下来复盘错题、找出规律、调整方法，甚至主动找老师和同学请教？

那么，如何才能培养出具备这两项能力的孩子？

**答案其实很简单，它就藏在我们日常的回应里。**

父母随口说出的话，往往会内化成孩子内心的声音。

与其焦虑指责，不如试着换一种沟通方式。

试试把那些**“正确却伤人”** 的话，换成这5句能帮孩子“开窍”的魔力话术。

**你会发现，内心强大的孩子，往往是“哄”出来的。**

  


#  01

**你想从哪里重新开始？**

**关键点：** 把选择权还给孩子。

当孩子失败时，他内心最大的感受其实是**“失控感”** 。

觉得自己不行，控制不了局面。

这时候，如果你直接告诉他“你应该先做A，再做B”，其实是在剥夺他找回控制感的机会。

**怎么说更好：**

当孩子看着不理想的成绩单，垂头丧气甚至想要放弃时，不要强迫他马上全部改正，试着温柔地问他：

> ❝
> 
> **宝贝，如果要攻克这张试卷，你想从哪里重新开始？**
> 
> **是先解决这几道因为马虎做错的计算题，还是先看看最后这道大题？**

**重拾自信的第一步，是“做出选择”。**

哪怕只是一个微小的决定，也能让孩子意识到**他对这件事是有掌控权的。**

一旦他做出了选择，由于是自己的决定，他会更有动力去执行，那种因失败而产生的无力感就会慢慢消散。

  


# 02

**你前面这部分做得很好，接下来我们可以怎么做？**

**关键点：** 肯定过程，而非结果。

很多家长习惯盯着孩子的错误看，觉得错了就是错了，有什么好夸的。

但这种思维方式会瞬间扼杀孩子的动力。

**怎么说更好：**

比如，当孩子做数学题卡住时，不要说**“这道题我教过你多少次了？”** ，而是说：

> ❝
> 
> **我看了一下，你前面的公式列得很对，思路也是清晰的，这一步做得非常棒！**
> 
> **那接下来，我们看看是不是计算哪里出了小问题？**

重点就是，**要把对孩子天赋的赞美，转移到对努力和策略的赞美上** 。

肯定过程，就是告诉孩子，你的努力我看在眼里，错误只是一个小插曲，并不代表你全盘皆输。

**这种“被看见”的感觉，是孩子继续尝试的动力。**

  


#  03

**刚才最难的地方在哪里？**

**关键点：** 把性格归因转为事实归因。

我们最容易犯的错误，就是把失败上升到性格层面。

比如：

**“你就是太马虎了”、“你就是没长脑子”。**

这些话像钉子一样，会把“我不行”的标签钉在孩子身上。

**怎么说更好：**

像个队友一样坐在他身边，好奇地问：

> ❝
> 
> **刚才把你难住的，是哪个环节？**
> 
> **是时间不够用了，还是这个单词不认识？**

这叫**问题外化** 。

我们要帮孩子把**“我这个人有问题”** 的想法，剥离成**“我遇到的这件事有个难点”** 。

只要找到了具体的难点，问题就从“无法改变的性格缺陷”，变成了“可以解决的具体障碍”。

**只要是障碍，就可以跨越。**

  


#  04

**你觉得下次用什么方法会更有用？**

**关键点：** 培养孩子“试错”的思维

在人工智能时代，死记硬背已经不再是核心竞争力。

**持续思考、不断假设、验证、修正** 的能力，才是未来的稀缺资源。

**怎么说更好：**

当孩子因为忘记带作业被老师批评时，不要仅仅责骂他丢三落四，而是问：

> ❝
> 
> **看来只靠脑子记这个方法不太灵，那你觉得下次换个什么方法能帮我们不再忘记？**

这句话给孩子传递一个简单的逻辑：

**既然A方法不行，那我们就试B方法。**

这种像科学家做实验一样的冷静态度，能让孩子在面对未来的复杂问题时，不情绪化，而是专注于寻找解决方案。

  


# 05

**失败是个好机会，它在提醒我们要换个方式了。**

**关键点：** 帮孩子重构对失败的认知。

很多孩子都有“完美主义”倾向，认为犯错是羞耻的。

我们要做的，是帮孩子给“失败”去污名化。

**怎么说更好：**

当孩子比赛输了，抱抱他，告诉他：

> ❝
> 
> **没关系，输了其实是个好机会。**
> 
> **它像个信号灯，在提醒我们要调整一下策略了。**
> 
> **正是因为这次跌倒，我们才知道哪里路不平，下次就能跑得更稳了。**

阿德勒说：**“决定我们自己的不是经验本身，而是我们赋予经验的意义。”**

如果你把失败定义为“羞耻”，孩子就会逃避挑战；

如果你把失败定义为“成长的台阶”，孩子就会越挫越勇。

**缩短孩子在挫折中情绪低落的时间，关键在于父母如何定义这次失败。**

  


#  06

**你的表情比语言更重要**

心理学研究表明，在人际沟通中，**文字内容只占信息传递的7%，而语调占38%，面部表情和肢体语言占55%。**

换句话说，哪怕你把上面这5句话背得滚瓜烂熟，如果你是板着脸、皱着眉、语调不耐烦地说的，孩子接收到的信号依然是：

**“爸爸妈妈在嫌弃我。”**

**“我又惹他们生气了。”**

那一刻，他的大脑皮层会进入防御状态，思考能力瞬间停滞，你说什么他都听不进去。

所以，在开口前，请先深呼吸。

**把你的眼神放柔和，把语速降下来。**

蹲下来，看着孩子的眼睛。

只有当孩子在你的表情里读到了“接纳”和“安全”，你的话语才能真正变成滋养他抗挫力的养分。

  


# 07

**孩子跌倒了，其实是成长的契机。**

很多时候，我们急着把孩子拉起来，是因为我们自己无法忍受看到孩子受苦的焦虑。

但请记得，**挫折本身不会伤害孩子，孤立无援的挫折才会。**

趁着孩子还在我们身边，趁着这些“失败”还只是学习上遇到一点困难、考砸了一次小测试这种“低成本试错”的时候，多陪他练习这5句话吧。

不要抱怨教孩子处理情绪很累。

终有一天，他要独自面对这个世界的风风雨雨。

到那时，他内心那个“跌倒了能自己爬起来，还要拍拍尘土笑着继续跑”的**稳定内核** ，就是你留给他最好的礼物。

**父母能给孩子最好的爱，不是为他扫清所有障碍，而是给他一颗越挫越勇的心。**
