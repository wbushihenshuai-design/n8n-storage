# 抑郁的孩子越来越多，这7句话，宁愿烂在肚子里，也别对孩子说了

>   
> 
> 
> **马歇尔·卢森堡博士在《非暴力沟通》书中指出："言语上的指责、嘲讽、否定、说教以及任意打断、拒不回应、随意出口的评价和结论给我们带来情感和精神上的创伤，甚至比肉体的伤害更令人痛苦。"**

这段话，是不是让你心头一颤？

看看我们身边正在发生的事：

  * **孩子的笑容越来越少；**
  * **"我好累"成了他们的口头禅；**
  * **心理咨询室里，稚嫩的面孔越来越多......**



当我们批评孩子时，往往是在表达自己的焦虑和期待，却忽视了孩子真正的感受。

但你有没有想过，那些脱口而出的批评，可能正在一步步摧毁孩子的自信。

以下7句话，宁可烂在肚子里，也别对孩子说：

  


# 1、你怎么这么没用？

这句话就像一把利刃，直接刺向孩子最柔软的内心。

当孩子听到这句话时，他们接收到的信息是： **"我这个人没有价值"。**

这种全盘否定会让孩子产生深深的无力感和绝望感。

**心理学研究表明，自我价值感的缺失是抑郁症的重要诱因之一。**

孩子会开始相信自己真的"没用"，进而放弃努力，陷入恶性循环。

**记住：批评行为，而不是否定人格。**

**可以说"这件事没做好"，但绝不能说"你这个人不行"。**

  


#  2、别人家的孩子都比你强！

**这句话的杀伤力在于：让孩子永远活在比较的阴影里。**

**"别人家的孩子"**  就像一座永远无法翻越的大山，压得孩子喘不过气。

无论孩子多么努力，似乎都达不到父母的期待。

长期的比较会让孩子产生习得性无助，觉得"反正我怎么做都不如别人"。

这种想法一旦形成，孩子就会失去前进的动力，自卑感如影随形。

每个孩子都是独特的，看见并肯定孩子的闪光点，比任何比较都更有力量。

  


# 3、累死累活赚钱就是为了供你上学！

**这句话把亲情变成了一笔交易。**

孩子会觉得自己是家庭的 **"负债"，**  而不是被爱的存在。

这种沉重的负罪感会让孩子觉得自己的存在给家人带来了痛苦。

**"我是不是不该出生？"这样的想法会在孩子心中生根发芽。**

当孩子把自己定位为负担时，他们会失去安全感，觉得自己不配被爱。

**爱应该是无条件的，而不是附带账单的。**

  


#  4、要不是为了你，早就和你爸离婚了！

**这句话让孩子成为了家庭矛盾的"背锅侠"。**

孩子会认为： **"都是因为我，爸妈才不幸福"。**

这种错误的归因会让孩子背负巨大的心理包袱。

他们可能会想： **"如果没有我，爸妈就会幸福了吧？"**

这种自责和内疚是抑郁症的温床，会严重影响孩子的心理健康。

大人的问题，永远不该让孩子来承担责任。

  


# 5、别哭了，有什么好哭的？

**这句话在告诉孩子：你的情绪不重要，也不被接纳。**

哭泣是情绪的自然出口，强行堵住这个出口，情绪就会在内心发酵。

当孩子学会压抑情绪，他们也失去了处理情绪的能力。

被压抑的情绪不会消失，只会以更糟糕的方式爆发。

长期的情绪压抑是抑郁症的重要成因之一。

接纳孩子的眼泪，就是接纳他们的心灵。

  


# 6、流眼泪多丢人，眼泪能解决问题吗？

**这句话传递的是：表达脆弱是无能的表现。**

孩子会学会把真实的情感藏起来，戴上 **"我很好"**  的面具。

当哭泣被贴上 **"没用"**  的标签，孩子就失去了情绪释放的正当性。

他们会觉得："连哭都不配，我还能做什么？"

这种无助感会让孩子越来越封闭自己。

其实，能够哭出来的孩子，往往比憋在心里的孩子更健康。

  


# 7、流眼泪会让别的小朋友看不起你！

**这句话用社会评价绑架了孩子的情绪表达。**

孩子会开始害怕在他人面前展现真实的自己。

**"我必须坚强"**  成了一种枷锁，而不是选择。

过度在意他人眼光，会让孩子活得越来越累。

当孩子连在家人面前都不敢真实，他们的内心该有多孤独？

这种情感隔离正是抑郁症患者的典型特征。

  


# 写在最后：

这7句话，每一句都可能成为压垮孩子的最后一根稻草。

记住，孩子需要的不是完美的父母，而是懂得尊重和接纳的父母。 当我们意识到这些话的杀伤力，就该把它们永远封存起来。

下次想说这些话时，不妨先问问自己：

**"我是在表达爱，还是在发泄情绪？"**

那些被爱的语言滋养的孩子，才有力量面对生活的风雨。

而那些伤人的话，就让它们永远烂在肚子里吧。

保护孩子的心灵，从改变我们的语言开始。
