# 一个家庭，最可怕的不是没钱，也不是争吵，而是父母张口闭口说的这3句话

如果你为人父母，想知道哪些话会不知不觉中伤害孩子。

如果你曾被语言伤害，想理解那些话语背后的心理根源。

那么请花3分钟读完这篇文章，我相信，**它会让你对“家”和“爱”的表达，有全新的思考。**

  


#  01.你看别人家的孩子

前段时间，朋友跟我讲了件让她特别难受的事。

她女儿小雅很有画画天赋，最近参加市里的比赛拿了二等奖，孩子高兴得一蹦三尺高。

朋友本来也挺开心的，可家庭聚餐时，亲戚一句“小雅真棒，以后肯定能当大画家”，她却下意识地回了句：

“画画好有什么用？你看她表哥，这次期末考了全班第一，数学竞赛也拿了奖。要是她能把画画的心思分一半到学习上，我就谢天谢地了。”

一瞬间，女儿眼里的光，灭了。

整个晚上，孩子都闷闷不乐，回到家还把画笔和奖状都收了起来。

朋友这才意识到，自己随口的一句“比较”，像一把冷水，浇灭了孩子所有的热情和自豪。

  


这句话，我们是不是太熟悉了？

  *  _**“你看别人家的孩子多外向，你呢，一天到晚闷在家里。”**_
  *  _**“你看隔壁小张多懂事，从来不让他爸妈操心。”**_



我们总以为，“比较”能激励孩子上进，却忽略了一个扎心的事实：

**当你说“别人家的孩子”时，潜台词就是“你不如他”。**

 _**发展心理学认为，孩子的自我认同感，很大程度上来源于父母的肯定。**_

长期被比较的孩子，很难建立真正的自信。

他们会陷入一种“我不够好”的自我怀疑里，觉得无论自己怎么努力，都无法达到父母的标准。

他们不是变得更努力，而是变得更自卑，甚至会为了“达标”，隐藏自己真正的兴趣和闪光点。

别再用别人家的标准，来衡量自家的宝贝了。

他或许不是跑得最快的，但也许是看得最远的；

他或许不擅言辞，但内心可能藏着一个温柔的宇宙。

**你的看见，才是他自信的最大底气。**

  


#  02.我这么做都是为你好!

如果说第一句话是刺向孩子的“匕首”，那这一句，就是捆住孩子的“枷锁”。

它听起来充满了爱与奉献，实际上却是一种不容置喙的控制。

**还记得电视剧《小欢喜》里的英子吗？**

妈妈宋倩对她无微不至，打着“为你好”的旗号，安排了她的一切：

 _**必须喝的药膳，必须穿的衣服，必须上的补习班，以及必须报考的清华。**_

英子热爱天文，梦想是南京大学天文系。

**但在妈妈眼里，这都是“不务正业”。**

当英子声嘶力竭地喊出“我就是想要逃离你”时，宋倩却崩溃地反问：

**“我做错了什么？我都是为了你好啊！”**

  


你看，这句话的可怕之处在于，它把单方面的“付出”和“牺牲”包装成爱，然后站在道德高地上，让孩子无法反驳。

一旦孩子提出异议，就等于“不领情”、“不懂事”、“白费了父母一片苦心”。

 _**马歇尔·卢森堡博士在《非暴力沟通》书里说：“也许我们并不认为，自己的谈话方式是暴力的，但我们的语言，确实常常引发自己和他人的痛苦。”**_

“为你好”就是这样一句“软暴力”。

它堵死了亲子间所有沟通的可能，让两代人之间出现一条无法跨越的鸿沟。

真正健康的爱，不是“我说了算”，而是“我们一起商量”。

是蹲下来，听听孩子内心的声音，尊重他是一个独立的个体，有自己的思想和梦想。

**以爱为名的控制，最让人窒息。**

  


#  03.就这点小事，至于吗？

天晓得，这句话对一个陷入困境的孩子来说，有多大的杀伤力。

**蒋方舟曾分享过一个经历。**

她小时候当大队长，每天要在校门口检查谁没戴红领巾，没戴的要扣两分。

你总能看到有小孩因为没戴红领巾，在那儿哭得天崩地裂。

大人觉得，多大点事？

不就两分吗？

但对那个孩子来说，这两分没了，他就“没了”。

他会因此被老师批评，被同学嘲笑，影响班级荣誉，这是他小小的世界里，天塌下来一样的“灭顶之灾”。

**窦文涛说得特别好：**

**你不能说小孩娇气。他们眼里的这两分，等价于你欠了几百万还不上的绝望。**

  


成年人最大的傲慢，就是用自己的世界观，去随意评判孩子的世界。

孩子考试没考好，沮丧地回到家，你来一句：

 _**“一次考试而已，至于吗？”**_

孩子跟好朋友闹矛盾了，伤心地哭，你来一句：

 _**“同学之间吵吵闹闹很正常，至于吗？”**_

本来他只是想找你疗伤，寻求一个拥抱，你这句话直接把他推开，顺便还在他的伤口上撒了一把盐。

他会觉得，自己的情绪是不被理解的，自己的感受是无足轻重的。

慢慢地，他会关上心门，再也不向你倾诉任何事。

永远要记住，一个成熟的父母，**会先去处理孩子的情绪，再去处理事情。**

当他崩溃时，最好的回应不是讲道理，而是走过去，抱抱他，告诉他：

“我知道你现在很难过，没关系，妈妈在这里。”

**先共情，再解决。这才是通往孩子内心的桥梁。**

  


#  04.写在最后

**语言是有能量的，它可以是春风，也可以是冰霜。**

一个家庭的温度，往往就藏在父母的口中。

 _**把“你看别人家的孩子”换成“我看到了你的努力”；**_

 _**把“我都是为你好”换成“我们聊聊你的想法好吗？”；**_

 _**把“这点小事至于吗”换成“我理解你的感受”。**_

你会发现，当语言变得温暖，家，才真正成为一个充满爱与力量的港湾。
