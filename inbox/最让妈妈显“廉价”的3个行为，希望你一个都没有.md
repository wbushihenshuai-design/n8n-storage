# 最让妈妈显“廉价”的3个行为，希望你一个都没有

跨年夜跟几个朋友在一起吃饭。

**大家都是当妈的，几杯热红酒下肚，聊的话题自然离不开孩子和老公。**

席间有个朋友，是个咖啡店的主理人。

她把高跟鞋一脱，盘腿坐在沙发上说了一句很有意思的话：

> ❝
> 
> 你们发现没？
> 
> **有的女的，明明一身名牌，但这日子过得就是透着一股子穷酸气。**
> 
> **有的女的穿个U衣库，你都觉得她贵气。**

这话糙理不糙。

所谓的“廉价感”，跟钱多钱少关系真不大。

我见过很多有钱的怨妇，也见过很多没钱但活得特飒的姑娘。

在养孩子、过日子这事儿上，我早就把自己剖析明白了。

**当妈这件事，在我人生里顶多占40%。**

**剩下的60%，我得留给我自己，留给玩儿，留给那些“没用”的快乐。**

那些让妈妈显得特别“廉价”的行为，说白了，就是自己看轻了自己。

尤其这3个行为，希望你一个都没有。

  


# 01

**把自己当成家里的“边角料”，还非要说是为了爱。**

我有次带我儿子去吃必胜客。

隔壁桌是一对母子。

披萨上来了，那个妈妈把中间全是料、拉着丝的部分全切给了孩子。

然后自己默默地啃那个又干又硬的披萨边。

孩子吃不下了，说：

**“妈妈我饱了。”**

那位妈立刻急了：

**“这么贵的东西，别浪费！妈妈舍不得吃都给你了，你必须吃完！”**

最后孩子一边哭一边硬塞，妈妈一边骂一边啃剩下的残渣。

何必呢？

真的，何必呢？

这种场景我见得太多了。

  


**很多妈妈，特别喜欢演“苦情戏”。**

穿衣服穿旧的，吃饭吃剩的，水果吃烂的。

如果你是真的穷得揭不开锅，那我敬佩你的牺牲。

**但现在大部分家庭，真不差那一口吃的。**

这种行为的潜台词其实是：

**我不配享受好的，只配用边角料。**

你都觉得自己不配，谁还会觉得你珍贵？

最可怕的是，这种牺牲感，通常都伴随着巨大的索取。

“我为了你舍不得吃、舍不得穿，你怎么能考得这么差？”

这不叫爱，这叫道德绑架。

这是在拿自己的“惨”，去换孩子的“愧疚感”。

我跟我女儿、儿子在家，那是绝对的公平。

好吃的大家一起分，谁也别惯着谁。

我要是想吃牛排，我就自己点一份最好的，我不吃孩子剩下的。

我要让他们知道，**妈妈也是个独立的人，妈妈也爱美食，妈妈也值得最好的东西。**

一个自尊自爱的妈妈，才养不出唯唯诺诺的孩子。

  


# 02

**为了几块钱，在公共场合一直叨逼叨。**

我老公这个人，最大的优点就是不磨叽。

哪怕买贵了，买了就买了，绝对不回头逼逼。

这跟我很像。

但我见过很多妈妈，拥有一种“穷忙”的特质。

带孩子出门旅游，为了省个几十块钱门票，在门口跟黄牛磨叽半小时；

打个车，司机稍微绕了一点路，能从上车骂到下车，心情全毁了。

**这就叫“因小失大”。**

你的时间、你的心情、你在孩子面前的体面，难道不比那几块钱值钱吗？

我以前去国外出差，在摩洛哥，被小贩坑了一笔钱。

我当时第一反应也是想回去吵架。

后来一想，我大老远飞过来是来体验生活的，不是来练吵架的。

花了钱买个教训，这事儿就翻篇了。

  


**对于金钱的态度，直接决定了你的生活质感。**

不是让你挥金如土，而是**要有“翻篇”的能力** 。

**不要为了昨天犯的小错内耗，也不要为了刚才多花的五块钱叨逼叨。**

哪怕家里不富裕，也要教孩子大大方方的。

买得起就买，买不起就坦荡荡地说“咱们预算不够”。

**最怕那种，一边买了一边骂，一边花钱一边心疼。**

**最怕那种，一边图便宜买次品，一边用着糟心想扔掉。**

这种纠结和拧巴，传达给孩子的感觉就是：

生活是沉重的，匮乏的，是不快乐的。

记住，**为了省钱而囤积自己不需要、也不喜欢的廉价物品，不仅浪费了钱，更浪费了你生活的居住空间。**

  


#  03

**没有名字，只有“某某妈妈”这一张标签。**

这是我最受不了的一点。

**有些人，加上微信，头像也是孩子，名字也是“某某妈妈”。**

朋友圈里发的内容，除了娃就是娃。

除了聊孩子，你跟她聊电影、聊小说、聊八卦，她完全接不上话。

仿佛生了孩子之后，她本人就原地蒸发了。

只剩下一个叫“妈妈”的角色。

这太可怕了。

我说过，我是无法接受没有自我的人生的。

  


**我女儿读小学了，她有她的圈子，我有我的圈子。**

周末她去找同学玩，我就去找朋友喝酒、看展、读书、写文章。

我见过那种把全部身心都挂在孩子身上的妈妈。

孩子就是她的业绩，就是她的KPI。

孩子稍微有点风吹草动，她就觉得天塌了。

这种爱，太沉重了，孩子会被压得喘不过气来。

我经常跟我家孩子说：

**我们就是在一辆大巴车上相遇的旅伴。**

**这辆车开往终点，我们恰好同路而已。**

**我能陪你走一段，但我有我自己的风景要看。**

**你将来也要下车，去换乘你自己的车。**

  


#  04

**写这篇文章，不是为了指责你。**

而是真的心疼那个总是把自己忘在脑后的你。

**我们都在说“生命力”。**

所谓的“生命力”，不是穿名牌、背好包。

**而是你身上那股“不卑不亢、把自己活得精彩”的劲儿。**

请你学会，试着从“妈妈”这个角色里稍微抽离一点，把目光收回来，多看看自己。

不要把孩子当成你人生的全部，更不要把他们当成你唯一的作品。

**赫尔曼·黑塞在《悉达多》** 里写过一段话，我读到的时候心情久久复归平静，送给所有在这个角色里迷失的妈妈：

**“我们这一生，要自己上路，路过踌躇满志，路过困顿怀疑，路过傲慢无礼，路过世俗功利，路过爱与不爱，路过幡然醒悟，路过接受理解，路过自己的一生。”**

你看，这一切体验，都需要你自己去完成。

孩子代替不了你，你也代替不了孩子。

**同理心，首先要给到自己。**

当你开始发光的时候，你才会发现，原来赢得尊重最好的方式，不是弯腰服务。

而是昂首挺胸地，过好你自己的生活。

**你先是你自己，然后才是谁的妈妈。**
