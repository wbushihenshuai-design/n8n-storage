# “远离低端女孩，特别是初中生”，我发现：孩子的“毒友谊”，比没朋友更可怕，我劝你做个“势利”的妈妈

你有没有发现，自家那个活蹦乱跳的女儿，最近回家总是一副没精打采的样子，问她什么都说“你不懂”。

你有没有发现，孩子不知道从什么时候起，心思全不在学习上，成绩掉得让你心慌，嘴里念叨的全是班里那些鸡毛蒜皮的八卦。

当妈的，有时候你不得不承认个扎心窝子的事儿：

**有些朋友，不仅帮不了你孩子，还在悄悄地把她往下拉。**

几个月前，有个叫琳姐的妈妈留言求助，打出来的字都透着绝望：

**“我感觉我女儿，快被她那个好闺蜜给毁了！”**

琳姐的女儿小优，上初二，本来是个挺拔尖的孩子，性格也好。

可自从跟班里一个女孩走得特别近之后，整个人都变了。

  


那个“闺蜜”天天拉着小优， _**不是聊哪个明星塌房了，就是吐槽谁谁谁穿得土**_ 。

慢慢地，小优也开始学着说话夹枪带棒，心思重得不行，把所有精力都用在怎么讨好她那个闺蜜、怎么挤进那个小圈子上。

琳姐看着着急，想管管，小优就红着眼睛喊：

**“妈！你根本不懂！我在班里就她一个朋友了！”**

这句话，像针一样扎在琳姐心上。

她想不通，好好的朋友，怎么就能把孩子往火坑里拽呢？

  


# 01.“低端”女孩，是孩子成长的“隐形杀手”

我知道，我说“低端”这两个字，肯定有人想怼我，觉得我给孩子分三六九等。

先别急，我说的“低端”，跟家里有没有钱、穿什么牌子没半点关系，我说的是**眼界、想法和人品上的“层次低”** 。

这种朋友，就是孩子成长路上看不见的“杀手”，她们通常有这么几个特点：

## 她们就是个“情绪垃圾桶”：

**脑子里装满了各种负能量，** 今天跟你女儿抱怨爹妈偏心，明天吐槽老师不公。

你家孩子成了她倒苦水的对象，天天被这些破事搞得心情也不好。

## 她们特别爱搞小团体：

**精力旺盛，但全用在拉帮结派、孤立别人上了。**

你女儿宝贵的学习时间，全被这些“谁跟谁好了”、“谁又不理谁了”的破事给搅和了。

## 她们的嘴里全是“毒鸡汤”：

**会悄悄给你女儿灌输“学习有啥用，长得好看才重要”、“再努力有啥用，人家里有矿”这种歪理。**

  


让你家孩子觉得努力没意思，干脆一起“躺平”算了。

说真的，初中这个年纪的孩子，最容易被朋友影响了。

交了这种朋友，就像把一棵好苗子种在了盐碱地里，慢慢就给养废了。

这种朋友，不动手打人，但她们对你孩子的伤害，是精神上的，是看不见的刀子，一刀一刀割掉孩子的灵气和上进心。

这真的比没朋友可怕多了！

  


# 02.父母的“势利”，是孩子成长路上的“铠甲”

发现问题了，咱当妈的该怎么办？

硬把她们拆开？

孩子肯定跟你急。

我的建议，可能有点不好听：

**咱得学着做个“势利”的妈妈。**

这里的“势利”，不是让你嫌贫爱富，教孩子去巴结谁。

而是说，咱心里得有杆秤，得**清醒地帮孩子把把关，看看她交往的是些什么人。**

**我听过一个特有智慧的说法：**

> ❝
> 
> **青春期的朋友，说白了就是互相影响。**
> 
> 当妈的所谓“势利”，就是帮孩子筛掉那些天天消耗她、拉低她的朋友，引导她去靠近那些能让她变得更好、更有劲头儿的圈子。

  


后来，琳姐就这么干的。

她没跟女儿硬碰硬，而是开始玩“曲线救国”。

 _**她发现那个“闺蜜”不爱运动，就给女儿报了个周末的羽毛球班；**_

 _**发现那个女孩不爱看书，就鼓励女儿多去图书馆，参加学校的读书会。**_

你猜怎么着？

小优在这些新地方，认识了新朋友。

有在球场上挥汗如雨、性格爽朗的女孩，也有聊起书来两眼放光的学霸。

她这才发现，哦，原来朋友之间不只是说闲话和比吃穿，还可以是一起流汗，一起进步。

**孩子的圈子大了，眼界宽了，自然就瞧不上原来那个小泥潭了。**

  


#  03.远离“低端”社交，建立“高端”友谊

**那具体怎么帮孩子交到真正的好朋友呢？**

这里说的“高端”友谊，也不是说多有钱，而是**精神上能互相滋养** 。

## 1.教会孩子怎么“看人”。

你有空得跟孩子好好聊聊，问问她：

**你觉得什么样的朋友才是真朋友？**

是那个你考好了，她比你还高兴的人？

还是那个见不得你好，说话酸溜溜的人？

让孩子心里有个谱，知道谁是真心对自己好，谁只是拉着自己一起混日子。

## 2.当妈的，要主动帮孩子“搭台子”。

**别总让孩子闷在家里。**

**多鼓励她参加一些有意义的活动，** 比如学校的兴趣小组、社区的志愿者活动、体育比赛什么的。

这些地方聚集的孩子，大都积极又阳光。

环境对了，遇到对的人的几率自然就高了。

## 3.最根本的，是让孩子自己“发光”。

说到底，打铁还得自身硬。

**你得把自家孩子的内心养得丰盈、自信。**

一个有自己的兴趣爱好、有自己目标的女孩，本身就对那些无聊的拉帮结派不感兴趣。

**你想让孩子交到什么样的朋友，就得先引导她自己成为那样的人。**

她自己亮了，自然能吸引来更多发光的人。

  


# 04.做个“势利”的妈妈，不是为了虚荣，而是为了保护

写到这儿，我知道这些话听着挺“功利”的，**跟我们从小受的“要和所有同学都搞好关系”的教育不一样** 。

但现实就是这么回事。

**孟母三迁的故事，从古至今都在上演。**

**“近朱者赤，近墨者黑”** 这句话，在孩子成长的关键几年，一个字都不能含糊。

咱们当妈的做这点“势利眼”，不是教孩子冷漠和虚荣，而是在她还分不清好坏、没长出翅膀的时候，拼尽全力为她撑起一把伞，护她周全。

**这真的不是无情，而是一个母亲最本能的保护和最深沉的爱。**
