# 1、练脸皮：放下脸面，你才会有面子！

**重要性:**

脸皮厚，才能吃得开！

男孩想要成事，就要敢于放下所谓的面子，去争取机会！

**如何做:**

**鼓励孩子参加学校的竞选。**

即使失败了，也是一次宝贵的经验，让他明白，输了也没什么大不了！

**让孩子去菜市场砍价。**

锻炼他的沟通能力和谈判技巧，让他知道，大胆尝试，才能获得更多！

**鼓励孩子在公共场合表演。**

唱歌、跳舞、讲故事，任何形式都可以，让他克服在众人面前表现的恐惧！

  


# 2、练自信：如果你都不自信，谁会相信你？

**重要性:**

自信是成功的力量！

只有相信自己，才能感染别人，才能勇往直前！

**如何做:**

多给孩子积极的肯定。

每天早上都告诉他：“你今天一定会很棒！”

让他带着自信开始新的一天。

让他挑战自己不擅长的事情。

比如学习一门新的乐器、参加一个他不熟悉的社团，让他体验克服困难后的成就感。

鼓励孩子说出自己的想法。

即使他的想法不成熟、不合理，也要认真倾听，并给予肯定。

  


# 3、练阅历：学会说话，会办事，学做人！

**重要性:**

阅历丰富的男孩，更成熟，更有魅力！

他知道如何与人相处，如何处理问题，如何走向成功！

**如何做:**

**带孩子去不同的地方旅游。**

不仅仅是看风景，更重要的是让他接触不同的文化，了解不同的生活方式。

**让孩子参与家庭的决策。**

比如装修房子、选择旅游地点，让他感受到自己被重视，并学会承担责任。

**鼓励孩子做志愿者。**

帮助他人，体验生活的艰辛，培养他的同情心和责任感。

  


# 4、会赞美：学会拒绝，会赞美，情商高！

**重要性:**

赞美别人，其实是赞美自己！

懂得赞美的男孩，人缘更好，机会更多！

**如何做:**

教孩子学会发现别人的优点。

**比如：**

“你的字写得真漂亮！”

“你的篮球打得真好！”

让他学会真诚地表达赞美。

不要敷衍了事，要发自内心地欣赏别人。

鼓励孩子每天赞美身边的人。

比如家人、朋友、老师，让赞美成为一种习惯。

  


# 5、练演讲：成功者大多数都是演讲高手！

**重要性:**  

口才好的男孩，更具竞争力！

他能清晰地表达自己的想法，说服别人，赢得机会！

**如何做:**

鼓励孩子参加学校的演讲比赛。 即使没有获奖，也是一次宝贵的锻炼机会。

让他对着镜子练习演讲。 注意他的仪态、语速、语气，帮助他提升演讲技巧。

让他给家人讲故事。 锻炼他的表达能力和组织能力。

  


# 6、练沉稳：再大的事，也要变得淡定！

**重要性:**

沉稳冷静的男孩，更值得信赖！

他能理性分析问题，找到解决办法，而不是惊慌失措。

**如何做:**

**遇到问题时，告诉他先冷静下来。**

深呼吸，数到十，或者喝一杯水，让他平静下来。

**教他分析问题，找到问题的根源。**

不要只看表面现象，要深入挖掘，才能找到解决办法。

**让他多经历一些事情。**

经历的多了，自然就淡定了。

  


# 7、练心智：不生气，不自证，不被他人激怒！

**重要性:**

内心强大的男孩，才能活得自在！

他不被情绪绑架，不活在别人的评价里，专注自己的成长！

**如何做:**

**教他学会控制自己的情绪。**

比如，感到生气时，可以转移注意力，做一些自己喜欢的事情。

**告诉他没必要向所有人证明自己。**

专注自身成长，做好自己就好，不必在意别人的眼光。

**教他理性看待不同的观点。**

每个人都有自己的想法，不必试图说服所有人，也不要轻易被别人的言语激怒。

  


# 8、练胆识：敢于与任何人眼神对视！

**重要性:**

眼神是自信的体现！

敢于与人眼神对视的男孩，更有气场，更容易赢得别人的信任！

**如何做:**

**鼓励孩子与人交流时，看着对方的眼睛。**

不要害羞，不要躲避，坦诚地面对别人。

**让他练习在镜子前与自己对视。**

锻炼他的自信心和自我认知。

**创造机会让他与陌生人交流。**

比如在餐厅点餐、在商店购物，让他习惯与人交流。

  


# 9、练气场：昂首挺胸，腰板挺直！

**重要性:**

气场强大的男孩，更具吸引力！

昂首挺胸，充满自信的姿态，更容易赢得别人的好感和尊重！

**如何做:**

**时刻提醒孩子保持良好的仪态。**

坐直，站直，走路时抬头挺胸。

**鼓励孩子参加体育锻炼。**

强健的体魄是气场的基础。

**培养孩子积极乐观的心态。**

积极的心态是气场的源泉。
