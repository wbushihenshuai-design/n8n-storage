# 一个恢复孩子灵气最好的方式：赋能

你是否见过，那个曾经眼里有星星、笑声清脆的孩子，不知从何时起，光芒变得黯淡？

**很喜欢一句话，最好的家庭教育是赋能，是爱与同理心。**

孩子不是一台需要不断优化的机器，而是一颗需要被点亮的星星。

只要孩子的心成长起来，心是活的，就不用过于担心。

当他内心充满光和热，自会找到属于自己的轨道，璀璨夺目。

  


# 01、别再给孩子贴“懒”的标签了，他可能只是“没电了”

我们常常看到孩子磨磨蹭蹭，对学习提不起劲，我们心急如焚，脱口而出：

**“你怎么这么懒！”**

但其实孩子看起来“懒”，不是缺乏动力或者叛逆，而是**内心根本没有足够的能量去支撑长久的努力** 。

你可以想象一下，手机只剩下1%的电，它会自动关掉所有高耗能的东西。

孩子的状态就是这样，他不是“懒”，他是**内心能量严重不足，“没电了”** 。

学习、思考这些都是高耗能的脑力劳动，需要耗费极大的心力。

当一个孩子内心缺乏力量，极度没有安全感，**他的绝大部分精力都花在寻找安全感，处于内耗之中** 。

  


**他不是在思考题目，而是在担心：**

> ❝
> 
> **我这次考不好，爸妈会不会骂我？**
> 
> **老师是不是觉得我很笨？**
> 
> **我是不是真的什么都不行？**

这种杂念挥之不去，就像一个看不见的贼，偷走了孩子所有的能量。

一旦能量被过度消耗，他只会转向其他的低级能耗行为来逃避。

比如刷短视频、看小说、打游戏等。

那不是堕落，而是一种求救：

**“我太累了，让我喘口气吧。”**

  


#  02、你的家，是“充电桩”还是“耗电器”？

**孩子的能量从何而来？**

**它来源于孩子被爱、被认可和被看见的体验。**

每一位父母，都要成为孩子的正向国王，正向的女王——**要么是温暖的“充电桩”，要么是不断消耗他的“耗电器”** 。

我们看看自己无意中变成“耗电器”的样子：

  * **纠错的目光：** 我们总能第一时间发现孩子做错了什么，却很少因为他每一次做对了什么而高兴。
  * **比较的语言：** “你看隔壁的谁谁” 这句话像根刺，让孩子觉得自己永远不够好。
  * **焦虑的传递：** 我们把对未来的焦虑和恐慌，苦口婆心地传给孩子，却忘了他小小的肩膀，根本扛不住。



  


**相反，“充电桩”式的父母，他们做的恰恰相反。**

就像植物需要阳光雨露，孩子的心灵也需要父母无条件的接纳与滋养啊。

他们的爱与认可，不看分数和表现。

这种爱，会转化为孩子内心最底层的安全感，告诉他：

“无论你怎么样，你都值得被爱的。”

**只有在温暖有爱的家里，孩子才敢放心地出去闯荡，不怕遇到困难。**

  


#  03、成为孩子的“超级充电宝”，只需记住这个心法

那么，如何从一个焦虑的“耗电器”，转变为一个智慧的“充电桩”呢？

**试试这样做：**

**用“体验”置换“说教”，用“看见”代替“评判”。**

具体怎么做？

**1.从“你必须”到“去感受”**

我发现，大部分父母都没耐心去理解孩子们复杂的情绪。

他们太急切地想将“人生的大道理”传授给孩子了，却忘记：**身教比言传重要** 。

你不要在孩子一上学的开始就施加各种压力，而是要和他强调：

> ❝
> 
> 你会交到很多朋友，你会遇到很多有趣的老师，还能吃到食堂里厨师叔叔做的美味饭菜！

明白了吗？

**你要和孩子去强调体验的过程，里面有快乐、有挑战，也可能有痛苦。**

你越是让他去发现生活中的美好，他过得越开心，他也能找到自己要走的方向。

  


**2.从“挑毛病”到“找亮点”**

不要因为孩子做错了什么而认为他有问题，而是因为他每一次做对了什么而高兴。

他今天主动帮你拿了拖鞋，马上夸赞：

“谢谢你，真是个体贴的孩子，妈妈感觉好温暖！”

他的一道数学题解法很有创意，哪怕结果是错的，也要肯定：

“哇，这个思路太酷了！你是怎么想到的？”

**记住，那些喋喋不休的批评、指责和否定，不但不能改变孩子的行为，反而会削弱孩子内心的力量。**

  


**3.从“解决问题”到“接纳情绪”**

当孩子因为考试失利而沮丧时，最差的回应就是愤怒地指责，“就你事儿，就你作”。

最好的回应，是静静地陪着他，抱抱他，然后说：

> ❝
> 
> 我知道你现在很难过，没关系的，即便会遇到一些挫折，也没关系，那些都是你人生中很珍贵的体验。

**先接纳他的情绪，再处理事情。**

当他的情绪被你温柔地看见，那份挫败感就不会成为消耗他的黑洞，反而会成为他成长的养料。

  


**我写这篇文章谈论的，其实并非孩子，而是我们自己。**

一个内心枯竭、时常焦虑的父母，无论掌握多少技巧，都无法伪装成一个恒定的“充电桩”。

你的每一次不耐烦，每一次失控，都在诚实地告诉孩子：

**“我的电量也不足了啊。”**

孩子只是家庭能量场中最敏感的探测器，他的黯淡，精准地反映了我们的损耗。

**所以，恢复孩子灵气的最好办法，是先回头看见那个同样“电量耗尽”的自己。**

**疗愈自己，就是疗愈你的家；**

**给你自己赋能，才是给孩子最好的赋能。**
