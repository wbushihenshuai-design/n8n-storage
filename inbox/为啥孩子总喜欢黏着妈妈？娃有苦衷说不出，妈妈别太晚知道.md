# 为啥孩子总喜欢黏着妈妈？娃有苦衷说不出，妈妈别太晚知道

都说当妈的膀胱是铁打的，我一个朋友说，不是，是压根就没机会用。

她一天上六次厕所，五次都有个孩子扒在门上，哭得撕心裂肺。

门关上的那一秒，就是世界末日。

拍门，嚎，哭声里带着绝望。

她坐在马桶上，听着外面的哭声，跟我说：

“真的，太窒息了。”

**“我不是不爱他，但我连三分钟完整的、属于自己的排泄时间都没有。”**

“这件事太让人崩溃了。”

  


# 1、他不是要黏你，他是真害怕

**我们总觉得孩子黏人，是“作”，是“不乖”。**

这么说吧，一个一岁的小孩，他的世界有多大？

可能就是从客厅到卧室的距离。

而在这个世界里，你，就是太阳。

太阳没了，整个世界就是一片漆黑，你让他怎么不怕？

**他不是故意要黏着你，挑战你的耐心。**

是他脑子里的“程序”就是这么设定的：

**妈妈在，等于安全。**

**妈妈消失，等于危险。**

就这么简单粗暴。

我们以为他只是在无理取闹，在他那边，这根本就是生死存亡的恐惧。

你让他不黏你，就等于让他一个人去对抗全世界的怪兽。

  


# 2、你一转身，他就以为你不要他了

**很多妈妈都是琢磨了很久才想明白这件事。**

你跟孩子玩“躲猫猫”，你用手把脸一捂，再一打开，他为什么笑得那么开心？

因为在他的认知里，你刚刚是真的消失了，然后又奇迹般地“死而复生”。

**他还没搞明白一个叫“客体永久性”的东西。**

**就是说，一个东西看不见了，不代表它不存在了。**

  


这个道理，我们大人觉得天经地义，但在他那里，压根就不是一回事。

你走进厕所，关上门。

**在你的世界里：**

我就是去上个厕所，两分钟就出来。

**在他的世界里：**

**妈妈没了。**

她走进一个门里，然后就没了。

她还会回来吗？

我不知道。

**我被抛弃了。**

为什么连上个厕所都像是一场生离死别，就是这个原因。

**他不是不信任你，是他“智商”还不够理解这件事。**

  


#  3、他不是在瞎胡闹，他是真的不舒服

还有一种黏人，是挂在妈妈身上哼哼唧唧，像个没骨头的小考拉，怎么放都放不下来。

你以为他是又要抱抱。

其实呢？

他可能就是说不出来的难受。

**我那个朋友的娃有一阵子就是这样，非要她抱着睡，一放床上就炸。**

后来她才发现，他那几天在长牙，牙龈又肿又疼，躺着比抱着更难受。

孩子不会说话，这是最要命的。

饿了是哭，困了是哭，衣服标签扎着了是哭，想拉屎了也是哭。

**所有的不舒服，出口都只有一个，就是“哭”和“找妈妈”。**

妈妈的怀抱，是他能想到的唯一的、能缓解所有痛苦的“万能药”。

他不是在跟你胡闹，他是在用他唯一会的方式求救。

  


# 4\. 现在抱得够，以后才敢自己走

我那位朋友说，她累，真的累。

**一天24小时，除了睡觉那几个钟头，身上几乎随时都长着个人。**

有时候她烦得不行，真想把孩子丢给谁，让她清静一个小时。

但她后来自己算了一笔账。

他真正这样毫无保留地、把全身心都托付给妈妈，需要妈妈的时间，一共才几年？

三年？

五年？

这几年，妈妈抱他的每一分钟，都不是浪费。

**是在给他一个叫“安全感”的账户里存钱。**

现在这个账户存得够多，满满的，他以后才有底气自己走出去，去幼儿园，去上学，去面对外面那些他搞不定的事情。

**如果现在这个账户是空的，他将来会花一辈子的时间，去寻找别人的认可，去填补这个窟窿。**

哪个成本更高，一算就知道。

  


**所以她后来跟我说，她不焦虑了。**

下次她再去上厕所，孩子在外面哭，她大概还是会烦躁。

但她会隔着门跟他说一句：

**“宝宝，妈妈在，妈妈马上就出来。”**

她不再觉得孩子是来折磨她的了。

他只是在用一种很笨拙的方式，拼命地确认：

**我的太阳，还在。**
