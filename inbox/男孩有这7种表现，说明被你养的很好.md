# 男孩有这7种表现，说明被你养的很好

我发现，很多家长评价一个孩子好不好，标准就是**听话** 。

听话就是乖，乖就是好。

但我必须泼一盆冷水。

在这个复杂多变的世界，仅仅是听话的男孩，未来往往很吃亏。

过于顺从，意味着他放弃了自我意志，意味着他没有独立思考能力。

**如果你的儿子有下面这7种表现，恭喜你，你给了他真正的爱与自由。**

  


#  01

**愿意和你分享心里话**

很多男孩进入青春期后，回到家就像个哑巴。

问他什么，都是“不知道”、“还行”、“知道了”。

为什么？

因为他们没有安全感。

**在他们的经验里，一旦说出心里的实话，换来的往往是评判、说教，甚至是嘲讽。**

如果你的儿子，愿意把他在学校的糗事、他喜欢的那个女孩、甚至他的害怕告诉你。

这说明了一件事：

**你对他不仅没有审判，而且充满了接纳。**

心理学上有个词叫**「心理安全感」** 。

只有当孩子确信，展示脆弱不会招致攻击时，他才敢开口。

**这说明你没有利用父母的威严去压制他，而是用同理心在托举他。**

这很难得。

  


# 02

**有主见，敢和你顶嘴**

很多妈妈看到孩子顶嘴，第一反应是暴怒：

你怎么敢这么跟我说话？

这是典型的**控制欲** 在作祟。

你在享受绝对的掌控权，容不得一点挑战。

遗憾的是，一个从来不敢说“不”的男孩，长大了就是标准的讨好型人格。

**在职场上不敢拒绝无理要求，在婚姻里没有底线。**

如果他敢拒绝你，敢为了维护自己的玩具、时间、观点跟你据理力争。

别生气。

这说明他的**自我边界** 建立得非常好。

  


# 03

**具备基本的生活自理能力**

我不止一次看到，十几岁的男孩，鞋带散了都要等妈妈蹲下来系。

这种孩子，是父母一手包办出来的。

有些妈妈，通过这种无微不至的照顾，来通过**“被需要”** 获得存在感。

这其实是一种自私。

真正带得好的妈妈，懂得**得体的退出** 。

你会发现，他会自己洗衣服，会简单的做饭，房间乱了知道自己收拾。

这不仅是技能，更是**责任感** 。

因为他知道，没人应该伺候他一辈子。

生活是自己的，谁也替代不了。

  


# 04

**懂得尊重弱者，对人有礼貌**

怎么看一个男孩的教养？

不要看他怎么对父母、对老师。

**要看他怎么对弱者。**

如果你发现他不管是**面对服务员，还是面对比他弱小的女生** ，都能保持基本的礼貌和尊重。

不轻视，不嘲笑，不利用身体优势去霸凌。

这说明什么？

**说明他的爸爸很尊重妈妈，或者，妈妈很尊重自己。**

  


#  05

**犯了错敢于承认，不推卸责任**

**孩子撒谎，通常就是因为害怕。**

因为他知道，一旦承认错误，就会受到爸妈的教训，甚至是被打一顿。

所以，孩子才会撒谎。

如果你的孩子考试考砸了，他第一反应不是藏着掖着，而是敢走到你面前承认。

这说明你给了他一种底气：

**“我爱你，优于你的表现。”**

  


#  06

**输得起，受挫后能自我调节**

现在的孩子，很多是玻璃心。

只能听鼓励的话，听不得一句批评。

**一输就哭，一败就崩。**

这是因为父母平时保护得太好了，人为地帮他屏蔽了所有的挫折。

带得好的男孩，身上有一股**韧劲** 。

球输了，他会难过，但不会耍赖，擦干眼泪下次再来。

被老师批评了，他会难受，但不会自我攻击，觉得自己一无是处。

因为你告诉过他：

**失败不代表你不行，只代表这件事没做好。**

  


#  07

**与你亲密，但精神独立**

这是最舒服的母子关系。

很多妈妈，把儿子当成了老公的替代品，或者情感的唯一寄托。

这就导致了严重的**“共生关系”** ，最后变成了婆媳矛盾的根源。

真正好的关系，是**亲密有间** 。

你们可以像朋友一样聊天，像朋友一样互助。

但是，你有你的生活，他有他的世界。

他不需要每天向你汇报行踪，你也不需要每分每秒监控他的情绪。

**心理学家阿德勒说：一切烦恼皆源于人际关系。而解决之道，在于课题分离。**

你分得清，那是他的人生，这是你的人生。

  


贝姬·肯尼迪在**《看见孩子》** 书中说：

**“孩子养育好坏不取决于有无冲突，而取决于冲突后能否关心孩子，了解他们感受，努力修补关系裂痕。”**

写这篇文章的初衷不是想告诉你“怎么做更完美的父母”，而是教你如何真正理解孩子、看见孩子。

**拥有同理心，看见他作为一个独立个体的需求，这才是教育的起点，也是终点。**
