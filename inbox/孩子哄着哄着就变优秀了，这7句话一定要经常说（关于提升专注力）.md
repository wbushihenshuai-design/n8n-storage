# 孩子哄着哄着就变优秀了，这7句话一定要经常说（关于提升专注力）

> ❝
> 
> **天底下只有一种方法可以促使他人去做任何事，给他想要的东西。**
> 
> **——戴尔·卡耐基**

孩子的专注力不是天生的，而是需要呵护和**“夸”** 出来的。

很多时候，孩子坐不住、爱走神，是因为大脑缺乏正向的反馈。

以下是7句能保护和提升孩子专注力的话，建议家长融入日常沟通。

  


# 1、我刚才看你写作业的时候，哪怕遇到难题也没有东张西望，而是在努力思考，这一点妈妈觉得特别棒！

孩子写作业遇到困难，眼神游离时？

别急着吼“你看哪儿呢！专心点！”

捕捉他坚持的那一刻，强化它。

**试着这样说：**

我刚才看你写作业的时候，哪怕遇到难题也没有东张西望，而是在努力思考，这一点妈妈觉得特别棒！

**很多家长只夸“你真聪明”或“做完了真棒”，这会让孩子为了结果而焦虑。**

这句话的目的是指出孩子“专注”这一行为，就像一面镜子，让孩子意识到：

**“原来我刚才那个样子就是专注，那个样子是被认可的。”**

  


#  2、这件事交给你，我相信你可以安排好时间。如果需要帮忙，随时喊我。

布置任务时，总是担心他磨蹭，忍不住反复催促？

唠叨会破坏孩子的掌控感，引发被动抵抗。

**试着这样说：**

这件事交给你，我相信你可以安排好时间。

如果需要帮忙，随时喊我。

**专注力的天敌是“被动感”。**

如果孩子觉得是在为你学习，他就会磨洋工。

这句话传递了信任，把任务的主导权还给孩子，激活他的内驱力。

当孩子感觉到被信任时，他们会更愿意启动“自我管理”模式，为了不辜负这份信任而主动专注。

  


# 3、刚才你一口气坚持了20分钟没停笔，现在的感觉是不是比拖拖拉拉舒服多了？

孩子完成任务后，别只顾着检查对错。

引导他去体会“专注”带来的愉悦感。

**试着这样说：**

刚才你一口气坚持了20分钟没停笔，现在的感觉是不是比拖拖拉拉舒服多了？

这样说，是在帮助孩子的大脑建立多巴胺奖赏回路。

让他明白，高效完成任务后的轻松感，远比边玩边写要快乐得多。

  


# 4、刚才那个地方是不是卡住了？没关系，谁遇到难题都会心烦。深呼吸一下，喝口水，我们重新来。

孩子因为做不出题而烦躁、摔笔？

别责怪他没耐心，情绪乱了，专注力就断了。

**试着这样说：**

刚才那个地方是不是卡住了？没关系，谁遇到难题都会心烦。

深呼吸一下，喝口水，我们重新来。

这样说，先接纳情绪，再处理问题。

**很多孩子分心是因为畏难情绪。**

当孩子遇到困难想逃避时，不要指责他“不专心”，而要共情他的困难。

**降低了焦虑，专注力才能回归。**

  


#  5、这次你比上次多坚持了5分钟才走神，这就是进步！说明你的大脑越来越强壮了。

孩子定力不足，才坐一会儿就跑了？

别总盯着他走神的时候，要盯着他进步的时刻。

**试着这样说：**

这次你比上次多坚持了5分钟才走神，这就是进步！

说明你的大脑越来越强壮了。

专注力像肌肉一样，是可以训练的。

用**成长型思维** 评价孩子，让孩子觉得专注力是可以像锻炼身体一样，通过练习慢慢变强的。

  


# 6、我也很好奇，这个题目你是用什么方法解出来的？能不能教教我？

孩子做对了一道复杂的题？

利用这个机会，让他进入深度的“费曼学习法”状态。

**试着这样说：**

我也很好奇，这个题目你是用什么方法解出来的？

能不能教教我？

**当孩子处于“教”的位置时，他的专注度是最高的。**

这样说，为了教会别人，孩子必须高度集中注意力去梳理逻辑。

这种高强度的思维活动，是训练深度专注的绝佳机会。

  


# 7、刚才我看你太专心了，连我走过来你都没发现，我就没舍得打扰你。

想给孩子送水果，或者看他写得怎么样了？

请尽量不要打断他！

如果不得不说话，事后要补上这句。

**试着这样说：**

刚才我看你太专心了，连我走过来你都没发现，我就没舍得打扰你。

**这是一句最高级的赞赏。**

很多时候孩子的专注是被家长的一句“吃水果吗”、“喝水吗”打断的。

这句话既保护了孩子的专注，又事后强化了他“沉浸其中”的状态。

他在潜意识里会对自己产生一种自豪感：

**“原来我进入状态时这么厉害”，从而更爱上这种心流体验。**

  


#  关键实施原则

**不打扰原则：** 当孩子在玩积木、画画或发呆思考时，只要不涉及危险，绝对不要去打断他。那是专注力的雏形。

**三明治沟通法：** 如果必须批评孩子分心，请用“肯定刚才的努力➕指出走神的问题➕表达对下次的期待”的方式。

**环境极简：** 桌面上只留当前要做的书本，减少物理干扰，比说一万句“专心点”都管用。

**父母的语言是孩子的镜子。**

**你越相信他能专注，他就越能静下心来。**

**从今天开始，试着把这些话变成口头禅吧！**
