# “妈妈，给我10块钱”，当孩子开口要钱时，这3句话最好别说

看到小区里一个妈妈对孩子大吼“你怎么天天要钱”时，我真的有点被震惊到。

我发现，很多父母真的被一种“教育洁癖”保护得很好，对孩子内心的真实世界，有一种近乎天真的误解。

  


# 01.“你怎么又要钱？”

我有个同事，她常在办公室抱怨：

“我家孩子上小学了，怎么三天两头就要钱？买文具、买零食、同学过生日凑份子，简直就是个无底洞！”

而当孩子伸出手，怯生生地说“妈妈，给我10块钱”时，一句脱口而出的“你怎么又要钱”，为什么杀伤力这么大？

**我请问：**

**不耐烦、质疑、甚至带着一点羞辱的语气，你以为是在培养孩子正确的金钱观？**

这样的质问，不光让孩子觉得自己的需求是“不应该的”，更是在告诉他：

**要钱，是一件可耻的事。**

还没有财商教育这个概念的时候，我们小时候听到的版本是：

 _**“家里哪有钱给你？”**_

 _**“就知道要钱！”**_

除了让孩子闭嘴，什么正面效果都没有。

你真以为孩子是圣人，能瞬间理解你的疲惫和压力？

**他只会记住那一刻，自己伸出手时的尴尬和被拒绝时的羞愧。**

  


**抵制这种“羞辱式”回应，我们应该把它看作一次沟通，一次教育，一次建立信任的机会。**

我们应该蹲下来问问他钱的用途，和他一起分析这笔花费是否合理，而不是用一句话，把他推开。

企图用一句简单的质问，掩盖自己财商教育的缺失和沟通上的懒惰？

“你怎么又要钱”，不是对孩子消费行为的引导，而是整个家庭财商教育缺失下，**把所有压力和负面情绪都推给了最弱小的那个** 。

我们需要的是开放的沟通、是规则的建立、是信任的传递，而不是一句冰冷的质问。

  


# 02.“爸爸妈妈赚钱多辛苦，你还乱花钱！”

**我发现，很多父母真的被“愧疚式教育”感动了，对这种方式的效果有一种近乎盲目的自信。**

**他们以为的“愧疚式教育”：**

孩子听完后，会立刻变得懂事、感恩、节俭，从此一分钱掰成两半花。

**实际上的“愧疚式教育”：**

孩子会产生巨大的**负罪感** 和**匮乏感** 。

他觉得花家里的每一分钱，都是在“榨取”父母的血汗。

钱不敢花，需求不敢提，自卑自责，真是两头堵，三面夹击。

很多为金钱焦虑的成年人还是没明白，我只好把话说得更难听一些。

**我作为一个孩子，我不会因为你的诉苦，就瞬间理解成人世界的复杂和辛劳。**

我只会把“钱”和“痛苦、牺牲、争吵”这些负面词汇牢牢绑定。

你把赚钱的辛苦毫无保留地灌输给我，我只会觉得钱是个坏东西，它让我的父母这么痛苦。

等我长大了，我可能不敢追求财富，因为我觉得那意味着要承担巨大的痛苦。

**甚至因为这种负罪感，我可能会用另一种方式“报复性消费”。**

  


 _**从心理学角度看，它是“童年匮乏感”的一种延续；**_

 _**从家庭关系来看，它是“情感绑架”的一种表现；**_

 _**从财商教育来看，它是“把孩子推向另一个极端”的后果。**_

我们需要的是教会孩子珍惜，而不是让他背负罪恶感。

你可以告诉他钱是通过劳动换来的，甚至可以带他体验，但不要把赚钱的痛苦全盘倒给他。

你应该让他明白，努力赚钱是为了让家人过上更好的生活，钱是实现梦想的工具，而不是带来痛苦的根源。

你应该建立好规则，比如固定的零花钱制度，让他学会在有限的范围内做规划。

**你应该鼓励他通过正当劳动获取报酬，理解“付出与回报”的逻辑。**

而不是简单粗暴地，将所有生活的沉重，全部压在一个孩子身上。

  


# 03.“等会儿再说！”

这句话听起来最温和，却可能是最冷漠的。

它背后的潜台词是：

**“你的需求不重要”、“我现在很忙，没空理你”、“别来烦我”。**

你以为的“等会儿再说”：

只是一个简单的拖延，一会儿想起来了就会处理。

实际上的“等会儿再说”：

孩子接收到的信息是**“我的感受和需求被忽略了”** 。

一次又一次的敷衍，关上的不只是钱包，更是亲子沟通的大门。

下次，他可能宁愿去问同学借，或者偷偷拿家里的钱，也不愿再来“打扰”你。

依然是那句话，**向下的沟通绝不是沟通** 。

  


孩子不需要看似温和实则充满忽视的所谓“缓冲”，只需要被看见，被尊重。

在父母和子女的关系之前，孩子首先是一个独立的个体。

**大家反对的，不是不能延迟满足，而是这种敷衍的态度。**

你可以说： _**“我现在有点忙，但你这件事很重要。我们约定10分钟后，坐在沙发上好好聊聊这件事，好吗？”**_

**这叫约定，这叫尊重。**

而“等会儿再说”，叫敷衍，叫无视。

做了父母也可以有更好的沟通方式，更好的人生，不要把自己局限于一个所谓的“权威者”角色。

当孩子再次伸出手说“妈妈，给我10块钱”时，请记住，这不仅仅是关于钱，更是关于爱、关于信任、关于一个孩子未来如何看待自己与这个世界的关系。

**不要让10块钱，成为你和孩子之间一道沉默的墙。**
