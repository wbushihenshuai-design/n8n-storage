# 越没本事的父母，越喜欢在3件事上装“大方”，苦的是孩子

我活了半辈子，发现一个扎心的规律。

**越是没什么真本事的父母，越喜欢在几件事上装“大方”。**

这种大方，不是真的慷慨，它是一种表演，一种心虚。

最后的账，全都得孩子来买单。

  


# 1、对外人挥金如土，对家人斤斤计较

**这种“大方”，是表演给外人看的，带着一股歇斯底里的证明欲。**

跟狐朋狗友吃饭，一顿饭花掉小半个月工资，眼睛都不眨一下。

买单的时候，抢着上，生怕别人觉得他小气，觉得他混得不好。

回到家，就变了个人。

**老婆想买件新衣服，他能念叨半个月，说她败家。**

**孩子想买双好点的运动鞋，他说孩子虚荣，旧的还能穿。**

甚至菜市场的葱是不是贵了两毛钱，他都能跟你吵一架。

  


那种感觉是什么？

就是在这种父母心里，家人连一个不熟的朋友的面子都不如。

**家人的需求，家人的渴望，是“不应该”的，是“浪费”的。**

**而他在外面维护那点虚无缥缈的面子，却是“必须”的。**

**这种家庭长大的孩子，会有一种骨子里的不配得感。**

他不敢要东西，不敢提要求，他觉得自己的欲望是可耻的。

他以后赚了钱，可能只敢给别人花，不敢给自己花。

**这种匮乏感，会像水鬼一样，死死拖住孩子一辈子。**

  


#  2、夸别人家孩子，损自家孩子

**还有一种精神上的“大方”，更要命。**

有些父母把所有的赞美，所有的肯定，都极其“大方”地给了别人家的孩子。

**“你看人家小芳，这次又考了第一！”**

**“你看隔壁小红，多懂事，放学就知道帮家里干活！”**

他们恨不得把人家孩子夸上天，然后话锋一转，对着自家孩子一顿劈头盖脸的打压。

**“你呢？一天到晚就知道玩！”**

**“哭什么哭，一天到晚就知道哭！”**

这根本不是激励，这是人格谋杀。

  


**这么做的父母，不是真的觉得别人家孩子有多好，他只是想通过赞美别人，来贬低孩子，让孩子产生愧疚感，让孩子觉得自己永远欠他的。**

这样，孩子就好控制了。

在这种“大方”的赞美和刻薄的对比下，孩子会怎么样？

会彻底被掏空。

孩子会觉得自己一无是处，无论怎么努力都够不着那个“别人家的孩子”。

**他存在的唯一价值，好像就是用来证明父母有多失败。**

这种精神虐待，比打一顿疼多了，那个伤口，几十年都好不了。

  


# 3、在外人面前过度炫耀孩子

**前两种是把好东西给外人，还有一种“大方”，是把孩子当成好东西，展示给外人。**

孩子成了父母的表演道具。

孩子考了个好成绩，拿了个什么奖，父母能“大方”地请客吃饭，恨不得用高音喇叭在全世界广播。

**饭桌上，把孩子夸得天花乱坠，让他表演个节目，背个唐诗。**

那一刻，孩子会觉得很风光吗？

不，他只觉得窒息。

因为他知道，父母不是为他高兴，是为父母自己。

孩子成了他们炫耀的资本，成了他们满足虚荣心的工具。

  


**这种“大方”的背后，是极度的自私。**

孩子考砸一次试试？

上一秒还在酒桌上吹嘘他，下一秒回到家，就能用最恶毒的语言咒骂他。

**因为孩子让父母丢脸了，孩子的失败，就是父母的失败。**

**在这种家庭，孩子不是他们的孩子，只是他们的“作品”。**

他不能有缺点，不能有情绪，他必须完美，必须一直给父母长脸。

这条路走下去，人会疯掉的。

  


**说到底，这三种“大方”，根子都是一样的：**

**父母内心是匮乏的，是虚的。**

如果我们没有能力从自己身上获得价值感，安全感，就只能拼命向外抓取。

抓面子，抓比较，抓一个能拿得出手的“作品”。

**这是一种生存本能，很可悲，但对孩子来说，这就是灾难。**

如果我们自己也是在这样的家庭里长大的，更要警惕把这种模式遗传下去。

  


别再让孩子内耗了，别再让他问自己“为什么我不值得”了。

**他不是不够好，他只是可能被我们当成了满足匮乏感的工具。**

现在，我们作为父母，要做的第一件事，就是把孩子当回一个“人”。

一个有权利拒绝，有权利犯错，有权利不完美，有权利只为自己而活的，一个普普通通的人。

这很难，我知道。

**但这是唯一的出路。**
