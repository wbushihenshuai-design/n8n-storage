# 那些从不检查作业的妈妈，培养出了什么样的孩子？

如果一个妈妈，把孩子的作业当成自己的KPI，那么她只会收获两个结果：

**要么是一个只会听指令的“伪学霸”，要么是一个用拖延来反抗的“真学渣”。**

没有例外。

当然，我并不是说妈妈要当甩手掌柜，完全不管不顾，那是失职。

我是说，如果你不仅盯着他写，还要帮他查错，甚至帮他把字擦了重写。

你以为你在负责任，其实你在剥夺他的**“痛感”** 。

一旦没有了痛感，他就永远学不会看路。

  


# 01

**谁写作业，谁负责**

很多妈妈陷入了一个误区：

我帮他检查出来了错误，他改正了，他学会了，这次作业全对，老师表扬了他，他的自信心建立了。

这个逻辑链条看似完美，其实第一环就断了。

**是你学会了，不是他学会了。**

当你拿起橡皮擦的那一刻，你就告诉了他一个潜意识信号：

 _你是不完整的，你需要我来修补你。_

 _你是不负责的，最后的防线是我。_

这在心理学上，叫做**“外部控制点”（External Locus of Control）** 。

当孩子习惯了有一个“外部检察官”存在，他的大脑会自动进入省电模式。

反正有人兜底，我为什么要费脑子去自查？

反正有人会吼，我为什么要记得带作业本？

于是，你越是认真负责，他越是漏洞百出。

**你越是追求作业的完美率，他越是丧失对学习的掌控感。**

  


#  02

**所有的“盯着写”，本质上都是不信任**

为什么我们管不住那只想要伸向作业本的手？

说白了，是我们自己焦虑。

我们无法忍受作业本上的那个红叉，觉得那个红叉是打在自己脸上的。

我们把孩子的作业，投射成了自己的面子。

这就是典型的**“课题分离”** 失败。

阿德勒心理学告诉我们要分清：

“这是谁的事？”

作业写错了，被老师批评，在同学面前丢脸，这是**他的事** 。

因为心疼他，或者为了维护完美的妈妈形象，帮他规避了这个后果。

这就像他学走路摔倒了，你为了不让他哭，此后余生都抱着他走。

**剥夺了孩子犯错的权利，就是剥夺了他成长的机会。**

你现在帮他检查出的每一个错字，都是他未来人生里一个个等着爆的大雷。

因为他从未真正体验过“因为粗心而付出代价”的刺痛感。

  


# 03

**不检查，是为了让他长出自己的“肌肉”**

那些真正聪明的妈妈，从不检查作业的“对错”，她们只检查“完成度”。

这里有一个很微妙的区别。

检查对错，你是**监工** ；

检查完成度，你是**合伙人** 。

**我建议家长做一个“无知的旁观者”。**

当孩子把作业递给你，不要拿过红笔开始扫雷。

你要问他：

 _**你自己检查过了吗？**_

 _**你确定没有遗漏了吗？**_

 _**如果明天老师判了错，你能接受这个结果吗？**_

**让他自己签字，让他自己把书包合上。**

如果第二天他被批评了，回来哭鼻子。

这太好了。

这才是教育发生的时刻。

这时候你不要说“你看我早说了吧”，你要说：

**看来这个知识点有点难度，我们一起看看怎么把它啃下来。**

**从“对抗关系”变成“支持关系”。**

**从“我盯着你做”，变成“你需要时我在”。**

  


#  04

**真实的成长，一定是粗糙的**

心理学家爱德华·L·德西在**自我决定论** 中反复强调：

**自主感（Autonomy）是内在驱动力的核心。**

一个被妈妈拿着放大镜审视作业的孩子，是没有自主感的。

他只是一个执行程序的机器。

**我们到底想培养什么样的孩子？**

是一个在那张完美的作业纸下，掩盖着空洞眼神的提线木偶？

还是一个作业本上虽然有红叉，但眼里有光，知道**“这个错误是我自己造成的，我有能力把它改过来”** 的独立个体？

你要允许他是个“红薯”。

红薯刚拔出来的时候，就是带泥的。

那上面的红叉、错题、涂改液的痕迹，就是泥土。

洗干净是需要时间的。

你不能嫌脏，直接给他换个假的水晶红薯摆在那。

那是好看，但那是假的。

**把笔还给他。**

**把错题还给他。**

把那份“可能会搞砸”的焦虑，留给自己消化。

**因为只有他自己走过的弯路，才算数。**
