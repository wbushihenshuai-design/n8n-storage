# 惊醒无数父母的“饿鼠效应”：童年有两样东西，给得越多，孩子未来输得越惨

咱们当父母的，都揣着一颗心：

 _**想把全世界最好的东西都给孩子。**_

 _**生怕他缺了什么，怕他比别人差。**_

但你有没有想过，有时候，我们这种**“倾其所有”** 的爱，可能正在不知不觉中，给孩子的未来挖坑？

这话听着有点刺耳， _**但一个快100年前的科学实验，早就把这个道理说明白了。**_

它就是听起来有点怪，但却特别有道理的——**“饿鼠效应”** 。

  


# 01.一个让人大跌眼镜的实验：吃饱的，居然输给了挨饿的

这事儿得从1925年说起。

美国有个科学家叫麦凯，他做了个特有意思的实验。

 _**他找来一群刚断奶的小老鼠，分成两组。**_

**A组，是“VIP富养组”。**

 _**好吃好喝伺候着，食物堆成山，想吃多少就吃多少，绝对管够。**_

这不就是我们心里最理想的养娃模式嘛，要啥给啥，绝不亏待。

**B组，是“普通穷养组”。**

 _**这组就有点“惨”了，每天给的食物，只有A组的六成。**_

也就是说， _**它们总是处于一种半饥不饱、有点盼头的状态。**_

实验就这么进行着。

按理说，肯定是A组吃得饱饱的，长得又壮又健康吧？

结果呢？

简直让人大跌眼镜。

A组那些吃撑了的**“饱鼠”** ，一个个胖得走不动道，整天懒洋洋的，还老生病。

最后， _**没活到1000天就都挂了。**_

再看B组那些挨饿的**“饿鼠”** ，反而一个个精神头十足，毛色发亮，身手敏捷。

 _**它们的寿命居然翻了一倍，活了2000多天！**_

身体的免疫力，也比那群**“富贵鼠”** 强得多。

 _**后来，科学家们用鱼、兔子、猴子，做了好多类似的实验，结果都差不多。**_

  


#  02.“饿鼠效应”到底在说啥？关键就俩字：节制

这个实验告诉我们一个特别扎心的大实话：

 _**稍微“饿”一点，啥都好；**_

 _**给得太满，反而容易完蛋。**_

你看，问题的关键，根本不在于**“吃”** 还是**“饿”** ，而在于**“节制”** 这两个字。

A组的**“饱鼠”** ，不是被食物毒死的，而是被自己**“啥都能得到”** 的放纵给害死的。

 _**因为什么都来得太容易，它们反而丧失了最基本的生存活力。**_

B组的**“饿鼠”** ，因为总有点得不到， _**反而激发了身体里所有的潜能，活得特别带劲。**_

你再想想我们自己，是不是也一样？

  *  _**拼命加班，拿身体换钱，最后业绩上去了，人也快倒了。**_
  *  _**抱着手机刷个没完，一抬头半天过去了，心里又后悔又焦虑。**_
  *  _**胡吃海喝最爽了，可一上秤，看着自己都来气。**_



我们大人都这么难抵抗**“过度”** 的诱惑，那你想想，如果我们把这种**“无限量供应”** 的爱，给了那个毫无抵抗力的孩子，会发生什么？

  


# 03.给孩子一种“微饿感”，是他将来最大的底气

**看到这儿，你可能会有点不舒服：**

什么意思？

难道要我饿着自己孩子？

不给他吃不给他穿？

 _**别误会，当然不是这个意思！**_

**“饿鼠效应”** 教给我们的，不是让我们对孩子**“抠门”** ，而是要学会在爱里，给孩子留一点恰到好处的**“微饿感”** 。

**啥叫“微饿感”？**

  *  _**它不是痛苦，而是一种“够不着、但跳一跳就能够着”的盼头。**_
  *  _**它是一种“这个我想要，但我得靠自己努力去换”的期待。**_
  *  _**它是一种“我很想现在就玩，但我知道写完作业再玩会更开心”的自控。**_



一个在**“微饿感”** 里长大的孩子，他的内心是丰盈又淡定的。

他不会因为得到一个玩具就得意忘形，也不会因为一次拒绝就哭天抢地。

 _**因为他心里有数，有对生活的掌控感。**_

更重要的是，这种在**“等待”** 和**“努力”** 中锻炼出来的**“抗打击能力”** ，会成为他一辈子的财富。

将来他自己去闯世界，遇到点挫折、受了点委屈，别人可能一下就趴下了，但他能拍拍土站起来， _**因为他心里有股劲儿，知道“没关系，我能搞定”。**_

这跟现在流行的**“鸡娃”** 可不一样。

 _**“鸡娃”是被动地被推着跑，孩子累，大人更累。**_

 _**而“微饿感”，是孩子自己心里有方向，主动想去跑。**_

这股劲儿，才是真本事。

  


# 04.所以，到底哪两样东西，给得越多，孩子输得越惨？

好了，绕了这么大一圈，咱们回到最开始那个问题。

 _**有两样东西，我们总以为是爱，其实给得越多，越是在“害”孩子。**_

##  第1样：给太多的东西，有求必应。

 _**这就好比实验里A组老鼠的无限食物。**_

孩子的玩具堆成山，衣服多得穿不完，任何物质要求都立刻满足。

结果呢？

 _**他很快就对一切都失去了兴趣，觉得什么都理所应当。**_

就像一个吃自助餐吃撑了的人，再看到山珍海味也提不起劲儿。

一个从小不知道**“珍惜”** 和**“努力”** 为何物的孩子，将来怎么可能指望他去创造和奋斗？

  


## 第2样：反应太快，马上满足。

 _**这一点，比给东西的危害还大。**_

  * 孩子一哭，马上抱；
  * 孩子一闹，马上妥协；
  * 孩子想看电视，二话不说就打开......



我们以为这是爱，其实是在剥夺他一项至关重要的人生技能——**“延迟满足感”** 。

说白了， _**就是为了一个更大的糖，愿意先不吃眼前这个小的。**_

 _**这是一个孩子学会自律、学会规划、学会忍耐的开始。**_

如果一个孩子从小就习惯了**“我想要，就必须马上得到”** ，那他的人生，就会被各种短暂的欲望牵着鼻子走，做不成任何需要长期坚持和努力的事。

  


# 05.写在最后

M·斯科特·派克在**《少有人走的路》** 书中指出，一个人要想学会自律，得学会四件事。

 _**推迟满足感，承担责任，忠于事实，保持平衡。**_

为了一个长远的好结果，愿意忍一忍眼前的舒服。

这不就是我们前面聊的那个**“微饿”** 状态吗？

心里总惦记着一个更好的目标，所以现在这点小诱惑，就能扛得住。

 _**就像电影《饥饿游戏》里有句台词，说得特别好：“一点点希望是很好的，而巨大的希望是毁灭。”**_

咱们养孩子也是一个道理。

别一下子把所有东西都塞给他，让他撑得走不动路。

咱们得学着给他一点点**“够得着”** 的希望，让他自己有盼头、有奔头，一直兴致勃勃地往前走。

 _**这种带着一点点“饿”的感觉去努力的状态，才是最有活力的。**_

而这种活力，比我们直接给他任何现成的东西，都珍贵一万倍。
