# 最在意你情绪的人，是你的孩子

周末我窝在沙发上，假装很认真地在看手机，其实脑子里一团乱麻。

**工作上出了个不大不小的纰漏，跟客户反复解释，说得口干舌燥，心里那股火不上不下地堵着。**

我以为自己掩饰得很好，表情平静，呼吸平稳。

女儿在旁边玩游戏，突然放下手里的ipad，穿着拖鞋跑到我身边，一言不发，从她自己的零食罐里拿了一块小饼干，小心翼翼地塞到我嘴边。

我愣住了，问她怎么了。

她看着我的眼睛，小声说：

**「妈妈，你吃块饼干吧，甜的。」**

那一刻我差点哭出来。

**我什么都没说，什么都没表现出来，她却好像什么都知道了。**

  


#  01.孩子总能感觉到你的不开心

**我们成年人，早就学会了一套表情管理。**

在公司，就算被老板骂得狗血淋头，出门也能对同事笑脸相迎；

在朋友面前，心里再苦，也能装作云淡风轻。

我们靠这套面具行走江湖，甚至有时候，连最亲密的伴侣都发现不了我们内心的风暴。

**但孩子不一样。**

**她们好像随身携带了一个情绪探测仪，专门对准自己的爸妈。**

你今天是不是真的开心，你那个笑容的弧度是不是发自内心的，你说话的语调里是不是藏着一丝疲惫，**她们全都能捕捉到** 。

你以为你只是叹了口气，但在她听来，就是一声打雷。

你以为你只是皱了下眉，但在她看来，天可能就要塌了。

**她们不是靠逻辑分析，她们是靠本能去感受。**

因为在她们的世界里，妈妈的情绪，就是天气预报。

**你是晴是阴，直接决定了她心里的温度。**

  


#  02.跟孩子聊天，真诚是必杀技

以前我也犯过错，习惯性地敷衍。

**「妈妈你怎么不高兴了？」**

**「没有啊，妈妈在想事情。」**

**「妈妈你为什么叹气？」**

**「没什么，你看你的动画片吧。」**

后来我发现，这种敷衍像一堵无形的墙，会把她推远。

她会觉得，妈妈的心事，是不可以靠近的禁区。

有一次我答应她去海洋馆，结果临时被领导一个电话叫回去加班，弄到很晚才回家。

一开门，就看到她坐在小板凳上等我，眼睛里全是失望。

  


换做以前，我可能会说：

**「妈妈工作忙，下次再去。」**

但那天，我蹲下来，看着他的眼睛，特别认真地说：

> ❝
> 
> 对不起闺女。
> 
> 妈妈食言了，没有安排好时间，这是我的错，让你白等了一天。
> 
> **妈妈非常非常抱歉。**

我以为她会哭闹，或者生闷气。

结果她伸出小手摸了摸我的脸，说：

**「妈妈你也很累吧。那你明天，可以陪我玩一会儿吗？」**

你看，孩子要的根本不是完美的父母。

他们要的，只是一个愿意放下架子，跟他说实话的普通人。

**你诚实地告诉他你的难处、你的错误、你的疲惫，她不但不会责怪你，反而会变成一个小孩姐，想要反过来保护你。**

**真诚，真的是跟孩子沟通的必杀技。**

  


#  03.孩子是无条件爱妈妈的人

**这个世界上，所有的关系似乎都遵循着某种“等价交换”。**

你工作出错，要承担后果；

你对朋友失约，要花心思弥补；

夫妻之间吵了架，也可能冷战好几天。

这些关系，都需要小心翼翼地经营和维护。

**唯独孩子对你的爱，是不计成本、不求回报的。**

你刚因为她打翻了牛奶而大发雷霆，声音大到自己都觉得刺耳。

她一边掉眼泪，一边还张着小手要你抱。

你因为累了一天而极不耐烦地推开她递过来的玩具，她会默默地走开，但过五分钟，又会拿着你最爱吃的水果，怯生生地凑过来。

**她把你的情绪看得比天大，却总能最快地原谅你的坏情绪。**

这份爱，有时候浓烈得甚至有点烫手。

我们明明做得那么不好，那么不值得，可她转个身，就忘了所有的不愉快，**依旧用全世界最干净、最纯粹的眼神依赖着你** 。

  


我常常在想，等她长大了，有了自己的朋友，自己的爱人，自己的生活，她肯定就不会再这样时刻关注我的情绪了。

她会忙着处理自己的烦心事，会把耐心留给更重要的人。

她不会再因为我叹一口气，就紧张地跑来问我怎么了。

**到那个时候，也许我才会真正明白，这个把你的喜怒哀乐当成全世界的人，其实，也就陪你这么几年。**

**真的。**
