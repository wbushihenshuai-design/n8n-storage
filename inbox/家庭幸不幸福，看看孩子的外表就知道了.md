# 家庭幸不幸福，看看孩子的外表就知道了

上个周末，跟一个很久没见的朋友喝咖啡。

她整个人看起来特别焦虑，手机划来划去，全是各种补习班和学区房的信息。

她愁眉苦脸地问我：

“你说，我到底是该送我儿子去学马术，还是学编程啊？愁死我了，真怕他将来落后别人。”

我没直接回答她，而是把她朋友圈里一张儿子的照片放大，递到她面前。

“你先别想那么远，”我跟她说，**“你仔细看看你儿子的这张脸，你觉得他现在开心吗？”**

照片里的小男孩，眼神有点怯怯的，嘴巴抿得紧紧的，完全是一副小心翼翼的样子。

朋友一下子就愣住了，半天没吭声。

其实我们都一样，总想着砸锅卖铁也要给孩子最好的，以为这就够了。

但我们常常忘了，一个家到底好不好，有没有爱，根本不用问，全清清楚楚地写在孩子的脸上。

不信你看这3个地方，准得很。

  


# 01\. 看看孩子的眼睛，亮不亮

你有没有见过那种眼睛里好像有小星星的孩子？

他们看你的眼神，特别清澈，对什么都好奇，一点都不怕生。

你跟他说话的时候，能感觉到那股劲儿，就是对世界充满希望的样子。

这种眼神里的光，可不是花钱能买来的， _**它来自于一个让他觉得特别踏实、特别安全的家。**_

在这样的家里，爸爸妈妈关系很好。

他们很少大声吵架，更不会把在外面受的气撒到孩子身上。

家里的气氛总是暖洋洋的。

  


孩子捡了块破石头，宝贝似的跑过来问：“妈妈你看这是什么！”

她听到的不是“赶紧扔了，脏死了”，而是“哟，真好看，我们一起看看这是什么石头？”

他把玩具拆得乱七八糟，爸爸看到了，不是一顿骂，而是蹲下来说：“你想知道这里面是啥样啊？行啊，研究完了咱俩一起把它装回去。”

说白了，**父母的耐心和笑脸，就是给孩子眼睛“充电”的。**

反过来，要是一个孩子天天生活在吵架、抱怨和指责里，他的眼神就会慢慢变得躲躲闪闪，没什么神采。

 _**那点光，就在一次次“你怎么这么笨”和“你看看人家”的唠叨里，给磨没了。**_

所以啊，别光顾着给孩子买护眼灯了，先让家里的气氛“亮”起来吧。

  


# 02\. 看看孩子的嘴角，是不是笑着的

我记得看过一个电视节目。

有个爸爸下班回家，没跟往常一样往沙发上一躺，而是先走到厨房，从后面抱了一下正在做饭的老婆，说了句：“老婆辛苦啦！”

这时候，镜头正好拍到旁边自己玩积木的儿子，那小家伙看到这一幕，嘴角就那样不自觉地、甜甜地翘了起来。

**那个画面，真是看着就觉得心里暖。**

孩子的快乐其实特简单。

他们对玩具、对零食可能很快就腻了，但对家里的气氛却敏感得像个小雷达。

一个家最好的风水，真不是房子有多大，而是家里总有笑声。

  


要是爸爸爱妈妈，妈妈也爱爸爸，俩人没事儿还开开玩笑，孩子在旁边看着，他心里就是踏实的，快乐的。

他脸上的笑，是从心里冒出来的，藏都藏不住。

可如果家里总是冷冰冰的，大人之间不说话，或者一说话就吵架，孩子就会变得特别“懂事”。

他会压抑自己的情绪，不敢大声笑。

**他脸上的笑容，也因此变得小心翼翼，** 那种懂事的样子，看着就让人心疼。

**说到底，你给爱人的一个好脸色，就是给孩子的一颗糖。**

想让孩子天天咧着嘴乐，那你得先多对自己的另一半笑笑啊。

  


# 03\. 看看孩子那张脸，有没有“被欺负过的痕迹”

**作家陈丹青，他说过一句话我印象特别深。**

他说，他刚到美国的时候，看到街上的年轻人， _**感觉他们脸上都有一种“没受过欺负”的样子。**_

**啥叫“没受过欺负的脸”？**

不是说这孩子霸道、蛮横，没人敢惹。

而是一种很放松、很自信、很大方的感觉。

 _**有这种气质的孩子，他敢说出自己的想法，不怕说错话；**_

 _**他跟人说话的时候，眼睛是看着你的，很笃定；**_

 _**他知道自己是值得被爱的，所以也懂得怎么去爱别人。**_

  


这张宝贵的脸，完完全全是在家里“养”出来的。

这张脸背后的底气，就是父母给的爱和尊重。

当他跟你顶嘴的时候，你没说“小孩家家懂什么”，而是听他说完；

当他考试考砸了，你没有劈头盖脸一顿骂，而是拍拍他肩膀说“没事儿，咱看看是哪儿没搞懂”；

当他在外面受了委屈，你永远是第一个站出来挺他的人。

**你给他的每一个“没关系”和“我相信你”，都在他心里盖起了一座自信的堡垒。**

我们总说要富养孩子，其实真正的富养，不是给他花多少钱，而是用爱把他养得底气十足，让他心里不缺安全感。

这张“没受过欺负的脸”，就是父母能给孩子最好的名牌，也是他将来在社会上闯荡时，最硬的底牌。

  


# 04.最后想说几句

所以啊，我的朋友们，别再光盯着分数和排名了。

有空的时候，好好看看你的孩子。

看看他的眼睛里，还有没有光； 看看他的嘴角，是不是自然地翘着； 看看他那张小脸上，是不是写满了“我爸妈超爱我”的自信。

我们拼死拼活，不就是想让孩子幸福吗？

其实，**对身边的那个人好一点，让家里多一点笑声，这才是给孩子幸福最简单、也最管用的办法。**

努力吧，让我们都亲手给孩子一张被爱“泡”大的脸。

那才是他这辈子，最宝贵的财富。
