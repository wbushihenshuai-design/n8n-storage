# 无论你的孩子多大，一定要告诉他这7句话，一生受益！（父母收藏）

夜深了，我悄悄走进你的房间，看着你熟睡的侧脸，心里百感交集。

还记得吗？

小时候的你，总爱拉着我的衣角，奶声奶气地说： _**“妈妈，我永远不要长大，我要永远和你在一起。”**_

那时候，我总是一边笑着，一边紧紧抱住你，心里却泛起一阵酸楚。

因为我知道，把你一直护在身后，是我最大的心愿，但让你独自走向远方，才是生命的必然。

 _**爸妈终将老去，无法陪你走完人生的全程。**_

我们能做的，就是将毕生的经验与爱，打包成一个行囊，让你在未来的路上，走得更稳、更远。

孩子，这个行囊里没有金银，只有7句话。

 _**在你未来遇到风雨时，希望它们能成为你的光。**_

  


#  1.生命高于一切，包括尊严和梦想。

孩子，你可能会觉得这是一句正确的废话，谁不知道生命最重要呢？

但爸妈怕的是，当你真的走到绝境时，会忘了它。

 _**当你考试失利，感觉前途无望时；**_

 _**当你被人误解，感到百口莫辩时；**_

 _**当你倾尽所有却一败涂地，觉得尊严扫地时……**_

请你一定记住，这一切都会过去。

 _**没有什么挫折，值得你用生命去交换。**_

留得青山在，不怕没柴烧。

只要你还在，人生就有无限可能。

先爱惜自己的生命，我们才有机会去谈论其他的一切。

  


# 2.最值得依靠的，永远是你自己。

 _**我这样说，不是要你变得冷漠孤僻，不相信任何人。**_

而是希望你明白，父母会老，朋友会有自己的生活，爱人也可能离去。

 _**把人生的希望完全寄托在别人身上，就像在沙滩上盖楼，看似美好，却经不起任何风浪。**_

别人可以是你的伞，但总会有淋湿的肩膀；别人能拉你一把，但路终究要自己一步步走。

 _**你要努力成为自己最坚实的依靠，拥有为自己人生兜底的能力。**_

当你内心强大，能为自己撑起一片天时，全世界都会对你和颜悦色。

  


# 3.自学能力，是你一生最重要的必修课。

 _**你总说：“这些知识，学校都教过了。”**_

可孩子，学校教给你的是一张渔网，而社会这片大海，风浪是瞬息万变的。

 _**今天适用的知识，明天可能就会过时。**_

 _**未来世界，唯一不变的就是“变化”本身。**_

能让你立于不败之地的，不是你背了多少课文，考了多少分数， _**而是你那颗永远保持好奇、不断汲取新知的心。**_

 _**学会自己寻找答案，比拥有答案本身更重要。**_

这门必修课，没有毕业的那一天。

  


# 4.你的选择，是一个自我实现的预言。

 _**你相信什么，就会看见什么；**_

 _**你选择什么，就会成为什么。**_

如果你从心底认定一件事**“太难了，我肯定做不到”** ，那你甚至不会去尝试，失败便成了定局。

可如果你告诉自己：**“我或许可以试试看”** ，你就会开始寻找方法、投入精力，哪怕最终结果不完美，你也收获了宝贵的经验，离成功更近了一步。

 _**人生就是由无数个选择构成的。**_

 _**别让悲观的念头，提前为你的人生判了“死刑”。**_

请选择相信美好的事物，选择相信努力的意义，你的世界，真的会因此而不同。

  


# 5.别怕犯错，那是成长的勋章，不是人生的污点。

**还记得你小时候学走路吗？**

你摔了无数次，每一次都哭着爬起来，看看我，然后又摇摇晃晃地迈出下一步。

你从未因摔倒而放弃走路。

 _**可越长大，你好像越害怕犯错。**_

 _**怕被嘲笑，怕被否定。**_

孩子，我想告诉你，**犯错是人生的常态。**

每一次错误，都是一个路标， _**它告诉你“此路不通”，请换条路试试。**_

真正可怕的，不是犯错，而是 _**在原地一动不动，假装自己永远正确。**_

大胆去尝试吧，爸妈永远是你最坚实的后盾。

我们不怕你犯错，只怕你因为害怕犯错而变得畏首畏尾。

  


# 6.感受不等于事实，它只是你对事实的部分估计。

**这句话可能有点难懂，但至关重要。**

当你被老师批评时，你的感受可能是**“我太差劲了，所有人都看不起我”。**

但事实仅仅是：**“我在这次任务上没有做好，老师指出了我的问题。”**

分不清**“感受”** 和**“事实”** ，是很多人痛苦的根源。

 _**他们被自己的负面感受所绑架，以为那就是世界的全部真相。**_

孩子，请你学着给情绪踩一脚刹车。

当坏情绪涌来时，问问自己： _**“这是我的感受，还是客观的事实？”**_

学会辨别它们，你才能成为情绪的主人，而不是它的奴隶。

  


# 7.不要过度沉迷游戏，虚拟的成就感换不来真实的人生。

**爸妈知道，游戏的世界很精彩。**

它有即时反馈，有清晰的目标，能轻易带给你在现实中难以获得的成就感。

但你必须警惕，它就像一颗甜美的糖，吃多了，会蛀掉你人生的根基。

 _**你可以把它当作放松，但绝不能把它当作逃避现实的港湾。**_

你在游戏里升到满级，关掉电脑后，依然要面对未完成的作业和普通的生活。

真正的成长，永远发生在充满挑战的现实世界里。

 _**请务必成为游戏的主人，而不是被游戏牵着鼻子走的人。**_

  


#  写在最后

孩子，这7句话，或许有些啰嗦，甚至有些严厉。

但请你相信，这背后是爸妈最深沉的爱。

**黎巴嫩诗人纪伯伦曾在一首小诗中，道尽了这种爱的真谛：**

> ❝
> 
> **你的孩子，其实不是你的孩子。**
> 
> 他们是生命对于自身渴望而诞生的孩子。
> 
> 他们借助你来到这个世界，却非因你而来。
> 
> 你是弓，儿女是从你那里射出的活着的箭。
> 
> 弓箭手望着未来之路上的箭靶，
> 
> 他用尽力气将你拉开，使他的箭射得又快又远。

孩子，爸妈愿意做那张被拉满的弓，满怀喜悦地目送你这支**“箭”** ，射向属于你的、我们无法企及的未来。

 _**我们会在原地，为你骄傲，为你祝福。**_
