# 废掉一个孩子最快的方式，就让他用喜欢的方式过暑假

一到暑假，咱们当家长的，心里就开始打鼓。

朋友圈里，一半是碧海蓝天、岁月静好，另一半是打卡、刷题、连轴转。

到底该怎么选？

**是让孩子疯玩两个月，还是狠心逼他一把？**

我们总觉得，爱孩子，就该让他自由，让他快乐。

但你有没有发现一个奇怪的现象？

一个完全**“自由”** 的暑假下来，很多孩子不仅没变开心，反而变得无精打采，甚至一开学就跟不上溜了。

这里面，藏着一个我们平时没注意到的秘密。

  


心理学上管这个叫**“自我损耗”** ，简单来说，就是人的**“意志力电量”是有限的** 。

你可以把孩子的意志力想象成一部手机的电池。

每一次他面临选择时，意志力的电量都在下降：

  *  _**是再睡1小时，还是起床读书？**_
  *  _**是玩一局游戏，还是写一张卷子？**_
  *  _**是刷会儿视频，还是去练会儿琴？**_



……

**每一个这样的问号，都在消耗这块电池的电量。**

对一个自控力还没那么强的孩子来说，一个完全放开的暑假，就像让他拿着一部满电的手机，去玩一天最耗电的游戏。

结果可想而知，用不了半天，电量就耗尽了，手机自动关机。

孩子也一样。

当他的**“意志力电量”** 被无穷无尽的选择耗光后，他就会进入**“低电量模式”** ，只能选择最不费脑子、最舒服的事情来干——**躺在沙发上，无休止地刷视频、玩游戏。**

这不叫自由，这是在 _**“掏空”孩子的精神劲儿**_ 。

  


# 01\. 孩子的“精神电量”，别在暑假就耗光了

如果说意志力是块电池，那一个没有计划的暑假，就像给了孩子 _**一个怎么刷都不会爆的信用卡，让他随便花。**_

**你觉得他会用这张卡去“投资”什么？**

是需要耐心和努力才能看到回报的**“长期理财产品”** ，比如攻克一道难题的喜悦，或是读完一本大部头的成就感？

别傻了。

他只会疯狂**“消费”** 那些能立刻带来快乐的**“垃圾食品”** —— _**短视频的哈哈一笑，游戏胜利的短暂刺激。**_

我身边就有个活生生的例子。

朋友两口子特别开明，暑假让儿子彻底放飞自我。

结果呢？

**那孩子每天都陷在“干点啥好呢？”，这种无休止的纠结里。**

精神头儿全被磨没了，到了下午就蔫头耷脑的，只能靠玩手机来打发时间。

他爸妈特纳闷： _**“给了他所有自由，怎么感觉比上学还累？”**_

 _**因为持续不断的“选择”，比按部就班的“执行”更累人。**_

  


反观另一个家庭，被亲戚们说成是**“虎爸虎妈”** ，给孩子做了张详细到小时的作息表。

外人看着都替孩子窒息，但对那个孩子来说，这反而是种解脱。

他根本不用在**“该干什么”** 上浪费一点儿**“电量”** ，他所有的精神头儿，都用在了刀刃上——比如 _**专心致志地研究一道题，或是在阅读时间里，安安静安心心地看书。**_

假期一结束，两个孩子的差距立马就出来了。

  *  _**一个精神涣散，开学半天都集中不了注意力；**_
  *  _**另一个呢，电量满格，在新学期里轻松跟上老师的节奏。**_



真正的自由，不是你想干啥就干啥，而是你不想干啥的时候，有能力不干。

咱们当父母的责任，不是把选择权一股脑地丢给孩子，而是通过合理的安排， _**帮他屏蔽掉那些耗电的干扰项，保护好他那点宝贵的“精神电量”。**_

  


#  02\. 大脑的“懒人模式”，一旦开启就很难关掉

如果说**“精神电量”** 解释了孩子为什么会失控，那 _**“大脑的懒人模式”就告诉我们，这种失控的后果有多可怕。**_

 _**我们的大脑，其实又聪明又“懒”。**_ 它总喜欢走最省劲儿的路。

学习，就像是吭哧吭哧地爬山。

每一步都很费劲，过程很累，要很久才能看到山顶的风景。

而刷手机、玩游戏，就像是坐滑滑梯。

嗖一下就到底了，刺激又好玩，毫不费力。

你想想，如果一个孩子，连续两个月，天天都在玩滑滑梯，他的身体会发生什么变化？

 _**他的“爬山”肌肉会严重萎缩。**_

  


大脑也一样。

当他习惯了被短视频喂养，习惯了遇到一点困难就马上切换到下一个娱乐项目， _**他的大脑就进入了深度“懒人模式”。**_

等到假期结束，你再让他去**“爬山”** —— _**去啃那些硬邦邦的知识点，他会觉得痛苦万分。**_

这不是他不听话，是他的**“大脑肌肉”** 已经退化了，他失去了深度思考的能力。

暑假，要么是孩子大脑的**“健身房”** ，要么就是思维的**“安乐窝”** 。

在这个大脑发育的关键期，放任他天天**“坐滑梯”** ，等于 _**亲手废掉了他的“学习肌”，这比什么都可怕。**_

  


#  03\. 人生的差距，就是在暑假这条“超车道”上拉开的

咱们都看过赛车比赛，直道上大家速度都差不多，胜负往往就在那几个弯道。

谁能在弯道上处理得好，谁就能实现**“弯道超车”** 。

暑假，就是人生这场比赛里，最关键的一个**“弯道”** 。

它看起来只是休息，但实际上，是拉开差距的**“超车道”** 。

尤其是对我们普通家庭的孩子来说，没有那么多试错的资本，也没有那么多退路可选。

 _**读书和学习，仍然是他们能抓住的、最公平的一根向上攀爬的绳索。**_

  


这个时代最让人焦虑的，不是别人家的孩子多聪明， _**而是那些优秀孩子的父母，更懂得如何科学地帮助孩子努力。**_

他们知道怎么给孩子的**“精神电量”** 充电，怎么避免大脑进入**“懒人模式”** 。

**于是，一个暑假下来：**

  * 你的孩子，痛痛快快玩了两个月，收获了一堆廉价的快乐， _**和开学后跟不上趟的痛苦。**_

  *  _**别人的孩子，可能也玩了，但更有节制，**_ 他们在**“弯道”** 上悄悄加了速，把你的孩子甩在了身后。




 _**假期的结束，不是新学期的开始，而是孩子们之间新的差距的开始。**_

  


#  04.写在最后

为人父母，谁不希望孩子快乐？

但真正的爱，不是满足他眼下每一个突发奇想，而是要帮他拥有在未来创造幸福、选择幸福的能力。

**这种能力的核心，叫“自律”。**

而自律，在孩子小的时候，是需要大人扶一把、推一把的。

这个**“大人”** ，就是我们。

所以，这个暑假，请收起你的于心不忍。

 _**那种“让他玩吧，太可怜了”的心软，其实是一种教育上的懒惰和不负责任。**_

  


请勇敢地为孩子制定一个有松有紧的计划。

这真不是在给他上枷锁，而是在为他的人生搭建一个脚手架，扶着他向上走。

有了这个**“脚手架”** ，他才能省下**“精神电量”** ，去攻克真正的难题；

他才能锻炼出强大的**“大脑肌肉”** ，去攀登知识的高峰。

我知道，这个过程肯定会有孩子的哭闹和你的心软。

但请一定相信，当他长大后，站在一个更高的地方回望今天，他最感谢的，一定是当年那个**“狠心”** 的你，没有让他用最舒服的方式，荒废掉自己宝贵的童年。

 _**真正的爱，不是给他一个舒服的现在，而是给他一个能自己选择的未来。**_
