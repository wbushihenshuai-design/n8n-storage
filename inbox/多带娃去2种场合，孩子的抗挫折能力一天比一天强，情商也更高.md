# 多带娃去2种场合，孩子的抗挫折能力一天比一天强，情商也更高

有位妈妈留言跟我发愁，说她家孩子哪儿都好，**就是太爱哭了，一点小事就“哇”地一下，哄都哄不好，感觉心里特别“脆”** 。

咱们当妈的，都懂这种心情。

看着别人家孩子摔倒了拍拍土就站起来，自家娃却像个小玻璃人儿，碰一下就碎了，心里又急又心疼。

其实呀，孩子心里能不能“扛事儿”，除了天生的性格，跟咱们怎么带他关系特别大。

现在的孩子，啥好东西都不缺，就是缺了点出去“闯荡”的机会。

想让他心里变强大，别怕他摔跤，也别怕他受点小委屈。

**多带他去下面这2个地方“玩”，孩子不知不觉就长大了。**

  


#  01、一个总有“小意外”的地方：大自然

咱们总想把啥事都给孩子安排得好好的，但生活哪有不出错的呀？

大自然，就是教孩子“习惯意外”最好的地方。

为啥呢？

**因为大自然可不听我们的话。**

它那里总有好多我们管不了的事情，但这正好能让孩子变得更坚强。

咱们常说，温室里的小花长不成大树，一个道理。

你带孩子出去玩，肯定遇到过这些事儿：

**本来高高兴兴要去爬山，突然下雨了。**

这时候，我们是生气地回家，还是笑着对孩子说：

“哇！下雨啦！我们穿上雨衣去踩水坑吧！”

孩子会发现，计划变了也挺好玩的嘛！

  


**爬山爬到一半，孩子腿又酸又累，哼哼唧唧想回家。**

这时候，我们是直接把他抱起来，还是拉着他的手说：

“宝贝加油！再走10分钟，山顶上有好吃的哦！”

等他真的靠自己爬上去了，他会觉得“哇！我好厉害！”，那种开心是抱上去体会不到的。

**在小河边用石头搭个小水坝，哗啦一下，又被水冲走了。**

孩子可能会急得直跺脚。

但多试几次，他可能自己就发现了：

“哦！要用大一点的石头才行！”

你看，他自己就想出办法了。

**这些在大自然里遇到的小麻烦，比如淋湿了衣服、觉得累了、搭不好东西，其实都不是坏事。**

这就像在给孩子的心里“练肌肉”，练得多了，心里的小肌肉就越来越壮。

以后他再遇到困难，就不会一下子就哭，而是会想：“没关系，我再试试看！”

  


# 02、一个要跟别人打交道的地方：真实的生活

“情商”这个词听起来挺复杂，说白了，就是**懂自己，也懂别人** 。

知道自己为什么不开心，也看得懂别的小朋友是高兴还是难过。

这本事可不是看书、上课能学会的，**必须得多跟人接触，在生活里一点点地“练”。**

我们身边到处都是帮孩子练情商的好地方，比如：

**带他去菜市场。**

让他帮你拿个西红柿，然后对卖菜的叔叔阿姨说声“你好”。

买完东西，鼓励他对老板说“谢谢”。

一来二去，他会发现，跟陌生人说话没那么可怕，有礼貌别人还会对他笑呢。

  


**带他坐公交车。**

他会看到大家都要排队，不能插队；车上不能大喊大叫，因为会吵到别人；

看到老奶奶上车了，我们可以站起来让她坐。

**这些小事，就是在教他心里要装着别人。**

**带他去图书馆。**

他会知道这里要安安静静的，走路要轻轻的，因为大家都在看书。

**他会明白，在不同的地方，要做不同的事，这就是在学着尊重别人。**

这些地方充满了“人味儿”，孩子在里面看、学、做，慢慢就懂得了怎么跟人相处，这比我们跟他讲100遍大道理都管用。

  


# 03、最后想说

带孩子出门，确实比让他安安静静待在家里麻烦多了。

要洗刷、要收拾，还要操心他磕了碰了。

但请相信，**我们养孩子，就像在院子里种一棵小树。**

咱们当然可以把它当成一盆小花，放在屋里，干干净净的。

但咱们心里，其实更希望他能长成一棵能遮风挡雨的大树，对不对？

小树苗只有经历过风吹雨打，根才能扎得更深，长得才结实。

  


有一句话说得特别好：

**“种一棵树最好的时间是10年前，其次是现在。”**

这句话的意思其实很简单：

就是告诉我们，**别总想着“哎呀，以前我怎么没这么做”，因为最好的时候，就是现在！**

孩子的成长呀，就像**滚雪球** 。

今天他敢自己爬上一块小石头，明天他就可能敢去挑战一个小山坡。

今天的一点点小勇敢，会“粘”上明天的一点点小自信，这个“勇敢雪球”就越滚越大。

所以，别再犹豫啦！

就从这个周末开始，带他去公园跑一跑，踩踩泥，跟邻居小朋友打个招呼。

**只要咱们开始了，就什么时候都不晚！**
