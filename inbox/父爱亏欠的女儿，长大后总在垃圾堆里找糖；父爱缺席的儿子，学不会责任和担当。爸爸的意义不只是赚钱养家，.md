# 父爱亏欠的女儿，长大后总在垃圾堆里找糖；父爱缺席的儿子，学不会责任和担当。爸爸的意义不只是赚钱养家，还是榜样和底气

**我身边有很多朋友，聊起感情问题，聊到最后，总能神奇地绕回到她们的原生家庭，尤其是她们的爸爸。**

这事儿特玄乎，但你仔细一想，又准得吓人。

我越来越觉得，爸爸这个角色，就像是盖房子时打下的地基。

地基要是没打牢，这房子就算盖得再漂亮，里面的人也住得不安心。

稍微有点风吹草动，整个房子都会晃，时间长了，墙上哪儿都是裂缝。

  


# 1、爸爸没给过糖，所以别人给块石头，女儿也当宝

我有个闺蜜，人长得漂亮，工作也上进，但她在感情里，有一个匪夷所思的外号，叫**“渣男回收站”** 。

她交往过的男朋友，有一个算一个，全是那种不靠谱的。

**其中最奇葩的一个，我俩都管他叫“石头男”。**

这男的做了什么惊天动地的事让她死心塌地呢？

说出来你都不信。

有一次下雨，他俩都没带伞，这哥们儿脱下自己那件薄外套，顶在了我闺蜜头上，然后自己淋着雨。

就这么一件屁大点儿的事，我闺蜜感动得一塌糊涂，跟我们复述了八百遍。

**她说：“他把唯一的外套给了我，他真的好爱我。”**

  


我们这些旁观者，白眼都快翻到天上去了。

这不就是任何一个稍微有点风度的男性都会做的事吗？

可她不，她觉得这是稀世珍宝。

后来我跟她深聊，才明白了。

**她从小到大，她爸对她，就是三个字：“别烦我”。**

她考了第一，她爸说“知道了”；

她生病了，她爸说“多喝水”；

她想聊聊天，她爸永远在看电视、玩手机。

**她记忆里，她爸从未给过她任何具体的、温暖的、哪怕是有点傻气的关爱。**

她的人生里，父爱那颗糖罐，一直是空的。

  


所以啊，当“石头男”给了她这么一块廉价的、随处可见的“石头”时，她没见过糖是什么滋味，就立马当成了宝贝。

**一个女孩，如果在童年里没有被父亲好好疼爱过，那她长大后，但凡有个男人对她释放一点点善意，她都会诚惶诚恐，加倍奉还。**

因为她内心深处，根本不相信自己是值得被无条件珍视的。

  


# 2、爸爸没做过示范，所以儿子不知道什么是“男子汉”

说完了女儿，再说说儿子。

闺蜜以前喜欢过一个男生，条件不错，人也温和。

但相处久了，发现他有个致命的问题：

**他像个没有骨头的“软体动物”。**

**她们计划去旅行，从头到尾都是她一个人做攻略、订酒店、找出路线。**

男朋友全程只会说三个词：

**“都行”、“随便”、“听你的”。**

有一次，她们跟餐厅因为上错菜有点小争执，男朋友立马躲到她身后，让她去交涉。

最让她下头的一次，是她俩的电脑同时坏了。

她这边自己拆机箱、清灰、换内存条，手忙脚乱。

  


**男朋友在旁边干嘛呢？**

在百度“电脑坏了怎么办”，然后唉声叹气，说“这可咋整啊，要不买个新的吧”。

那一刻，她真的就够了。

**她说自己不是需要一个全能超人，但至少需要一个能并肩站着，而不是躲在自己身后的男人。**

**她男朋友的爸爸，就是我们父辈里最典型的那种“甩手掌柜”。**

在家里，油瓶倒了都不扶。

他妈一个人活成了一支队伍，而他爸，就是个只负责回家的影子。

她男朋友从来没见过，一个男人是如何解决问题的，是如何承担责任的，是如何在一个家庭里成为支柱的。

**他爸爸没有给他做过任何示范。**

所以，她男朋友不知道什么是“男子汉”。

他以为的“男人”，就是生理上的男性而已。

**他学不会担当，因为他的榜样，一辈子都在逃避担当。**

  


#  3、别只做“会走路的钱包”，孩子要的不是提款机

很多当爹的，特别喜欢说一句话：

**“我天天在外面拼死拼活，不就是为了这个家吗？”**

这话听着特伟大，但其实是一种偷懒。

**把赚钱养家当成自己唯一的功能，** 说白了，就是把最简单的事做了，然后心安理得地忽略了所有更重要的事。

**因为跟花时间、花心思、花情绪去陪伴比起来，给钱，真的是最容易的。**

你有没有发现？

孩子小的时候，你给他买个几百块的玩具，他开心一天。

但你花一个小时，跟他一起用纸箱子造个破城堡，他能记好几年。

  


**你以为孩子需要的是一个ATM，取之不尽的物质满足？**

别搞错了，他们需要的是一个“人”。

一个会笑、会闹、会跟他们聊些没用废话、会在他们搞砸了之后说“没事儿，老爸给你兜着”的活生生的人。

**只提供金钱的父爱，是一种最廉价的父爱。**

它会让孩子觉得，所有的一切，包括亲情，都是可以被量化和交易的。

**当有一天，你这个“会走路的钱包”给不了更多的时候，那你在他心里的价值，也就清零了。**

  


#  4、爸爸给的最好礼物，是让孩子心里有底，不怕事

我爸，是个特别普通的男人。

他没赚多少钱，也没什么社会地位。

**但他给了我一样东西，我觉得比任何财富都重要，那就是“底气”。**

我记得小学时，有一次跟同学打闹，不小心把学校展览室的一个花瓶打碎了。

那花瓶据说挺贵的。

**我吓坏了，老师让我叫家长。**

我一路哭着回家，脑子里想了一万个被我爸痛骂的场景。

结果我爸听完，就问了我一句话：

**“你手划伤没？”**

我说没有。

**他点点头，说：**

“行，人没事就行。别哭了，多大点事儿。走，爸带你去学校，该赔多少钱咱赔，该道歉咱道歉。天塌不下来。”

**就这么几句简单的话，我瞬间就不慌了。**

说到底，爸爸的意义，从来不是一个简单的称谓，也不是一个冰冷的供养者。

他应该是榜样，是玩伴，是孩子面对世界时，最坚实的底气。

**别让你孩子的世界里，爸爸只是一个模糊的形容词。**
