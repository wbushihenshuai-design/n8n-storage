# “你的生日就是母亲的受难日”，这句话听起很不对劲，我发现一个真相：永远不要跟孩子索取感恩

上周，刷朋友圈时，看到一位刚生完宝宝不久的朋友发了动态。

照片里，小婴儿粉嫩可爱，蛋糕精致诱人，一切都洋溢着幸福。

可配的文字却让我心里咯噔一下：

 _**“儿子一周岁啦！去年的今天，是妈妈的受难日，也是妈妈的重生。宝贝，你以后一定要懂得感恩啊。”**_

看着那句扎眼的**“受难日”** ，我竟有些恍惚。

这句话，我们听了多少年？

从长辈口中，从影视剧里，甚至我们自己，也曾在某个瞬间，想对孩子说出这句话，来提醒他自己付出了多少。

我们总觉得，这是在教孩子孝顺，是在让他明白爱的重量。

可那天晚上，我看着身边熟睡的女儿，她均匀的呼吸声像一首安宁的小诗。

我突然前所未有地想明白了一件事：

**为人父母，永远、永远不要跟孩子索取感恩。**

这句话，听起来有点“凉薄”，“甚至有些反直觉”。

但它不是什么育儿的旁门左道，而是一个母亲，在无数个辗转反侧的夜里，想通的、关于爱的真相。

  


# 01.爱是心甘情愿的奔赴，不是需要偿还的债务。

我们总喜欢跟孩子算一笔账。

怀胎十月的辛苦，一朝分娩的剧痛，无数个喂奶换尿布的清晨和深夜……这些都是我们付出的“成本”。

于是，孩子的生日，成了我们提醒他还债的“纪念日”。

 _**“你的生日，就是妈妈的受难日”，这句话的潜台词是：你看，我为你受了这么多苦，所以，你欠我的。**_

可我们是不是忘了？

把这个小生命带到世界上来，是我们自己的决定。

是我，心甘情愿地想去经历这一切，去迎接一个新生命的到来。

那份剧痛，伴随的是初见的狂喜；

那些疲惫，换来的是他趴在我胸口的安宁。

**那是我为人母的勋章，是我人生的一场盛大奔赴，怎么能变成捆绑孩子的债务？**

爱一旦变成了需要计算和偿还的账本，就失去了它最纯粹的温度。

当我们在饭桌上，用一句“我辛辛苦苦做饭，你还不吃”来要求他光盘时，他学会的不是珍惜粮食，而是看眼色；

当我们在辅导作业时，用一句“我为你付出了这么多，你就考这点分”来激励他时，他感受到的不是爱，而是沉重的愧疚。

 _**别把我们的付出，变成孩子的精神负担。**_

真正的爱，不是一张记满了付出的账单，而是一个源源不断的泉眼。

它自然地流淌，不求回响，只为滋养那棵你亲手栽下的小树苗。

  


# 02.被索取的感恩，会变成孩子的枷锁。

你有没有想过，一个总被要求“感恩”的孩子，会活成什么样子？

 _**他会变得小心翼翼，敏感而脆弱。**_

他不敢表达自己真实的需求，因为怕被说“不懂事”、“不知感恩”；

他不敢有自己的主见和梦想，因为他的人生似乎只有一个任务——**报答父母** 。

他活成了一个“懂事”的木偶，用顺从和讨好来表演“感恩”，那份表演的背后，藏着多少压抑和疲惫。

**就像一个朋友曾跟我说的，她从小最怕过生日。**

因为每到那天，全家人都会聚在一起，复述一遍她妈妈当年生产时多么多么不容易。

饭桌上，她要第一个给妈妈敬酒，说一句“妈妈辛苦了”，然后接受所有亲戚“你可要好好孝顺你妈”的叮嘱。

她说：“ _**我爱我妈妈，但我讨厌那种感觉。**_ 好像我是一个罪人，我的快乐是建立在她的痛苦之上的。那份爱，太重了，重得我喘不过气。”

你看，被索取的感恩，就像一把无形的枷锁。

它锁住了孩子自由的灵魂，也让亲子关系变得不再轻松。

我们养育孩子，不是为了培养一个对自己言听计从的“报恩者”，而是为了 _**引导一个独立的灵魂，去自由地探索他自己的世界。**_

当他发自内心地给你一个拥抱，当他兴高采烈地跟你分享学校的趣事，当他笨拙地为你倒一杯水……那份不带任何压力的、纯粹的爱，远比一句被规训出来的“谢谢您”要珍贵一万倍。

  


# 03.真正的感恩，是种出来的，不是要出来的。

**那么，不索取，我们就得接受一个“白眼狼”吗？**

**当然不是。**

恰恰相反，一个家庭里，给孩子最好的感恩教育，不是喋喋不休地提醒他“要感恩”， _**而是父母自己，就活在一个充满感恩的氛围里。**_

是爸爸对妈妈说的那句：

 _**“老婆，今天辛苦你了。”**_

是我们在接受了别人的帮助后，认真地对孩子说：

 _**“你看，叔叔帮了我们大忙，我们要好好谢谢他。”**_

是我们对生活本身的姿态——珍惜一顿饭的美味，感叹一朵花开的美好。

**孩子是家庭里最敏锐的观察家。**

他看到的是父母如何彼此相爱、如何尊重他人、如何热爱生活，他自然会把这份温暖和善意，内化成自己的品格。

他会懂得，爱是相互的，付出是珍贵的。

 _**他不是因为“被要求”，而是因为“被爱浸润”，才长成了一个懂得爱与被爱的人。**_

就像一颗种子，你不能天天把它刨出来，质问它为什么还不发芽。

你只需要给它阳光、空气和水，它自然会在某一天，开出让你惊喜的花。

所以，别再对孩子说“你的生日是我的受难日”了。

  


# 04.写在最后

**我想对我的女儿说：**

宝贝，你的生日，是妈妈的“纪念日”。

 _**是纪念妈妈这个身份的开始，是纪念一个全新的、更完整、更强大的我，和你一起来到这个世界的那一天。**_

妈妈对你最深的爱，不是让你背负着我的辛苦前行，而是希望你放下所有的包袱，轻盈地、快乐地，去过你自己的人生。

去吧，去成为一个快乐的大人。

因为，你的幸福，就是对我所有付出的，最好报答。
