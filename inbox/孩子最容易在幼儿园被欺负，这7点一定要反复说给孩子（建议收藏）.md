# 孩子最容易在幼儿园被欺负，这7点一定要反复说给孩子（建议收藏）

**孩子在幼儿园被欺负，是大家非常担心的问题。**

幼儿园阶段的孩子年龄小，自我保护意识和能力都比较弱，很容易成为被欺负的对象。

**为了帮助孩子更好地应对，以下这7点一定要反复、耐心、用孩子能理解的方式说给孩子听：**

  


>   
> 
> 
> #  1、认识什么是欺负行为
> 
> 要告诉孩子什么是**“欺负”** 。
> 
> 很多孩子可能不明白**“欺负”** 这个词，或者不认为某些行为是欺负。
> 
> **你需要用孩子能理解的语言来描述：**
> 
>   * 如果有人抢你的玩具，不让你玩，还凶你，这就是欺负。
> 
>   * 如果有人打你、推你、踢你、抓你、掐你，让你疼，这就是欺负。
> 
>   * 如果有人骂你，说你坏话，给你起难听的外号，让你不开心，这也是欺负。
> 
>   * 如果有人不跟你玩，故意把你推开，让你一个人玩，让你觉得孤单，也可能是欺负。 **(注意：区分偶尔的不想一起玩和持续的排挤)**
> 
> 

> 
> **强调欺负是不对的、不好的行为。**
> 
> **让孩子明白，被欺负不是自己的错，欺负别人的人才是做错事。**

  


>   
> 
> 
> #  2、大声说不
> 
> **教孩子遇到欺负时，要勇敢地说不。**
> 
> **很多孩子被欺负是因为害怕或者不好意思拒绝。**
> 
> **请告诉孩子：**
> 
> 如果有人抢你的东西，你要大声说：**不行！这是我的！我不给你！**
> 
> 如果有人打你，你要大声说：**不要打我！我不喜欢！**
> 
> 如果有人骂你，你要大声说：**不要骂我！我不喜欢你这样说！**
> 
> **鼓励孩子用坚定的语气和眼神表达自己的不满。**
> 
> **语气弱弱的“不要”可能不起作用，要教孩子用稍微大声、坚定的声音说“不”。**

  


>   
> 
> 
> #  3、远离不友好的孩子
> 
> 告诉孩子，如果对方不听**“不”** ，或者继续欺负你，要立刻走开，远离不友好的孩子。
> 
> **不要和欺负者纠缠，避免冲突升级。**
> 
> **请告诉孩子：**
> 
> 如果他抢你的玩具，你不给他，他还抢，你就把玩具放下，走开，去找老师。
> 
> 如果他打你，你说了不要打我，他还打，你就立刻跑开，跑到老师或者其他小朋友多的地方。
> 
> **强调走开不是害怕，而是保护自己。**
> 
> **让孩子明白，走开是为了避免受到更多的伤害，是一种聪明的做法。**

  


>   
> 
> 
> #  4、告诉老师
> 
> **反复强调遇到欺负，一定要告诉老师。**
> 
> **这是最重要的一点！**
> 
> **幼儿园老师的职责之一就是保护孩子，解决孩子之间的冲突。**
> 
> **请告诉孩子：**
> 
> 如果有人欺负你，你一定要告诉老师，让老师来帮助你。
> 
> 老师是保护你们的，有什么事情都可以告诉老师，不要害怕。
> 
> **可以和孩子模拟情景，练习如何告诉老师。**
> 
> **例如：老师，他抢我的玩具！老师，他骂我！**

  


>   
> 
> 
> #  5、告诉爸爸妈妈
> 
> 鼓励孩子放学后也要告诉爸爸妈妈在幼儿园发生的事情，特别是被欺负的事情。
> 
> 家长是孩子最信任和依靠的人。
> 
> **请告诉孩子：**
> 
> 放学回家，要告诉爸爸妈妈今天在幼儿园发生了什么事情，如果有人欺负你，一定要告诉爸爸妈妈。
> 
> 爸爸妈妈会保护你，会帮助你解决问题，不要自己一个人害怕和难过。
> 
> **家长要营造一个开放、信任的家庭氛围，让孩子愿意和家长分享心事。**
> 
> **不要责怪孩子，要耐心倾听，并给予孩子支持和安慰。**

  


>   
> 
> 
> #  6、学会分享，但不是被迫
> 
> **要强调分享的美好。**
> 
> 分享玩具、食物可以让大家一起玩得更开心，交到更多朋友。
> 
> **分享是一种友好的行为。**
> 
> 分享玩具、食物给小朋友玩，大家会一起更开心，你也会交到更多朋友。
> 
> **但要区分“愿意分享”和“被迫分享”。**
> 
> **教孩子如何礼貌地拒绝分享。**
> 
> **比如：**
> 
> 我现在还想玩一会儿，等我玩好了再借给你好吗？
> 
> 这个玩具我现在不想借给别人玩，下次可以借给你其他的。
> 
> **强调物品所有权和个人意愿。**
> 
> 玩具是你的，你有权利决定借不借给别人。
> 
> 你的感受很重要，如果不想分享，没有人可以责怪你。
> 
> **要教孩子区分小朋友是礼貌地询问是否可以借玩，还是蛮横地抢夺。**

  


>   
> 
> 
> #  7、你不是一个人
> 
> 告诉孩子，被欺负不是他一个人的问题，很多小朋友都可能遇到这种情况。
> 
> **这可以减轻孩子的羞耻感和孤立感。**
> 
> **请告诉孩子：**
> 
> 幼儿园里，有时候会有小朋友欺负人，这很正常，你不是一个人遇到这种情况。
> 
> 老师会帮助所有被欺负的小朋友，不用觉得不好意思。
> 
> **可以分享一些绘本故事或者动画片，讲述孩子如何应对欺负，让孩子学习榜样。**
