# 孩子挨打后，为啥“哭着哭着就睡着了”？知道原因后，你就不忍打了

咱们当父母的，可能都见过一个特别奇怪的现象。

孩子犯了错，你火冒三丈，一顿批评，孩子哇哇大哭。

哭得那叫一个惊天动地，可哭着哭着，声音就小了，最后竟然没动静了。

**你蹑手蹑脚过去一看，好家伙，他居然睡着了，脸上还挂着两道泪痕。**

这时候，我们心里通常会想：

 _**“行了，总算哭累了，睡着了就消停了。”**_

说实话，以前我也这么觉得。

直到后来我才明白，我们所以为的“懂事”，背后藏着的，是孩子说不出口的求救信号。

  


# 01.你以为的哭累了，其实是身体拉下的电闸

这事儿，得从我们的大脑说起。

它其实特别聪明，有个自我保护的开关。

就像咱们的手机，用久了发烫，系统就会强制关机降温一样。

当一个孩子面对的恐惧、痛苦和压力，大到他小小的内心根本承受不住的时候，大脑为了不让自己“烧坏”，就会“啪”地一下，拉下电闸，强制关机。

  


所以，那个哭着睡着的孩子，根本不是我们想的那样，是哭累了犯困了。

**他那是进入了一种叫“创伤性休眠”的状态。**

**说得再直白一点，他不是睡着了，他是疼到“昏”过去了。**

这是他身体能做出的、最后的防御。

你想想，得是多大的害怕和无助，才会让一个孩子的身体，选择用“昏过去”的方式来逃避痛苦？

想到这，心里是不是咯噔一下？

  


# 02.当现实令人恐惧，睡眠是孩子唯一的避难所

咱们大人心情不好的时候，是不是也老说“啥也别说了，让我睡一觉”？

因为在那个当下，现实太难熬了，我们想暂时躲起来。

孩子也是一样，甚至比我们更需要一个“避难所”。

在他眼里，爸爸妈妈就是他的天，是他的全世界。

可突然之间，这个天塌了，这个最温暖的港湾变成了风暴的中心。

他最爱的人，正用最可怕的表情和声音对着他。

  


他能跑到哪儿去呢？

他无处可逃。

唯一的出口，就是“睡着”。

**因为只要“睡着了”，那个愤怒的爸爸、吼叫的妈妈就都消失了。**

那个可怕的世界，总算能暂时按下暂停键。

我听一个心理医生朋友聊过，他说，**如果一个孩子在家里，总是显得很疲惫，或者挨骂后很快就睡着，其实是个信号。**

这个信号在说：

**对他而言，“醒着”这件事，可能太累、太可怕了。**

那一刻，我才明白，我们轻描淡写的一句“至于吗”，在孩子的世界里，可能就是一场需要紧急避难的灾难。

  


# 03.你换来的不是安静，而是孩子放弃挣扎的投降

很多家长觉得，打一顿、骂一顿，孩子就“长记性”了，下次就不敢了。

没错，他确实“不敢”了。

但这种“不敢”的背后，代价太大了。

一开始，他可能还会哭着跟你解释：

 _**“不是我干的！”**_

后来，他发现解释没用，只会换来一句 _**“还敢顶嘴！”**_

再后来，他学“聪明”了。

他发现，哭、闹、反抗，所有这些挣扎，只会让风暴更猛烈。

  


那怎么办？

**唯一的办法，就是放弃抵抗，彻底“投降”。**

那个哭着睡着的孩子，其实是在用行动告诉你：

**“我认输了。我斗不过你，我说不过你。怎么样都行，只要这一切能快点结束。”**

**这种状态，叫“习得性无助”。**

你得到的不是一个听话懂事的孩子，而是一个过早学会了压抑自己、看人脸色、不敢再表达真实想法的小孩。

他的心门，就那样悄悄地，对你关上了。

  


# 04.真正让孩子听话的，不是让他怕你，而是让他信你

**那到底该怎么办？**

难道孩子犯错就放任不管吗？

当然不是。

管，是一定要管的。

**但关键在于，你是站在他对面，还是他身边。**

真正能让孩子从心里“听话”的，从来不是让他怕你，而是让他无条件地信任你。

  


其实很简单，试着转变一下角色：

**从一个“法官”，变成一个“队友”。**

当他把玩具弄坏了，正伤心呢。

“法官”会说：

 _**“早就跟你说要小心！你看，坏了吧！活该！”**_

而“队友”会说：

 _**“哎呀，弄坏了肯定很难过吧？来，我们一起看看，还能不能修好它。”**_

当他考试没考好，拿着卷子不敢回家。

“法官”会说：

 _**“你怎么学的！整天就知道玩！”**_

而“队友”会说：

 _**“没关系，这次没考好，我们一起来分析分析，看看是哪里没弄懂，下次我们把它弄明白就行。”**_

  


你看，当孩子知道，无论发生什么，你永远是那个会帮他、支持他的“队友”，而不是随时会审判他的“法官”时，他才会把心里的门向你敞开。

**别再用你的愤怒，逼孩子按下身体的“关机键”了。**

用你的拥抱和理解，给他一个随时可以安心“重启”的世界吧。
