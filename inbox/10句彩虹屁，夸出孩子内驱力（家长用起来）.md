# 10句彩虹屁，夸出孩子内驱力（家长用起来）

**脑科学家洪兰教授说过：“大脑是可以被语言改变的。它会一直不停地因为外界的需求而改变神经的分配。”**

这句话再次印证了语言的巨大力量，特别是充满能量和方向感的语言，对孩子成长有着超乎想象的影响。

它不是空洞的表扬，而是帮助孩子看见自己、相信自己，从而激发他们由内而外生长的动力，也就是我们说的**“内驱力”** 。

心理学中有很多理论都支持这一点，**比如我们熟悉的“罗森塔尔效应”——你越相信孩子是什么样的，** 并用言语和行动传递这种相信，孩子越有可能朝着那个方向发展。

但很多家长可能会困惑，自己知道要夸，但总感觉夸得很空，孩子根本不领情。

**别担心，今天我要说的“彩虹屁”不一样！**

**它不是无原则的吹捧，而是基于《正面管教》理念，更关注孩子的努力、过程、选择和感受。**

目的是帮助他们建立真正的自信，从内在产生动力，甚至能勇敢地从错误中学习。

**记住，真诚和具体，是夸奖抵达孩子内心的秘密武器！**

**下面这10句“彩虹屁”，请家长们收藏起来，用起来吧！**

它们会帮你发现孩子身上那些闪闪发光的瞬间，并把这份光芒放大！

  


# 1、孩子回家主动写作业

**当孩子一进门就放下书包，主动开始写作业时，** 这可不是件小事！

这意味着他们开始拥有很强的自我管理意识和主动性。

**这时候，你的“彩虹屁”可以这么放，让他知道你有多欣赏这份自觉：**

“哇！你到家就直接开始写作业了，完全不需要提醒！这简直太棒了！我看到你这么主动高效，心里特别高兴，都有点佩服你了！你真厉害！”

  


# 2、做强化逆行思维

**孩子遇到挫折或失败时，** 比如考试没考好，事情没办成，这时绝不是打击他们的时刻。

我们要做的，是引导他们回头看付出的努力，向前看成长的潜力。

**这种“强化逆行思维”能把失败变成养分。**

**你可以这样说，肯定他的过程和未来的可能性：**

“宝贝，这次结果可能不是最理想的，但妈妈看到了你为了做好这件事付出的所有努力和汗水！那些坚持和尝试非常宝贵！我们从这次经历中学到了什么？没关系，下次我们带着这些经验再去尝试，我相信你一定能做得更好、更接近你的目标！”

  


# 3、夸大自律

**有时候孩子心里可能一百个不愿意，但还是凭着一股强大的意志力完成了任务，** 比如不想写作业但最终坚持写完了。

这背后是强大的自律！

**你的“彩虹屁”要显得你有多么不可思议地赞赏这份自我控制力：**

“天哪！妈妈知道你今天有点不想写作业，你心里肯定经过了一番小小的挣扎，但是你竟然还是凭着自己的力量坚持把它写完了！这太酷了！告诉我，你是怎么做到克服那种不想写作业的感觉的？这个秘诀太宝贵了！快教教妈妈呗！”

  


# 4、肯定行为

**避免空洞的表扬“你真棒”，要夸就夸孩子具体做了什么，** 以及这个行为带来了什么样的好结果。

这让孩子清楚知道自己哪里做得好，下次可以继续这样做。

**比如看到他收拾了玩具，把积极行为和结果连接起来：**

“哇！宝贝，妈妈刚才看到你把所有的玩具都整整齐齐地放回了它们的小家里，看，房间一下子变得又干净又宽敞了！你把东西整理得井井有条，做得太棒了！”

  


# 5、敞开心扉

**当孩子选择向你倾诉心里话，甚至是他的“小秘密”或一些困扰时，** 这是多么珍贵的信任！

第一时间要传递的是被他们选择的感激和尊重，让他们感到安全和被看见。

**深情地、真诚地对孩子说：**

“亲爱的宝贝，谢谢你愿意把这么重要、这么贴心的事情告诉妈妈，这说明你非常非常信任我，我感到特别荣幸和开心！被你这么信任，这对我来说意义重大。请放心，我一定会牢牢地帮你保守这个秘密。”

  


# 6、挫折教育

成长路上哪有不遇到坎坷的？

**当孩子面对失败或不如意时，** 我们的角色不是替他们扫清障碍，而是教会他们如何面对，如何从经验中学习，培养他们的韧性。

给他们一份温暖而坚定的力量，让他们相信自己。

**你可以这样鼓励，强调过程和价值：**

“宝贝，没关系！你看，你已经非常非常努力地去做了，过程中肯定也学到了不少新东西，这些尝试和经验就是你最宝贵的财富！这次的经历会帮你找到更好的方法、发现可以调整的地方，妈妈完全相信，下一次你一定能做得更棒，离你的目标更近！”

  


# 7、鼓励表达

**孩子有情绪、有烦恼是很正常的，他们也许会通过抱怨、发脾气来表达。**

这时，先接纳他们的情绪，让他们知道自己的感受是被允许的，再引导他们清晰地表达原因和需求，学习健康的沟通方式。

**温和而坚定地问，表示你想理解：**

“宝贝，妈妈看到你现在有点生气，你愿意慢慢告诉我发生了什么吗？我很想听，想了解你。”

  


# 8、鼓励教育

当孩子挑战一件对他来说有难度的事情时，他们的坚持和努力本身就值得被看见和赞美。就算结果不是完美，过程中学习新技能、克服困难的成长也意义非凡。

**你可以这样强调他的勇气和毅力：**

“哇！宝贝，妈妈知道这件事做起来有多么不容易，它有很多地方都需要花心思、需要你克服困难去学习和尝试。但是你看看，你不但没有被吓倒、没有退缩、没有放弃，还这么认真地一路坚持下来，并且完成得这么出色！你挑战困难的勇气和这份毅力真的太了不起了！妈妈为你感到骄傲！”

  


# 9、夸奖自律

**自律表现在很多小地方，比如约定好看电视的时间，到点了孩子能自己主动关掉；**

**约定好玩游戏的时间，能自己停下来。**

这是自我约束力、责任感和尊重约定的体现，要大力肯定，让他看到自己的靠谱！

**你的“彩虹屁”要凸显他的主动性和可靠：**

“我的天！宝贝，你看看时间到了，妈妈都还没来得及提醒你，你就自己主动把电视关掉了！这说明你特别特别遵守咱们的约定，说到做到，有很强的自我管理能力，非常值得信任。这一点真的非常棒！简直是个自律小达人！”

  


# 10、耐心倾听

**当孩子兴致勃勃地跟你分享他的想法、梦想或者对某件事的看法时，全身心地倾听、理解并给出积极回应，就是最好的肯定。**

这不仅鼓励他表达，也让他感受到自己的思考是被尊重的，从而激发他更深入地探索和思考。

**充满兴趣地回应，并引导下一步：**

“哇！宝贝，听到你分享的这些想法妈妈觉得特别特别有意思！你真的思考得很深入，而且表达得特别清楚，逻辑也很棒，我完全听明白了！这是一个很棒的起点，那接下来，咱们一起想想看，怎么一步一步去把这些很棒的想法变成现实呢？我很期待和你一起探讨！”

  


**看，这些“彩虹屁”是不是不一样？**

**它们不再是空洞的“你真棒”，而是有场景、有细节、有指向的积极反馈。**

它们像一束束光，照亮孩子身上那些可能连他们自己都没注意到的美好品质和行为。

持续地、真诚地运用这样的语言，你不是在“培养”一个优秀的孩子，而是在“看见”并“滋养”孩子内在本来就有的力量和光芒。

**这种力量一旦被唤醒，将驱动他们主动去探索、去学习、去成长，长成他们自己想要成为的样子。**

从今天就开始试试吧！

你夸的每一句“彩虹屁”，都在为孩子积攒闪闪发光的内驱力！
