# 高配得感的孩子，从小就被教会了这件事

你身边有没有这样的人？

**他们就像自带灵气，往那一站就让人觉得舒服、自信。**

别人夸他一句，他大大方方说“谢谢”，而不是诚惶诚恐地解释“没有没有，运气好而已”；

哪怕搞砸了事情，他也能很快爬起来，把“事”和“人”分开，不会觉得自己是个废物。

这种特质，在心理学上有一个的词，叫**高配得感** 。

**简单说，就是一种深信“我值得拥有美好事物”的底层底气。**

我观察了身边很多所谓优秀但活得很累的大人，发现他们和那些松弛的孩子之间，最大的区别不在智商，也不在才艺。

而在于从小是否被教会了一件事：

**你的存在本身，就是价值。**

今天，想跟你们聊聊，如何养出一个高配得感的孩子，以及，如何重新养育那个曾经小心翼翼的自己。

  


# 1、高配得感，不是被夸出来的

很多父母有个误区，觉得要让孩子自信，就得拼命夸。

“你真棒！”

“你最聪明了！”

**其实，这种空洞的表扬，反而可能摧毁配得感。**

因为孩子很敏锐，他会把“爱”和“优秀”挂钩。

他潜意识里会觉得：

 _只有我考了100分，妈妈才爱我；_

 _只有我乖，我才配得上那个玩具。_

真正的配得感，来自于**无条件的接纳** 。

美国心理学家纳撒尼尔·布兰登在《自尊的六大支柱》书里提到过一个概念：

**自我接纳** 。

**意思是，我接受我所有的优点，也接受我的笨拙、我的失误、我的坏脾气。**

这就要求我们在孩子“不可爱”的时候，给出的反馈至关重要。

  


**举个生活里的常见场景：**

孩子不小心打碎了一个昂贵的花瓶。

低配得感的家庭反应往往是：

“你怎么这么笨？这很贵的你知道吗？”

**哪怕没说出口，眼神里的嫌弃也藏不住。**

这时候，孩子接收到的信号是：

**我不够好，我是个麻烦，我不配被原谅。**

而高配得感的家庭，第一反应通常是：

“吓到了没？碎了就碎了，咱们一起收拾。”

这时候，孩子学到的是：

**我犯了错，但我依然是被爱的，我有能力解决问题。**

这种安全感，就是高配得感的地基。

  


# 2、敢于表达需求，是孩子自带的权利

**我发小就是个典型的“懂事”孩子。**

小时候想吃冰激凌，不敢说；

想要那个洋娃娃，被大人问喜不喜欢时，违心地摇摇头。

长大后，这种模式延续到了职场和婚姻里。

不敢提加薪，怕老板觉得她贪婪；

老公让自己受委屈了，不敢说，怕对方不高兴。

说真的，这种日子过得很内耗。

那些高配得感的孩子，从小就被允许做一件事：

**大声说出自己的需求，且不为此感到羞耻。**

当孩子说“妈妈我想要这个”时，如果我们暂时买不起，或者不想买，可以诚实地告诉他原因，但千万不要羞辱他的欲望。

别说：“你怎么这就知道要东西？家里没钱给你造。”

试着换一种说法：“妈妈看到你很喜欢这个，但这周我们的预算超了。我们可以把它加进心愿单，下次节日的时候作为备选，好吗？”

要让孩子明白：**你的欲望是正常的，你的需求被看见了。哪怕暂时没得到，也不是因为你“不配”。**

  


#  3、父母的状态，就是孩子的天花板

**这也是我想说的最重要的一点。**

我们很难给到孩子自己身上没有的东西。

如果一个妈妈，自己买件几百块的衣服都要纠结半个月，吃饭总是把好肉留给孩子，自己吃剩菜，口头禅是“妈妈这就凑合用用”......

那孩子很难拥有高配得感。

因为他看到的榜样，是一个“低价值感”的生命状态。

**他会因为妈妈的付出感到愧疚，一个背负着愧疚感的孩子，是不敢让自己过得太幸福的。**

所以，想养出高配得感的孩子，最好的捷径是：

**你自己先成为一个高配得感的人。**

如果你也是那个从小被打压、习惯性自我怀疑的成年人，请从现在开始，对自己进行一场“再教育”。

**哪怕是日常的小事，也可以刻意练习：**

别人夸你今天气色好，别说“哪有，昨晚没睡好”，试着微笑着说**“谢谢，我也觉得今天状态不错”** 。

忙了一周，去吃顿好的，告诉自己**“我这么努力，值得这顿大餐”** ，而不是边吃边心疼钱。

遇到不合理的对待，试着温和而坚定地表达愤怒，告诉自己**“我的感受很重要”** 。

  


# 4、写在最后

我很喜欢一句话：

**“除了身体的病痛之外，一切痛苦本质都是虚幻的，你的价值观决定了你会吃哪一种苦。”**

高配得感，不是傲慢，也不是目中无人。

它是一种内心深处的笃定，相信自己值得被爱，值得美好的生活，值得因为存在本身而获得掌声。

教育的本质，是一棵树摇动另一棵树。

当你开始发自内心地爱自己、款待自己，你的孩子自然会感受到：

**原来，生命是可以这样舒展、鲜活、理直气壮地绽放的。**

这种力量，比任何说教都管用。
