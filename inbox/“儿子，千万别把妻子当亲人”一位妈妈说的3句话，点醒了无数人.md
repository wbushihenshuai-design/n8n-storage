# “儿子，千万别把妻子当亲人”一位妈妈说的3句话，点醒了无数人

上个周末，儿子从大学放假回家。

晚饭后，我给他收拾行李，他靠在门边，有一搭没一搭地跟我聊着学校里的趣事，脸上是藏不住的少年意气。

看着他高出我一头的挺拔身影，我心里一阵恍惚。

那个曾经连路都走不稳、整天跟在我身后要抱抱的小不点，竟已长成了一个即将撑起一片天的大人。

他临走前，我拉住他，鬼使神差般地，郑重其事地说了一句：

 _**“儿子，你记住，以后结了婚，有了自己的家，千万、千万别把媳妇当亲人。”**_

空气瞬间凝固了。

儿子脸上的笑容僵住了，眼神里满是错愕和不解，他大概在想，妈妈是不是今天受了什么刺激，怎么会说出这种**“挑拨离间”** 的话。

我看着他一脸困惑的样子，没忍住，笑了。

拍了拍他的手，示意他坐下。

**“你是不是觉得妈在开玩笑，或者提前给你打预防针，让你别娶了媳妇忘了娘？”**

我轻声说：**傻孩子，妈怎么会是那种妈妈。**

你还记得吗？

小时候你肺炎住院，烧得迷迷糊糊，整晚都攥着我的手指头，嘴里念叨着妈妈在，我就不怕。

那一刻，我觉得我是你的全世界，能为你遮挡所有风雨。

我知道，我曾是你生命里最重要的人。

 _**但我也深深地明白，这份“最重要”是有有效期的。**_

总有一天，你会遇到一个女孩，你会心甘情愿地把你的全世界，双手奉上给她。

而妈妈的任务，就是教会你，如何去守护好你的新世界。

所以，我今天想跟你说的这3句话，不是什么婆媳关系的权谋，而是一个母亲，也是一个过来人，想让你真正幸福的肺腑之言。

  


# 第1句：“亲人是退路，但爱人是征途。”

 _**儿子，你可能会觉得，“亲人”这个词，代表着最深的羁绊和信任。**_

没错。

 _**但也正因为是亲人，我们才常常变得“有恃无恐”。**_

你会心安理得地把臭袜子扔给妈妈洗，因为你知道我不会真的生气；

你会在外面受了委屈，回家冲我发一通无名火，因为你知道我一定会原谅你。

亲人，是我们卸下所有伪装和客气的地方，是我们的退路和港湾。

 _**但你的妻子，她不能成为你的“退路”。**_

 _**她是你选择并肩前行的“征途”。**_

恋爱时，你为她排几个小时的队买一杯奶茶，她皱一下眉你都心疼得不行。

可一旦那张结婚证到手，很多男人就像完成了任务，迅速切换到了**“亲人模式”** —— _**耐心没了，脾气上来了，觉得“都老夫老妻了，还计较什么？”**_

可你想过没有，她不是你血脉相连、打断骨头连着筋的亲人。

 _**她没有义务无条件包容你的懈怠和理所当然。**_

 _**她是一个被父母捧在手心里长大的姑娘，因为爱你，才选择离开自己的港湾，来到你的世界里，陪你一起乘风破浪。**_

你对亲人可以有万般的随意，但对爱人，请保留最初的那份敬畏和珍惜。

你对她说一句**“老婆辛苦了”** ，她也许就能乐呵呵地多做两个菜；

你睡前给她一个拥抱，比送任何昂贵的礼物都更能让她安睡。

 _**别把她的付出当成习惯，别把她的隐忍当成理所当然。**_

爱情这趟征途，最怕的不是路远， _**而是一个人走着走着，发现另一个人早已把“同行”当成了“负重”。**_

  


#  第2句：“你对她的态度，就是你家的温度。”

以后你也会有自己的孩子。

你会像所有父亲一样，想拼尽全力给他最好的生活。

**但儿子，妈妈要告诉你一个过来人的经验：**

一个家庭里，给孩子最好的教育资源，不是昂贵的学区房，也不是报不完的兴趣班， _**而是爸爸对妈妈的态度。**_

你有没有想过，孩子其实是家里最敏感的**“天气预报员”** ？

如果你和妻子总是争吵，家里气氛冰冷，孩子就会变得小心翼翼，过早地学会察言观色，深怕自己一不小心就点燃战火。

那种孩子，你看他总是很**“懂事”** ，但那份懂事里，藏着多少不安和委屈啊。

反过来，一个充满欢声笑语的家是什么样的？

 _**是你下班回家，先过去抱抱忙碌了一天的妻子；**_

 _**是你在饭桌上，夹一块她爱吃的肉放进她碗里；**_

 _**是你们俩偶尔开着玩笑，互相逗乐。**_

孩子看在眼里，他会觉得安全、温暖。

他从你身上，学到的不是如何用吼叫解决问题，而是如何用爱去表达情感。

你疼爱他的妈妈，他才会从心底里感到安稳；

你尊重他的妈妈，他才会懂得如何去尊重他人、尊重爱。

你对她的每一句呵斥，都是在孩子心里刮起的一场寒风；

你对她的每一次温柔以待，都是在孩子心里播下的一颗爱的种子。

所以，儿子，别怕自己当不好爸爸。

 _**你只需要记住，好好爱你的妻子，你的家，就永远是春暖花开的温度。**_

  


#  第3句：“别把她当亲人，因为家是用来合作的，不是用来扶贫的。”

这句话听着有点重，但话糙理不糙。

**为什么很多婚姻会累？**

是因为一方把另一方当成了**“功能性亲人”** —— _**一个免费的保姆，一个不拿薪水的管家，一个理应为这个家无私奉献的“扶贫对象”。**_

他会想：**“我在外面赚钱这么辛苦，你不就做做家务、带带孩子吗？”**

他把家务的琐碎、育儿的辛劳， _**看作是妻子的“分内之事”，是她作为“亲人”应尽的义务。**_

但儿子，婚姻不是一场分工明确的交易，而是一场同舟共济的合作。

  * 她愿意为你洗手作羹汤，是因为心疼你的胃，不是因为她的天职就是厨娘；

  * 她愿意半夜起来无数次给孩子喂奶换尿布，是因为母性的本能和对你们这个家的爱，不是因为她精力旺盛不知疲倦；

  * 她愿意包容你的臭脾气和坏习惯，是因为她心软，还爱着你，不是因为她真的不委屈、没感觉。




 _**千万别做一个只会索取，却吝于付出的“巨婴式”丈夫。**_

当你在沙发上打游戏，请回头看看她在厨房里忙碌的背影；

当她因为孩子的哭闹而精疲力尽时，请主动搭把手，而不是不耐烦地说一句**“你怎么连个孩子都搞不定”。**

 _**每一次沉默的付出，都在消耗着她的热情；**_

 _**每一次被忽略的委屈，都在冷却着她的真心。**_

别等到她有一天，攒够了失望，平静地对你说**“我们分开吧”** ，你才追悔莫及。

  


**我看着儿子若有所思的脸，最后补充道：**

儿子，妈妈说这些，不是让你去怕老婆，更不是让你忘了妈。

恰恰相反，正因为我是你最亲的人，我才必须告诉你这些。

妈妈也曾是别人的儿媳，也曾是你爸爸的妻子。

我太明白一个女人在一个家里，最渴望的是什么，最害怕的又是什么。

 _**这世界上所有的爱都指向相聚，唯有父母对孩子的爱，指向分离。**_

妈妈对你最深的爱，不是把你永远捆在身边，而是教会你如何去爱你选择的那个人，然后体面地退出，看着你们的小家幸福美满。

所以，儿子，千万别把妻子当成理所当然的亲人。

 _**亲人，是你的来处，是血缘给你的底气；**_

 _**而爱人，是你的归宿，是你后半生幸福的源泉。**_

你得像对待最珍贵的合伙人一样去经营你们的关系，像对待最耀眼的明星一样去欣赏她的光芒。

你把她捧在手心，她才能，也愿意，把整个家，都温暖在你心里。

 _**去吧，去好好爱那个，决定把余生都托付给你的姑娘。**_
