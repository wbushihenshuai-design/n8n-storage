# 孩子哄着哄着就变优秀了，这8句话一定要经常说（关于激励技巧）

很多父母养孩子，不知道怎么激励。

家庭作业、课外活动、备考......无论在什么情况下，你如何鼓励孩子都会产生很大的影响。

今天，我分享可以在日常生活中立即运用激励孩子的8句话。

  


# 1.不要说“你为什么做不到？”，而要说“你已经做了这么多！”

消极的话语会让孩子灰心丧气。

与其问“你为什么犯错？”，不如先找出他们做得好的地方，然后告诉他们“你已经做了这么多”或“你快完成了”。

安全感会给孩子再次尝试的力量。

  


# 2.赞扬努力本身，而非结果。

与其说“你得了100分，太棒了”，不如说“你今天坚持到了最后，真了不起”，这更能体现努力的过程。

**只评价结果会让人觉得“如果失败了，一切都毫无价值”。**

**认可努力本身，能够培养即使失败也能重新振作的能力。**

  


#  3.支持你的孩子，问他们：“我和你一起做这个好吗？”

当你的孩子不愿意坐在书桌前时，不要责骂他们：“快点！”，试着支持他们，说：“我和你一起看一会儿好吗？”

**有大人在旁边鼓励他们，有时能让他们出奇地容易上手。**

  


#  4.问问孩子“接下来想做什么？”，让他们展望未来。

与其问他们“你做作业了吗？”之类的问题，不如和他们一起想想未来想做的有趣的事情，比如问他们“做完作业后想做什么？”。

**这样能激励他们采取行动。**

**在心里设定一个有趣的目标来鼓励他们也很有效，** 比如“做完作业我们一起吃点零食吧”。

  


# 5.犯错时，不要说“你怎么又错了？”，而要说“这次又学到了新东西！”

**当孩子把事情搞砸时，指责只会让他们害怕尝试。**

与其说“我早就告诉过你了”，不如把错误看作是学习的机会，对他说“没关系，我们看看从中学到了什么”。

这会把“失败”重新定义为“成长”，让孩子敢于探索，不怕犯错。

  


# 6.面对难题，问“你有什么想法吗？”，而不是直接给答案。

当孩子遇到困难向你求助时，不要立刻说“你应该这样做”。

试着把问题抛回去，问他“关于这个，你有什么好主意吗？”。

让他先思考，哪怕想法很幼稚。

**这传递的信号是“我相信你有能力解决问题”，能够极大地培养他的独立思考能力和自信心。**

  


#  7.接受孩子的情绪，说“我看到你很难过”，而不是“不许哭”。

孩子的情绪是真实的，压抑并不能让它消失。

当孩子因为一点小事哭泣时，不要说“这有什么好哭的？”，试着蹲下来，看着他的眼睛说“妈妈知道你现在很难过”。

先接纳情绪，再处理问题。

**这会让孩子觉得被理解和被爱，而不是被评判。**

  


#  8.不只在成功时，更要在平时说：“我就是喜欢你本身。”

不要只在孩子取得好成绩时才拥抱他，这会让他误以为爱是有条件的。

试着在任何一个日常的时刻，比如一起散步时，告诉他“有你陪着妈妈，真好”。

**这传递的信息是：**

**你的价值不在于你的成就，而在于你本身。**

这种无条件的爱，是孩子自信心最足的底气。

  


**我们的语气和措辞会极大地影响孩子的学习动力。**

重要的是关注他们做得好的地方，认可他们的努力，并给予他们对未来的希望。

**父母的鼓励可以激励孩子“去尝试一下！”**
