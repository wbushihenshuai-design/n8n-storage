# “别把女孩打扮得太好看”，心理学家的劝告，家长要听进去

我最近越来越认同一个观点。

**就是养女儿，千万别把她打扮得太好看。**

这听着很奇怪，对吧？

哪个当妈的不希望自己女儿漂漂亮亮的，跟个小公主一样。

我以前也这么想，疯狂给她买各种公主裙，扎好看的辫子，出门收获一堆“哇，你女儿好漂亮啊”的夸赞。

说实话，我心里特美。

但后来，我发现事情完全不对劲了。

**因为你会发现，全世界对她的善意，几乎都只建立在“好看”这件事上。**

  


亲戚朋友来了，第一句话永远是：

**“宝宝今天真漂亮。”**

小区里的阿姨见了，永远是：

**“这小姑娘，长得跟个洋娃娃似的。”**

幼儿园老师的评语，也总有那么一句：

**“是个文静又漂亮的小女孩。”**

漂亮，漂亮，漂亮。

好像她整个人，就只剩下漂亮这一个优点了。

我女儿搭了三个小时的乐高，没人看见。

她磕破膝盖自己爬起来，没人看见。

她能讲一个很复杂的科学小故事，也没人看见。

他们只看见她那条裙子，她头上的蝴蝶结。

  


于是，我女儿自己也开始变了。

她会因为一点泥点弄脏了新裙子而大哭，她不愿意去玩滑滑梯，因为会弄乱头发。

她做不出数学题，第一反应不是思考，而是沮丧，觉得“我不行，我不好看了”。

我整个人都快窒息了。

**我突然想明白了，把一个女孩从小就往“好看”这条绝路上一路猛推，是这个世界上最残忍的事情。**

好看，是最肤浅，最不靠谱，最经不起推敲的东西。

**它完全是外界的评价，是被动的。**

今天有人夸你，明天就可能没人理你。

它依赖的东西太多了，依赖衣服，依赖发型，依赖年轻的胶原蛋白。

这东西根本不属于你自己，是需要别人点头才能成立的价值。

  


你把她所有的自信都建立在这上面，你图什么呢？

这跟把房子建在沙滩上，有什么区别？

一个浪打过来，就全完了。

**那些真正内核的东西，比如坚韧，比如智识，比如解决问题的能力，这些才是一个人安身立命的根本。**

这些东西，不好看，甚至很“粗糙”。

你需要她满头大汗地去思考，需要她一次次地失败，需要她不畏惧别人的眼光，需要她拥有掀桌子的勇气。

以上这些，都不需要漂亮的裙子，但是需要神气和魂魄，和百折不挠的意志力，都靠着精神层面的东西在撑住。

  


所以，我开始做出改变。

公主裙？

收起来了。

我现在就给她穿方便跑跳、耐脏耐磨的运动裤和T恤。

我看着她穿着这些衣服在泥地里打滚，在草地上奔跑，我觉得这比她穿着公主裙安静地坐着，要好看一万倍。

有人再夸她漂亮，我会立刻打断，然后说：

“是吗？我觉得她刚才独立拼完那个模型更酷！”

或者说：“谢谢，她最近还学会了自己修玩具，动手能力特别强。”

我在干什么？

**我在拼命地，把外界和她自己对她的注意力，从她那张脸上，挪到她的脑子里，她的行动上，她的心里。**

我要让她清晰地知道，你的价值，是你做了什么，而不是你长得什么样。

  


**我猜很多女孩长大后，活得特别累，战战兢兢，就是因为从小被“好看”这个评价体系绑架了。**

她们不敢胖，不敢老，不敢不迎合。

因为一旦“不好看”了，她们就觉得自己一无所有了。

然后就开始疯狂地往脸上投资，往身材上投资，陷入一种永无止境的焦虑。

  


**一个人的精力是有限的，你耗在这里，就没法耗在别的地方了。**

当你的所有感官和思考，都只用来审视自己的外表时，就是你人生走下坡路的开始。

再好的伴侣和再多的钱也救不了你。

**说到底，我不是要让她变得不好看。**

**而是要让“好看”，成为她所有优点里，最不值一提的那一个。**

等她长大了，别人夸她“你真漂亮”的时候，她能发自内心地、平静地回一句“谢谢”，然后心里想的是：**“呵，我厉害的地方，可多着呢。** ”
