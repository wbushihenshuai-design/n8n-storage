# 一定要从小教会孩子3件事：要东西、敢拒绝、会转弯

我经常带娃在小区边公园里玩，就特别喜欢蹲边上看一群孩子互动。

真的，小孩的社交，就是我们最原始、最不加掩饰的社会实验。

**看久了你就发现，很多孩子的人际关系，从三岁起就埋下了各种各不合群的种子。**

你以为是性格问题？

是“这孩子天生就内向”或者“这孩子脾气不好”？

压根不是。

**绝大部分的别扭、吵架、受委屈，都不是性格问题。**

**就是纯粹的沟通能力。**

硬邦邦的。

**尤其是这3件事，有没有人教，区别太大了。**

  


#  1、要东西

**你去看小区里，那些玩不到一起去的孩子，矛盾是怎么开始的。**

大概率是，一个孩子想玩另一个孩子的玩具。

你听他是怎么说的。

「给我玩！」

「你的车给我！」

三个字，五个字，像个小领导。

注意，这不是礼貌问题，**不是加个「请」和「谢谢」就能解决的** 。

**这是沟通方式的出厂设置就有问题。**

他觉得，我要，所以我说，你就得给。

逻辑是直挺挺的。

对方不给，他就崩了。

要么上手抢，要么哇哇哭，喊**“妈妈他欺负我”** 。

被要东西的那个孩子呢？

也很懵。

你谁啊，凭什么命令我？

**这么说话的孩子，最后会走向两个极端。**

要么，他发现这招管用，爸妈爷奶都听他的，于是变得**越来越霸道** ，没人愿意跟他玩。

要么，他发现这招不管用，一开口就要被别的小朋友讨厌，他就怕了，**他就不敢说话了** 。

他不是内向，他是在沟通上吃了太多亏，习得性闭嘴。

**真的，好好要东西，是孩子社交的起手式。**

  


#  2、敢拒绝

光会要还不行，另一头的问题更要命。

**就是不敢拒绝。**

很多孩子，自己心爱的玩具，被别的小朋友一把拿走，他第一反应是愣住，然后眼巴巴地看着。

他心里一百个不愿意，但嘴巴像被胶水粘住了。

**为什么？**

因为他从小听得最多的一句话就是：

**「你是哥哥，要懂得分享。」**

分享，成了一种无法反驳的规矩。

不分享，就是小气，就是坏孩子。

这种标签太重了，孩子背不起。

于是，他宁愿委屈死自己，也要当个「好孩子」。

**更可怕的是什么？**

**是那种憋了半天，终于鼓起勇气拒绝的孩子，一开口，火药味冲破天际。**

「不给！这是我的！」

「你走开！我讨厌你！」

你看，他没有中间地带。

要么憋死，要么炸了。

因为他脑子里没有一个叫「温和地拒绝」的概念。

**他不知道可以说：**

**「这个我现在想玩，待会再给你好吗？」**

他被剥夺了拒绝的权利，自然也学不会拒绝的技巧。

**最后呢，要么在讨好中失去自我，要么在暴怒中失去朋友。**

  


#  3、会转弯

这个就更少了，属于高阶能力。

什么叫会转弯？

**就是一条路走不通的时候，知道换条路。**

**我见过一个小孩，教科书级别的。**

他想玩另一个孩子的恐龙，那个孩子正玩在兴头上，直接说「不给」。

场面一度有点僵。

那个想玩恐龙的孩子，没哭也没闹，他看了看自己手里的奥特曼，说：

**「我用我的光辉赛罗跟你换，他是最厉害的，能打败哥莫拉。你的恐龙也很厉害，他们可以做朋友。」**

你猜怎么着？

成了。

两个孩子高高兴兴地交换了玩具。

这就是会转弯。

大部分孩子，只会一条道走到黑。

「我要玩恐龙」

「我就要玩恐龙」

「你为什么不给我玩恐龙」，这是一个死循环。

他们被自己的执念卡住了。

会转弯的孩子，是在解决问题。

**他知道自己的目标是「玩到恐龙」，而不是「逼对方就范」。**

**他能看见对方的需求，也能利用自己的资源。**

这哪里是情商，这简直是商业谈判的天赋。

  


说到底，这3件事，要东西、敢拒绝、会转弯，本质上都是一回事：

**看见自己，也看见别人。**

**知道自己的需求，也懂得尊重别人的边界，还能在中间找到一条路。**

这些东西，学校里没人教。

很多时候，我们自己都没活明白，现在要教孩子。

但没办法，你不教，就会有人用更残酷的方式去教他。

与其等他头破血流了再自己悟，不如我们早点，把这些最基本的社交技能，掰开揉碎了，塞给他。

记住，别让孩子活成一座孤岛。
