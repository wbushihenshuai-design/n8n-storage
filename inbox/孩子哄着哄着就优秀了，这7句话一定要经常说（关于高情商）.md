# 孩子哄着哄着就优秀了，这7句话一定要经常说（关于高情商）

**成年人的世界里，对“高情商”有着极深的误解。**

很多人以为，高情商就是圆滑，就是八面玲珑，就是委屈自己让别人舒服。

简直是胡扯。

**那不叫高情商，那叫“讨好型人格”。**

真正的高情商，从来不是靠压抑自己来实现的。

而是**让自己舒服，同时允许别人舒服，并且在两者冲突时，守得住界限。**

遗憾的是，我们这一代父母，很多自己就是“讨好型”的受害者。

在职场上不敢拒绝无理要求，在亲密关系里只会隐忍。

然后，又把这种焦虑传给了孩子。

**天天教孩子要“听话”，要“懂事”，要“大方”。**

结果呢？

养出了一堆外表乖巧，内心自卑的孩子。

如果你不想让你的孩子未来活得那么累。

请哪怕是死记硬背，也要把下面这7句话，刻进你们的日常对话里。

  


# 01

**你不需要让所有人都开心，你最重要的任务是先照顾好你自己的感受。**

很多家长喜欢说：

 _你把玩具分给弟弟，弟弟就开心了。_

这句话的潜台词是：

**别人的情绪，比你的感受更重要。**

长大后，他们就会变成那种在办公室里累死累活，还要看别人脸色的烂好人。

你要告诉他，善良的前提，是自我保全。

只有当他自己的杯子满了，溢出来的那些爱，才能滋养别人。

先爱己，再爱人。

顺序不能乱。

  


# 02

**你现在很难过对不对？那做你自己的好朋友，你会对自己说什么安慰的话？**

孩子一哭，家长的第一反应往往是去哄，去解决。

要么说“别哭了”，要么说“妈妈给你买糖吃”。

但这剥夺了孩子一项核心能力：

**自愈力。**

你不可能哄他一辈子。

你要做的，是让他学会把自己“剥离”出来。

当他学会了像对待朋友一样对待自己，他也就在情绪的泥潭里，找到了一根救命稻草。

**这叫内观。**

能自己把自己哄好的孩子，内心才是真正的强大。

  


# 03

**这世上肯定有人不喜欢你，就像你也不是喜欢所有口味的冰淇淋一样，这太正常了。**

我们在职场上，哪怕做得再完美，也总有领导挑刺，总有同事嫉妒。

**这是客观规律。**

但是很多孩子不懂，他们会因为同学的一句“我不跟你玩了”，而崩塌一整天。

因为他们觉得：

**我不被喜欢，肯定是我哪里做错了。**

你要用冰淇淋这个最简单的例子告诉他：

**被讨厌，不是你的错，只是口味不同。**

接受“被讨厌的勇气”，才是终结讨好型人格的唯一解药。

  


# 04

**你是现在需要我只听你发发牢骚，还是需要我帮你一起想办法？**

这句话，建议所有夫妻也朗读背诵全文。

多少家庭矛盾，是因为一方只是想宣泄情绪，另一方却非要当**“人生导师”** 给建议。

孩子也一样。

有时候他抱怨作业多，抱怨老师凶，他只是在倒情绪。

你非要上去讲大道理，那就是在堵他的嘴。

**分清楚“情绪需求”和“解决问题需求”。**

这一点，能解决亲子沟通90%的无效撕扯。

  


# 05

**你说得很有道理，但我刚才只听到了情绪。你能不能把情绪关小一点，把道理放大一点再说一次？**

孩子大吼大叫的时候，别急着吼回去。

你吼回去，就是两只失控的野兽在对咬。

这句话两个作用。

第一，肯定了他说得有道理，给了他尊严。

第二，指出了他的说话方式问题。

他在那一瞬间会明白：

**原来只要我好好说话，我的诉求是可以被听见的。**

这才是有效沟通的启蒙。

  


# 06

**你刚才那句话如果别人对你说，你会觉得舒服吗？如果不舒服，那就要换个说法。**

同理心，不是天生的。

**同理心是需要训练的。**

很多熊孩子说话伤人，不是因为坏，是因为缺乏同理心。

是因为他们缺乏想象力，想象不到别人会痛。

不要空洞地教他“要礼貌”。

要让他去**体验** 。

**镜像反射，是培养同理心最直接的手段。**

只有当他能穿进别人的鞋子里走两步，他才会真的懂得什么叫尊重。

  


# 07

**如果不经过允许就帮忙，那不是热心，那是打扰。**

这一点，很多家长自己都做不到。

看见孩子搭积木笨手笨脚，非要上去帮一把。

**这不叫爱，这叫边界入侵。**

这会让孩子觉得：

我不行，我离不开你。

你要教会孩子：

**尊重别人的独立性，忍住不插手，是一种更高级的教养。**

无论对朋友，还是对未来的伴侣。

不打扰，是成年人世界里最顶级的温柔。

  


# 08

**写在最后**

所谓的“哄”孩子，不是无底线的溺爱，也不是虚伪的夸奖。

而是**看见** 。

看见他的情绪，看见他的需求，看见他作为一个独立个体的尊严。

上面这7句话，看似是在教孩子怎么说话。

**其实是在帮他们构建一个强大的底层操作系统。**

**在这个系统里，有边界，有尊重，有接纳，更有自我。**

拥有这个系统的孩子，根本不需要你去逼他优秀。

因为一个内核稳定、情绪健康的人。

他想不优秀，都难。
