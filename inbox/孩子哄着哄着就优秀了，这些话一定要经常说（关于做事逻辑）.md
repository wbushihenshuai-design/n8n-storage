# 孩子哄着哄着就优秀了，这些话一定要经常说（关于做事逻辑）

**你是不是也希望你家孩子做事更有条理？**

**想让他们告别拖延，行动力更强？**

**希望他们遇到困难能积极想办法，而不是轻易放弃？**

每个父母都渴望孩子拥有清晰的做事逻辑，但很多时候，我们却不知道如何正确引导。

与其苦口婆心地讲道理，不如试试用**“哄”** 的方式，用对语言，激发孩子内在的优秀潜能。

**经常和孩子说这10句话，你会发现，培养孩子的做事逻辑，其实可以如此轻松有效。**

  


>   
> 
> 
> #  1、你做事之前都会先计划好，真棒！周全的计划是成功的一半，你的做事逻辑性很强。
> 
> **是不是经常看到你家孩子，做事情毛毛躁躁，想到哪做到哪？**
> 
> **这句话，就是要肯定孩子在计划方面的努力，哪怕只是一个小小的计划。**
> 
> “你做事之前都会先计划好”， 这句话能让孩子意识到，父母看到了他的计划性，并且给予了肯定。

  


>   
> 
> 
> “周全的计划是成功的一半”， 这句话用谚语点明了计划的重要性，让孩子明白计划不是可有可无的，而是成功的关键因素。
> 
> “你的做事逻辑性很强”， 这是一个高度的评价，直接点赞孩子的逻辑思维能力，增强孩子的自信心。
> 
> **记住，观察孩子的行为，捕捉到任何一点计划性的闪光点，及时用这句话鼓励他。**
> 
> 比如孩子说“我先想一下，等下再写作业”，你就可以说这句话。

  


>   
> 
> 
> # 2、你做事的步骤很清晰，一步一步，有条不紊！清晰的步骤能帮助你更好地完成任务，避免出错。
> 
> **有些孩子做事容易混乱，丢三落四，往往是因为步骤不清晰。**
> 
> **这句话，就是要引导孩子关注做事的步骤，培养他们的条理性。**
> 
> **“你做事的步骤很清晰，一步一步，有条不紊”，** 详细地描述孩子的行为，让孩子知道父母看到了他的细致和耐心。

  


>   
> 
> 
> **“清晰的步骤能帮助你更好地完成任务，避免出错”** ，点明清晰步骤的好处，让孩子明白这样做的好处，从而更愿意采用这种方式。
> 
> **可以结合具体的场景来称赞孩子，** 比如孩子整理书桌后，你可以说：“哇，你整理书桌的步骤很清晰，先整理书，再整理文具，最后擦桌子，一步一步，有条不紊！ 这样看起来真整洁！”

  


>   
> 
> 
> # 3、你做事的方法很得当，效率真高！找到正确的方法，能事半功倍，你很会思考方法。
> 
> **很多孩子做事效率低下，是因为方法不对。**
> 
> **这句话，就是要鼓励孩子思考做事方法，提升效率。**
> 
> **“你做事的方法很得当，效率真高！”** 用惊喜的语气，表达你对孩子做事方法的赞赏和肯定。

  


>   
> 
> 
> **“找到正确的方法，能事半功倍”，** 用简洁明了的语言，点出方法的重要性，让孩子理解方法对了，效率自然就高了。
> 
> **“你很会思考方法”，** 这是一个积极的评价，暗示孩子有思考方法的天赋和能力，鼓励孩子继续保持和发展这种能力。
> 
> **当孩子用更巧妙的方法解决了问题，或者比平时更快地完成了任务，一定要及时说这句话，** 比如孩子用思维导图复习功课，效率很高，你可以说：“你用思维导图复习功课，方法真得当，效率真高！你很会思考方法！”

  


>   
> 
> 
> # 4、你一旦决定要做，就会立刻行动，执行力真强！强大的执行力是实现目标的关键，你做得非常好。
> 
> **拖延症是很多孩子的通病，想得多，做得少。**
> 
> **这句话，就是要赞扬孩子的行动力，克服拖延。**
> 
> **“你一旦决定要做，就会立刻行动，执行力真强！”** 用肯定和赞赏的语气，强调孩子行动迅速，不拖泥带水。

  


>   
> 
> 
> **“强大的执行力是实现目标的关键”，** 点明执行力的重要性，让孩子明白行动力是达成目标不可或缺的品质。
> 
> **“你做得非常好”，** 直接肯定孩子的行为，增强孩子的自信心和行动力。
> 
> **当孩子说要做什么事情，并且立刻付诸行动时，不要吝啬你的赞美，** 比如孩子说“我要去写作业了”，然后立刻去书桌前，你可以说：“真棒！你一旦决定要做，就会立刻行动，执行力真强！这样肯定能很快完成作业！”

  


>   
> 
> 
> # 5、你遇到问题会积极想办法解决，而不是逃避，真棒！解决问题的能力非常重要，你正在不断提升。
> 
> **面对困难，有的孩子选择逃避，有的孩子则积极应对。**
> 
> **这句话，就是要鼓励孩子积极面对问题，培养解决问题的能力。**
> 
> **“你遇到问题会积极想办法解决，而不是逃避，真棒！”** 清晰地描述孩子的行为，并表达你的赞赏。**“而不是逃避”** 这部分对比，更能突出孩子积极应对问题的可贵之处。

  


>   
> 
> 
> **“解决问题的能力非常重要”，** 强调解决问题能力的重要性，让孩子意识到这项能力对未来的发展至关重要。
> 
> **“你正在不断提升”，** 用鼓励性的语言，肯定孩子在这方面的进步，激励孩子继续努力。
> 
> **当孩子遇到难题，没有退缩，而是积极尝试各种方法去解决时，一定要及时肯定他，** 比如孩子拼图遇到困难，没有放弃，而是尝试不同的拼法，最终完成，你可以说：“你拼图遇到问题会积极想办法解决，而不是放弃，真棒！解决问题的能力非常重要，你看，你又进步了！”

  


>   
> 
> 
> # 6、你做事目标很明确，知道自己想要什么，并朝着目标努力！明确的目标能指引方向，让你更有动力。
> 
> **有些孩子做事盲目，没有目标，容易迷失方向。**
> 
> **这句话，就是要引导孩子树立目标，并为之努力。**
> 
> **“你做事目标很明确，知道自己想要什么，并朝着目标努力！”** 肯定孩子目标明确，并且有为之奋斗的行动力。

  


>   
> 
> 
> **“明确的目标能指引方向，让你更有动力”，** 点明目标的重要性，让孩子明白目标就像灯塔，指引前进的方向，并提供前进的动力。
> 
> **当孩子为了达成某个目标而努力时，要及时认可他的目标感，** 比如孩子为了期末考试考好成绩，每天认真复习，你可以说：“你做事目标很明确，想要期末考试考好成绩，并朝着这个目标努力复习，真棒！明确的目标能指引方向，让你学习更有动力！”

  


>   
> 
> 
> # 7、你做事很注重效率，能在短时间内完成很多事情，真厉害！高效做事能让你有更多的时间去做其他事情。
> 
> **磨蹭拖拉是很多孩子的坏习惯，浪费时间，也影响效率。**
> 
> **这句话，就是要赞扬孩子做事效率高，珍惜时间。**
> 
> **“你做事很注重效率，能在短时间内完成很多事情，真厉害！”** 直接赞扬孩子做事效率高，用**“真厉害”** 表达你的惊喜和赞赏。

  


>   
> 
> 
> **“高效做事能让你有更多的时间去做其他事情”，** 点明高效做事的好处，让孩子明白效率高，就能节省更多时间做自己喜欢的事情。
> 
> **当孩子在规定时间内高效完成任务时，要及时表扬，** 比如孩子在规定的时间内完成了作业，并且质量很高， 你可以说：“你今天写作业很注重效率，这么快就写完了，而且写得很好，真厉害！高效做事能让你有更多的时间玩啦！”

  


>   
> 
>

> 
> **优秀的做事逻辑，不仅仅是按部就班，更要懂得不断改进和优化。**
> 
> **这句话，就是要鼓励孩子持续改进，追求卓越。**
> 
> **“你还会不断优化做事流程，让事情做得更好更顺畅，真棒！”** 赞扬孩子有改进意识，并且付诸行动，让事情变得更好。

  


>   
> 
> 
> **“不断优化流程，体现了你的进步和思考”，** 高度评价孩子的行为，肯定孩子在进步和思考，鼓励孩子继续保持这种积极的思考习惯。
> 
> **当孩子在完成任务的过程中，主动改进方法，让流程更顺畅时，要及时鼓励，** 比如孩子整理房间，发现先整理大件物品再整理小件物品更高效，并主动调整，你可以说：“你整理房间还会不断优化流程，先整理衣服再整理玩具，让房间变得更整洁更顺畅，真棒！不断优化流程，体现了你的进步和思考！”

>

> **“不仅......还.......” 的句式，强调了实践的重要性。**

  


>

>

>
