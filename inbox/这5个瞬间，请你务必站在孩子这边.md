# 这5个瞬间，请你务必站在孩子这边

说真的，当爸妈最累的，有时候不是身体上的跑前跑后，而是心累。

你明明掏心掏肺，可孩子好像就是不领情，感觉俩人中间隔着一堵墙。

我们总想着在孩子发脾气、闹情绪的时候好好教育他，但其实，真正能让孩子心里踏实、有底气的，反而是那些我们一不留神就错过了的小事。

**在下面这几个时刻，先别急着讲道理、论对错。**

你只要选择站在他那边，就等于在他心里种下了一颗**“安全感”** 的种子，这比什么都重要。

  


# 01.孩子开始跟人“比”的时候

 _**“妈妈，同桌的画比我的好看。”**_

 _**“爸爸，同学他们家都有车，就我们家没有。”**_

听到这话，我们的第一反应是不是就想马上纠正他？

 _**“别瞎比！你也很棒！”**_

 _**“有啥好比的，过好自己的就行了。”**_

我们本想安慰他，可孩子听到的却是： _**“我的感觉是错的，妈妈不理解我。”**_

要知道，当孩子主动跟你说这些时，他心里那份**“我不如别人”** 的失落感，是真真切切的。

他不是来听你上课的，他是在跟你求助，告诉你他有点不自信了。

 _**这时候，你得先站到他“难过”的情绪这边。**_

试着蹲下来，看着他的眼睛说：“哦？你觉得他的画更好看，心里有点不舒服了，是吗？”

 _**先让他知道“你的感受，我看到了”，再跟他聊别的。**_

这份被理解的感觉，比100句**“你最棒”** 更能让他重新挺起小胸膛。

  


# 02.孩子跑来跟你“告状”的时候

 _**“妈妈！哥哥把玩具弄坏了！”**_

 _**“爸爸！她抢我的东西！”**_

特别是对三四岁的孩子来说，他不是天生爱打小报告。

 _**他的小世界里，正在建立一套黑白分明的“规矩”。**_

当他看到规矩被破坏，他心里是慌的。

他跑来**“告状”** ，其实是在找你这个**“大法官”** 帮忙，希望你能 _**恢复他心中的秩序**_ 。

如果你不耐烦地摆摆手：“知道了，多大点事儿。”

那他学到的是什么？——**“原来规矩不重要，我的感受也不重要。”**

 _**这时候，你得站到他正在萌芽的“规矩感”这边。**_

认真地回应他：“是吗？谢谢你告诉我，我们一起去看看怎么回事。”

 _**你这一次的认真，就是在帮他建立是非对错。**_

这比你苦口婆心地讲100遍**“要守规矩”** 都有用。

  


# 03.你逼着孩子“道歉”的时候

两个孩子闹矛盾，你为了快点解决问题，对着自家孩子说：

**“快！跟弟弟说对不起！”**

孩子可能满脸不服气，但还是小声说了。

你觉得事情解决了，孩子也**“懂礼貌”了** 。

可实际上呢？

一次心不甘情不愿的道歉，对他来说毫无意义。

他学会的不是**“我错了”** ，而是**“对不起“** 这三个字，能让发火的大人闭嘴。

 _**我们亲手堵住了他自己反思、自己感到内疚的路，**_ 他心里那股**“我就是不想给他玩！”** 的火，根本就没灭。

 _**这时候，请务必站到他“真实的情绪”这边，而不是“面子”那边。**_

一个假惺惺的道歉，远不如一次真实的情绪沟通。

先让他把心里的委屈和愤怒说完，再引导他想想对方的感受。

教会他从心里负责，而不是从嘴上敷衍。

  


# 04.孩子想帮忙却“搞砸了”的时候

 _**他看你拖地，也抢着要拖，结果弄得到处是水；**_

 _**他想帮你端碗，结果“啪”一声，碗掉地上碎了。**_

是不是一股火直冲脑门？

“谁让你动的！又给我添乱！”

先等一下。

在他心里，他不是想**“捣乱”** ，他是想**“帮忙”** 。

他是在用自己的方式，笨拙地表达对你的爱，想参与到这个家里来。

你这一吼，可能会把他那颗想主动付出的心，给彻底浇凉了。

下一次，他可能就缩回手，再也不敢尝试了。

 _**这时候，请你务必站到他那份善良的“心意”这边。**_

笑着对他说：“哎呀，谢谢你这么想帮妈妈呀！碗碎了没关系，人没伤到就好。我们一起来收拾，下次妈妈教你怎么拿能更稳一点。”

**先肯定他的好心，再教他做事的方法。**

  


#  05.孩子开始拒绝你“抱抱”的时候

孩子慢慢大了，你像以前一样想去抱抱他、亲亲他，他却一下子躲开了。

心里是不是咯噔一下，有点失落？

 _**“怎么了？妈妈抱一下都不行了？”**_

其实，你应该为他高兴才对。

他推开你，不是不爱你了。

他是在用行动告诉你： _**“这是我的身体，我说了算。”**_

**这是孩子自我意识长大的一个重要标志，他在学习保护自己的“边界”。**

如果你觉得被冒犯，非要抱，那其实是在破坏他刚刚建立起来的自我保护能力。

 _**这时候，请务必站到他正在建立的“自我”这边。**_

尊重他的**“不”** ，他将来才懂得如何对别人说**“不”** ，也才懂得，真正的爱，首先是尊重。

  


# 06.写在最后

我知道，你可能会担心：

 _**“总是这么顺着他，会不会把他惯坏了？”**_

放心，这真不是惯。

 _**站在孩子这边，不等于说“你永远是对的”，而是告诉他：**_

“不管你遇到了什么事，开心、难过、还是做错了，爸爸妈妈永远是第一个愿意理解你、支持你的人。”

孩子从来不是被我们骂一顿就突然长大的，而是 _**在一次次被看见、被接纳的瞬间里，内心才慢慢变得强大。**_

你每一次选择站在他那边，都是在给他的人生加满一箱叫**“底气”** 的油。

有了它，他才敢独自去面对未来的风风雨雨。
