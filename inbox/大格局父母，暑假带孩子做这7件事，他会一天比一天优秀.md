# 大格局父母，暑假带孩子做这7件事，他会一天比一天优秀

暑假快开始了，你是不是也开始着急，担心孩子光顾着玩，心都玩野了，开学跟不上？

暑假，难道只能让孩子去上补习班，或者让他一直玩吗？

其实，聪明的爸妈，都知道怎么利用好暑假这段时间，来帮助孩子提高。

早上好，我是语嫣妈妈。

暑假不是学习的结束，反而是让孩子进步更快的好机会。

想让孩子在新学期表现得更好，只让他去补课肯定不行。

今天，我来分享7件暑假可以带孩子做的事，坚持做，你会发现他真的在一天天变得更棒。

  


# 01\. 复习上学期的课本

别以为放假了，书本就可以放一边不管了。

**如果上学期的内容没学好，那开学后听新课就会很吃力。**

**比如：**

下学期要学分数乘除法，可孩子上学期连**“最大公约数”** 和**“最小公倍数”** 都没弄明白，那新知识对他来说就太难了，肯定听不懂。

**小贴士：**

**暑假复习，关键不是让孩子做多少题，而是帮他找到没学会的地方，把这些知识点重新学会、弄懂。**

这比让他去上很多补习班、累得不行要好得多。

  


# 02\. 让孩子自己计时，解决问题

你家孩子是不是也写作业磨蹭，考试时总抱怨时间不够用？

这往往是因为，他平时对**“时间”** 这个东西没感觉，不知道一道题该花多久。

为了帮他建立起时间观念，我们可以把做练习变成一个**“计时挑战”** 的游戏。

**具体可以这样做：**

拿出一页练习题，跟孩子说：“宝贝，我们来个比赛，看你能不能在10分钟内做完这几道题？”

一开始他可能做不完，但这种比赛的感觉能让他更认真，主动想着要快一点。

**小贴士：**

这个方法的重点，是让孩子自己去感受时间，而不是被爸妈催着。

练得多了，孩子自己就知道怎么安排时间，考试的时候才能心里有底，不慌张。

  


# 03\. 重要的事情，上午做完

**我们都知道，上午的精神最好，注意力也最集中。**

把那些重要的、需要多思考的事情安排在上午做，效率会高很多。

**比如：**

**跟孩子商量好一个“上午安排”：**

9点到10点写作业，10点到11点看书。

你会发现，让孩子在精神最好的时候去做那些难一点的学习任务，他更容易完成，也不怎么抱怨了。

**小贴士：**

**这事儿最好有点仪式感。**

比如每天早上，用一杯牛奶或一句鼓励的话，提醒孩子“高效时间”开始了，让他更有动力。

  


# 04\. 陪孩子玩“情绪桌游”

以后孩子长大了，能不能过得好，不光看他聪不聪明，**更要看他会不会跟人相处、懂不懂得控制自己的脾气。**

能管好自己情绪的孩子，以后遇到困难才不容易被打倒。

**比如：**

**现在有很多教孩子认识情绪的卡片和桌游。**

在玩的过程中，孩子能很自然地学会说出自己的感觉，比如“我现在很生气”、“我感到有点难过”。

**当他能说出自己的情绪时，就离控制好情绪不远了。**

**小贴士：**

玩游戏的时候，爸妈的态度最重要。

当孩子说他难过时，别说“这有啥好哭的”，而是说**“妈妈知道你现在不好受”** 。先接受他的情绪，他才能学会处理情绪。

  


# 05\. 在小失败中，炼出真本事

**总护着孩子，他是长不大的。**

暑假正好是个机会，可以让他去面对一些小小的失败和困难，锻炼他承受挫折的能力。

**比如：**

  * **让他自己拼一个复杂的模型，就算拼错好几次；**
  * **让他学骑自行车，就算会摔倒；**
  * **让他试着自己洗袜子，就算洗不干净。**



这些“小失败”，能让孩子以后面对真正的困难时更坚强。

**小贴士：**

爸妈要做的，不是帮他把事情做好，而是在旁边陪着他、鼓励他。

当他想放弃的时候，跟他说一句**“再试一次，我看你快成功了！”** ，这比你直接动手帮他要好得多。

  


# 06\. 一起去好大学看看

我们总跟孩子说**“要考个好大学”** ，但他可能根本不明白是啥意思。

**带他亲眼去看一看，比你说100遍都有用。**

比如：

暑假带孩子去附近最好的大学里走一走，感受一下那里的环境，看看图书馆里认真看书的大哥哥大姐姐。

你可以告诉他：“你看，这些哥哥姐姐学习多努力，这里就是他们努力后来到的地方。”

**小贴士：**

**重点不是学校多有名，而是让孩子心里产生一个向往。**

**有了一个明确的目标，孩子才知道要为什么努力，学习起来才更有劲儿。**

  


#  07\. 一起看纪录片

**一个孩子以后能走多远，要看他的想法和见识。**

除了多读书、多旅游，看一部好纪录片，是让孩子长见识的一个非常好的方法。

**比如：**

  * 陪孩子看《动物世界》，他会知道大自然有多神奇；
  * 看一些历史纪录片，他能了解很多有趣的故事。这些都能让他的眼界更开阔。



**小贴士：**

看完了一定要跟孩子聊一聊。

问问他**“你最喜欢哪个部分？”“你有什么想法？”**

多跟孩子这样聊，才能让他把看到的东西，真正变成自己的想法和见识。
